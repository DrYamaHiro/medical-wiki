[F06.7] 軽度認知障害 / Mild Cognitive Impairment / MCI

主訴・ROS

主訴: 家族の勧め / もの忘れ外来希望 / 区の検診 / 介護保険申請のため

中核症状 (認知機能)

  • 短期記憶: 同じことを何度も聞く / 置き忘れ / 約束を忘れる
  • 見当識: 日付が不明 / 道に迷う / 季節感がない
  • 実行機能: 料理の手順が悪くなった / 家電が使えない / 釣銭計算ができない

周辺症状 (BPSD)

  • 精神系: 幻視・幻聴 / 物盗られ妄想 / 嫉妬妄想 / 易怒性・暴言 / 徘徊 / 夜間せん妄 / 睡眠障害 / 介護抵抗

生活機能 (DASC-21等)

  • ADL: 自立 / 一部介助 / 全介助
  • IADL: 買い物 / 服薬管理 / 金銭管理

背景・ROS

  • 全身症状: 体重変化 なし / 増加 / 減少 ＿kg, 食欲 良好 / 低下 / 過食 / 異食
  • 精神・神経: 物忘れ 有, 呂律難 有 / 無, 歩行障害 有 / 無, 睡眠 良好 / 不眠
  • 学歴・職歴: ＿
  • 嗜好品: 飲酒 なし / 機会飲酒 / 多飲 ；アルコール性除外
  • 家族歴: 認知症 / 脳卒中

所見・検査

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

神経学的所見

  • 麻痺: 有 / 無
  • 歩行障害: 小刻み / すくみ / ワイドベース / なし
  • 不随意運動: 固縮 / 振戦 / ミオクローヌス / なし

神経心理検査

  • HDS-R: ＿/30点, MMSE: ＿/30点
  • 時計描画試験(CDT): 正常 / 異常

院内迅速

  • 血糖: ＿ mg/dL, HbA1c: ＿ %

除外診断用採血 (外注)

  • 甲状腺機能(TSH/FT4), ビタミンB1/B12/葉酸, 電解質, 肝腎機能

診察前対応

（データなし）

評価

  • 診断根拠: 物忘れの訴えあり、HDS-R 21-26点程度の客観的低下あり、ADLは保たれている。
  • リスク因子: 高血圧 / 糖尿病 / 難聴 / 社会的孤立

方針・処方

  • 治療戦略: 生活習慣病の徹底管理。認知的介入（運動・社会的交流）。現時点で抗認知症薬は適応外。

指導・計画 (必須)​

  • 患者指導: 「認知症の予備軍」ではなく「今なら引き返せるチャンス」と動機づけ。社会的交流やデュアルタスク、運動を推奨。
  • 生活指導: 難聴があれば補聴器の早期使用を強く推奨。
  • 文書: [ID: Doc-Gen-Lifestyle] 生活習慣病指導用パンフレット
  • 再診目安: 6-12ヶ月毎の経過観察（認知症への移行チェック）

💡 Dr.Advice

[Treatable Dementiaの除外] 「治る認知症」を見逃さないこと。1. 慢性硬膜下血腫(CT依頼), 2. 正常圧水頭症(iNPH)(CT依頼), 3. 甲状腺機能低下症, 4. ビタミン欠乏症(B1, B12, 葉酸)の除外はプライマリケア医の責務である。

[聴取テクニック] 本人に「今日は何を食べましたか？」と聞き、答えられても安心せず家族に確認せよ。「取り繕い反応」に騙されないことが診断の第一歩。

[自動車運転] 診断がついた時点で道路交通法に基づき運転は「禁止」となる。「法律で決まっている」「事故を起こしてからでは遅い」と明確に引導を渡すこと。

[Revert (正常復帰)] MCIから認知症への移行率は年10-15%だが、健常レベルに戻るケースも16-40%ある。
