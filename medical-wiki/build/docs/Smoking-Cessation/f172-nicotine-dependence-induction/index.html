[F17.2] ニコチン依存症 / 導入 / Nicotine Dependence - Induction

主訴・ROS

喫煙状況

喫煙本数: ＿ 本/日 × ＿ 年 ブリンクマン指数: ＿ 起床から1本目まで: 5分以内 / 30分以内 / 60分以内 禁煙の動機: 健康不安 / 家族の勧め / 経済的理由 / 職場環境 過去の禁煙歴: なし / あり: ＿回, 最長＿ヶ月 失敗原因: イライラ / 飲み会 / ストレス / 体重増加

スクリーニング

TDSスコア(ニコチン依存症スクリーニングテスト): ＿/10点 5点以上で依存症 直近の喫煙: ＿時間前 禁煙宣言書の署名: 済 / 未

合併症・禁忌確認

精神疾患: なし / うつ病 / 不眠症 ；バレニクリン慎重投与 皮膚疾患: なし / かぶれやすい ；パッチ注意 循環器: 不安定狭心症 / 心筋梗塞直後 ；ニコチン製剤注意 職業: 運転手 / 高所作業 ；バレニクリン運転禁止説明済

所見・検査

身長: ＿cm, 体重: ＿kg, BMI: ＿ 血圧: ＿/＿ mmHg 呼気一酸化炭素濃度(CO): ＿ ppm ノンスモーカー<7 / 喫煙者>10-15

診察前対応

（データなし）

評価

  • 診断根拠：TDSスコア ＿点 (≧5)、ブリンクマン指数 ＿ (≧200 または35歳未満)。
  • 依存度判定：軽度 / 中等度 / 重度
  • 治療方針：12週間の禁煙プログラムを開始。
  • 薬剤選択：ニコチンパッチ / バレニクリン

方針・処方

  • 禁煙宣言書への署名取得。
  • 初回処方（2週間分）。
  • 次回予約（2週間後）の厳守指導。

貼付薬（ニコチンパッチ）を選択の場合

  • 使用法: 起床時に上腕・腹部・背中などに貼り、就寝前に剥がす（貼り替え）。

内服薬（バレニクリン）を選択の場合

  • チャンピックス錠（スターターパック） 1シート 14日分
  • 使用法: 最初の1週間は飲みながら喫煙可。8日目から完全禁煙。

文書・同意取得

  • 禁煙治療に関する同意書・計画書

💡 Dr.Advice

[保険適応要件] ニコチン依存症管理料を算定するには、①TDS≧5点、②ブリンクマン指数≧200、③直ちに禁煙する意思、④文書による同意（禁煙宣言書）の4条件が必須である。35歳未満はブリンクマン指数の要件が免除される。

[パッチの注意点] 「寝る前に剥がす」ことを忘れると、睡眠中にニコチンが入り続け、悪夢や不眠の原因となる。

[チャンピックスの吐き気] 吐き気は「コップ一杯の水で飲む」「食直後に飲む」ことで軽減できる。「空腹時に飲むと気持ち悪くなる」と指導せよ。
