<h3 className="soap-a">評価</h3>

- **診断根拠:** 排便回数減少、残便感、硬便。ブリストルスケール 1-2 / 3-5 / 6-7。
- **原因:** 機能性、薬剤性、器質性（大腸癌等）。

<h3 className="soap-p">方針・処方</h3>

- 浸透圧性下剤をベースに、必要に応じて刺激性下剤を頓用。
- 難治例には新規便秘薬（リンゼス・グーフィス等）を併用。

#### 処方・処置 (選択)

**便秘治療セット**

- <span className="med-id">[Rx-GI-Const-Mag]</span> 酸化マグネシウム
- 酸化マグネシウム錠330mg 1回1錠 1日3回 毎食後 28日分 *— 基本薬*
- <span className="med-id">[Rx-GI-Const-Lin]</span> リンゼス
- リンゼス錠0.25mg 1回2錠 1日1回 食前 28日分 *— 腹痛・膨満感ある時*
- <span className="med-id">[Rx-GI-Const-Goo]</span> グーフィス
- グーフィス錠5mg 1回2錠 1日1回 食前 28日分 *— 胆汁酸利用*
- <span className="med-id">[Rx-GI-Const-Sen]</span> センノシド(頓用)
- センノシド錠12mg 1回2錠 頓用(便秘時・就寝前) 10回分 *— 刺激性*

**処置（院内）**

- <span className="med-id">[Rx-GI-Const-Gly]</span> グリセリン浣腸

> **[脱却]** 刺激性下剤（センノシド、アローゼン等）の連用は耐性を生むため、ベース薬（Mag、Lin、Goo）でコントロールし、刺激性はあくまで頓用に留めさせること。
