<h3 className="soap-s">主訴・ROS</h3>

主訴: 腹痛 / 便秘 / 下痢 / 血便 / 肛門痛 / 残便感

**便通・性状**

頻度: ＿ 日に1回
便性状: 硬便 / 兎糞状 / 普通便 / 泥状 / 水様 / 粘血便 / 黒色便
随伴症状: 残便感 / 腹部膨満感 / 排便時痛 / しぶり腹
腹痛との関連: 排便で軽快する / 排便後も続く / 関係ない

**肛門症状**

出血: 紙に付く / ポタポタ落ちる / 便器が真っ赤 / 粘液のみ
疼痛: 排便時のみ / 持続的激痛 / 腫れを伴う
脱出: なし / 排便時のみ / 指で戻る / 戻らない

**リスク・背景**

生食歴: 肉 / 魚 / 卵 / なし, 周囲の流行: 有 / 無
アラームサイン: 体重減少 / 発熱 / 家族歴:大腸癌

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**腹部診察**

視診: 平坦 / 膨満 / 手術痕なし
聴診: 腸雑音 正常 / 亢進 / 減弱 / 金属音
触診: 軟 / 筋性防御, 圧痛: 心窩部 / 臍周囲 / 右下腹部 / 左下腹部 / 全体
腫瘤: なし / あり: ＿, S状結腸便塊: 触知せず / あり

**肛門視診・指診 (DRE)**

視診: 皮垂 / 外痔核 / 裂肛 / 瘻孔 / 脱肛 / 異常なし
指診: 腫瘤触知せず / 痔核触知 / 圧痛あり, 前立腺: 正常 / 肥大
指袋付着: 鮮血 / 暗赤色血 / 粘液 / 便のみ / なし

<h3 className="soap-a">評価</h3>

- **診断根拠:** 軟便・下痢の持続。
- **原因:** 感染性 / 薬剤性 / 過敏性 / 寒冷刺激 / 暴飲暴食

<h3 className="soap-p">方針・処方</h3>

- 整腸剤による腸内フローラ改善。
- 止痢薬は感染性を否定できた場合のみ使用。

#### 処方・処置 (選択)

**整腸剤選択**

- <span className="med-id">[Rx-Gen-GI-Bio]</span> ビオフェルミン
- ビオフェルミン錠 1回1錠 1日3回 毎食後 14日分
- <span className="med-id">[Rx-Gen-GI-Bio3]</span> ビオスリー
- ビオスリー配合錠 1回1錠 1日3回 毎食後 14日分
- <span className="med-id">[Rx-Gen-GI-Miya]</span> ミヤBM
- ミヤBM錠 1回1錠 1日3回 毎食後 14日分 *— 酪酸菌(強力・抗生剤併用可)*

**止痢薬（非感染性のみ）**

- <span className="med-id">[Rx-GI-Dia-Lop]</span> ロペミン(頓用)
- ロペミンカプセル1mg 1回1カプセル 1日2回 朝夕食後 頓用 10回分

> 💡 **Dr.Advice**
>
> **[指診の重要性]** 「お尻を見られるのは恥ずかしい」と拒否する患者もいるが、直腸癌の約半数は指診で届く範囲にある。カーテンやタオルで羞恥心に配慮しつつ、可能な限り実施せよ。
>
> **[ミヤBM]** 酪酸菌（宮入菌）は芽胞を形成するため胃酸や抗生物質に強く、生きたまま腸に届く。抗生剤投与時の下痢予防や、他の整腸剤が無効な場合に第一選択となる。
