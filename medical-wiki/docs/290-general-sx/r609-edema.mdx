<h3 className="soap-s">主訴・ROS</h3>

**全身症状**

発熱: なし / 微熱 / 高熱 / 弛張熱, 期間: ＿週間
倦怠感: なし / 朝強い / 夕方強い / 一日中 / 活動後増悪
体重変化: なし / 減少: ＿kg[＿ヶ月で] ；意図しない減少はRedFlag
その他: 寝汗 / 関節痛 / 皮疹

**浮腫・リンパ節**

浮腫: なし / 下肢 / 顔面 / 全身, 左右差: 有 / 無
しこり: なし / 頸部 / 腋窩 / 鼠径, 痛み: 有 / 無

**背景**

内服薬: なし / あり: 薬剤熱・浮腫の被疑薬確認
接触歴: 渡航歴 / 動物接触 / 生食 / 性交渉 / 結核接触
既往歴: 癌 / 結核 / 膠原病家族歴

<h3 className="soap-o">所見・検査</h3>

バイタル: BT: ＿℃

**頭頸部**

眼瞼結膜: 正常 / 貧血, 側頭動脈: 正常 / 怒張 / 圧痛
リンパ節: 触知せず / 軟・可動 / 硬・癒着 / 圧痛あり
Virchow節: 触知 / なし ；左鎖骨上窩
甲状腺: 正常 / 腫大

**胸腹部・四肢**

心雑音: なし / あり, 肝脾腫: なし / あり
浮腫: Pitting / Non-pitting, 皮疹: なし / あり

**院内迅速検査**

尿検査
- 蛋白(＿) 潜血(＿)
末梢血一般(CBC), CRP
- WBC ＿, CRP ＿, HbA1c ＿, 血糖 ＿

**画像**

胸部単純X線検査
- **所見:** 異常なし / 肺門リンパ節腫脹 / 胸水
- **AI解析結果:** 異常なし / 異常所見あり: ＿

<h3 className="soap-ns">診察前対応</h3>

**院内迅速検査**

- 尿定性
- 末梢血一般(CBC), CRP

**画像**

- 胸部単純X線撮影

<h3 className="soap-a">評価</h3>

- **診断根拠:** 圧痕性(Pitting)または非圧痕性(Non-pitting)の腫脹。
- **分布:** 両側性 / 片側性
- **鑑別:** 心不全、腎不全、肝硬変、甲状腺機能低下、薬剤性、DVT（片側急激）。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 片側急激ならDVT疑い即紹介。両側慢性なら原因検索と対症療法（漢方・弾性ストッキング）。

#### 処方・処置

**スクリーニング検査 (外注)**

- 心不全マーカー
- BNP (またはNT-proBNP)
- 血栓マーカー
- Dダイマー ；DVT除外用
- 尿定性 ；蛋白尿チェック
- 甲状腺スクリーニング ；粘液水腫除外

**対症療法 (1つ選択)**

- ツムラ当帰芍薬散(23)エキス顆粒 2.5g 1回1包 1日3回 食前 28日分 *— 女性のむくみ*
- ツムラ五苓散(17)エキス顆粒 2.5g 1回1包 1日3回 食前 28日分 *— 利水剤*

#### 指導・計画

- **生活指導:** 弾性ストッキング着用、減塩。

> 💡 **Dr.Advice**
>
> **[Red Flags]** 「意図しない体重減少」「寝汗（結核・リンパ腫）」「鎖骨上窩リンパ節腫脹（Virchow転移）」「側頭動脈の圧痛（巨細胞性動脈炎）」は危険信号。徹底的な精査が必要。
>
> **[設備制約]** 当院にはCT・エコーがないため、深部リンパ節、膿瘍、固形腫瘍の評価はできない。Red Flagsがある場合や、初期スクリーニングで原因不明の場合は、抱え込まずに総合病院へ紹介せよ。
>
> **[薬剤性浮腫]** アムロジピン（Ca拮抗薬）やNSAIDs（ロキソニン）、甘草（漢方）、糖尿病薬（ピオグリタゾン）は浮腫の頻度が高い。検査漬けにする前に、まず薬歴を確認し、被疑薬の変更を試みよ。
>
> **[高齢者の浮腫]** 高齢者の下腿浮腫の多くは「廃用性（動かないことによる静脈還流不全）」や「低栄養（低アルブミン）」である。利尿薬は脱水を招くため、安易に投与せず、まずは運動と弾性ストッキングを推奨せよ。
