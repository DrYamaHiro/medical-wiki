<h3 className="soap-s">主訴・ROS</h3>

**頭痛 (OPQRST)**

発症様式: 突然[雷鳴] / 急性 / 徐々 / いつもの頭痛
増悪・寛解: 体動 / 入浴 / アルコール / 光・音 / 安静
性状: ズキズキ / 締め付けられる / 殴られたような / 電撃的
部位: 片側 / 両側 / 後頭部 / 目の奥 / 全体
強度: NRS ＿/10 生活に支障あり / 寝込む
経過: ＿時間/日 持続、頻度 ＿回/月

**随伴症状**

悪心・嘔吐、閃輝暗点、流涙・鼻汁、肩こり
しびれ、麻痺、言葉が出にくい、発熱

**てんかん・発作**

意識消失 有 / 無、痙攣 全身 / 一部、前兆
目撃情報: 白目 / 泡を吹く / 硬直 / 脱力

**背景**

既往歴: 片頭痛 / 高血圧 / 癌 / 副鼻腔炎 / 脳卒中
市販薬使用: なし / あり: ＿錠/月 ；MOHリスク評価

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**神経学的所見**

意識レベル: 清明 / JCS ＿ / GCS ＿
髄膜刺激症状: 項部硬直 有 / 無、Kernig徴候 有 / 無
局所徴候: 麻痺 有 / 無、Barre徴候 有 / 無
脳神経: 眼瞼下垂 有 / 無、瞳孔不同 有 / 無
小脳症状: 指鼻試験 正常 / 測定障害

**局所**

側頭動脈: 正常 / 怒張 / 圧痛 ；GCA疑い
圧痛点: 副鼻腔叩打痛 有 / 無、大後頭神経圧痛 有 / 無

**除外項目(Negative ROS)**

雷鳴頭痛の否定、神経脱落症状なし、項部硬直なし

<h3 className="soap-a">評価</h3>

- **診断根拠:** 反復する中等度～重度の頭痛。拍動性、片側性、日常動作で増悪。悪心・光過敏あり。
- **POUNDing基準:** Pulsating / duration 4-72h / Unilateral / Nausea / Disabling 4項目以上で高確率。
- **頻度:** ＿ 回/月
- **生活支障度:** 高度 / 中等度 / 軽度

<h3 className="soap-p">方針・処方</h3>

- 治療戦略:
- 急性期治療: トリプタン製剤（第一選択）。嘔気があれば制吐剤併用。
- 予防療法: 月2回以上の発作、または急性期治療無効例で導入。
- 生活指導: 誘発因子（ストレス解放時、寝過ぎ、空腹、アルコール、チラミン）の回避。

#### 処方 (選択)

**急性期治療 (トリプタン)**

- リザトリプタンOD錠10mg 1回1錠 頭痛発作時 頓用 5回分
- エレルトリプタン錠20mg 1回1錠 頭痛発作時 頓用 5回分
- イミグランキット皮下注3mg 1キット 頭痛発作時 頓用

**急性期治療 (その他)**

- ロキソプロフェンNa錠60mg 1回1錠 (疼痛時) 1日3回まで
- レバミピド錠100mg 1回1錠 (疼痛時)
- ナウゼリン錠10mg 1回1錠 頭痛発作時 頓用 5回分

**予防療法**

- ミグシス錠5mg 1回1錠 1日2回 朝夕食後 28日分
- デパケンR錠200mg 1回1錠 1日2回 朝夕食後 28日分
- インデラル錠10mg 1回1錠 1日3回 毎食後 28日分
- エムガルティ皮下注120mg 1本 皮下注射

> 💡 **Dr.Advice**
>
> **[SNOOP4 (Red Flags)]** 以下のサインは二次性頭痛（危険）を示唆する。一つでもあれば、当院で抱えずCT/MRIのある施設へ紹介せよ。
>
> **[雷鳴頭痛の定義]** 「突然」とは「何時何分」と言えるレベルの急激さであり、「バットで殴られたような」と表現される。痛みのピークが1分以内に完成する場合は、CTが正常でもルンバール（髄液検査）が必要なSAHの可能性がある。
>
> **[緑内障発作]** 「頭が痛い」と言って来院しても、実は「眼圧上昇（緑内障発作）」であることがある。特に高齢女性の「片頭痛＋吐き気＋眼痛＋充血」は要注意。ペンライトで瞳孔を確認し、散瞳・対光反射消失があれば即眼科へ。
>
> **[トリプタン無効例]** トリプタンが効かない場合、①飲むタイミングが遅すぎる、②用量不足、③そもそも片頭痛ではない（緊張型や副鼻腔炎）、④薬剤乱用頭痛の合併、を疑え。ナウゼリンを併用して胃の動きを良くすると吸収が改善し効くようになることがある。
>
> **[予防薬の導入]** 「頭痛が起きてから治す」のではなく「起きにくい脳を作る」治療であると説明せよ。効果判定には2ヶ月かかるため、焦らず継続させることが重要。
>
> **[CGRP製剤]** エムガルティ等のCGRP関連抗体薬は、従来の予防薬が無効な難治例に対し劇的な効果（発作半減～消失）を示すことが多い。高額だが、「人生が変わる」レベルの恩恵を受けられる患者がいるため、選択肢として提示する価値はある。
