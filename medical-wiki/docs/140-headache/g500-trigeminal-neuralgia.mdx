<h3 className="soap-s">主訴・ROS</h3>

**頭痛 (OPQRST)**

発症様式: 突然[雷鳴] / 急性 / 徐々 / いつもの頭痛
増悪・寛解: 体動 / 入浴 / アルコール / 光・音 / 安静
性状: ズキズキ / 締め付けられる / 殴られたような / 電撃的
部位: 片側 / 両側 / 後頭部 / 目の奥 / 全体
強度: NRS ＿/10 生活に支障あり / 寝込む
経過: ＿時間/日 持続、頻度 ＿回/月

**随伴症状**

悪心・嘔吐、閃輝暗点、流涙・鼻汁、肩こり
しびれ、麻痺、言葉が出にくい、発熱

**てんかん・発作**

意識消失 有 / 無、痙攣 全身 / 一部、前兆
目撃情報: 白目 / 泡を吹く / 硬直 / 脱力

**背景**

既往歴: 片頭痛 / 高血圧 / 癌 / 副鼻腔炎 / 脳卒中
市販薬使用: なし / あり: ＿錠/月 ；MOHリスク評価

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**神経学的所見**

意識レベル: 清明 / JCS ＿ / GCS ＿
髄膜刺激症状: 項部硬直 有 / 無、Kernig徴候 有 / 無
局所徴候: 麻痺 有 / 無、Barre徴候 有 / 無
脳神経: 眼瞼下垂 有 / 無、瞳孔不同 有 / 無
小脳症状: 指鼻試験 正常 / 測定障害

**局所**

側頭動脈: 正常 / 怒張 / 圧痛 ；GCA疑い
圧痛点: 副鼻腔叩打痛 有 / 無、大後頭神経圧痛 有 / 無

**除外項目(Negative ROS)**

雷鳴頭痛の否定、神経脱落症状なし、項部硬直なし

<h3 className="soap-a">評価</h3>

- **診断根拠:** 三叉神経領域（特に第2・3枝）の数秒間の電撃痛。
- **トリガー:** 洗顔、歯磨き、咀嚼、接触、風が当たる。
- **除外:** 帯状疱疹（皮疹なし）、脳腫瘍（MRIで血管圧迫を確認）。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** カルバマゼピン（テグレトール）が特効薬（診断的治療）。
- 副作用（ふらつき、薬疹）に注意し少量から開始。
- 無効・副作用時は脳外科紹介。

#### 処方・紹介 (選択)

**治療薬**

- テグレトール錠100mg 1回1錠 1日2回 朝夕食後 14日分 *— 少量開始*
- メチコバール錠500μg 1回1錠 1日3回 毎食後 28日分

**紹介**

- **紹介先:** 脳神経外科（微小血管減圧術MVD、ガンマナイフ適応評価）

> 💡 **Dr.Advice**
>
> **[SNOOP4 (Red Flags)]** 以下のサインは二次性頭痛（危険）を示唆する。一つでもあれば、当院で抱えずCT/MRIのある施設へ紹介せよ。
>
> **[雷鳴頭痛の定義]** 「突然」とは「何時何分」と言えるレベルの急激さであり、「バットで殴られたような」と表現される。痛みのピークが1分以内に完成する場合は、CTが正常でもルンバール（髄液検査）が必要なSAHの可能性がある。
>
> **[緑内障発作]** 「頭が痛い」と言って来院しても、実は「眼圧上昇（緑内障発作）」であることがある。特に高齢女性の「片頭痛＋吐き気＋眼痛＋充血」は要注意。ペンライトで瞳孔を確認し、散瞳・対光反射消失があれば即眼科へ。
>
> **[ふらつき注意]** テグレトールは高齢者でふらつき・眠気が出やすい。通常量(200mg x2)から始めず、必ず100mg x2程度の低用量から開始し、痛みが取れる最小量を探れ。効果は劇的である。
