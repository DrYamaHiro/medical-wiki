<h3 className="soap-s">主訴・ROS</h3>

**頭痛 (OPQRST)**

発症様式: 突然[雷鳴] / 急性 / 徐々 / いつもの頭痛
増悪・寛解: 体動 / 入浴 / アルコール / 光・音 / 安静
性状: ズキズキ / 締め付けられる / 殴られたような / 電撃的
部位: 片側 / 両側 / 後頭部 / 目の奥 / 全体
強度: NRS ＿/10 生活に支障あり / 寝込む
経過: ＿時間/日 持続、頻度 ＿回/月

**随伴症状**

悪心・嘔吐、閃輝暗点、流涙・鼻汁、肩こり
しびれ、麻痺、言葉が出にくい、発熱

**てんかん・発作**

意識消失 有 / 無、痙攣 全身 / 一部、前兆
目撃情報: 白目 / 泡を吹く / 硬直 / 脱力

**背景**

既往歴: 片頭痛 / 高血圧 / 癌 / 副鼻腔炎 / 脳卒中
市販薬使用: なし / あり: ＿錠/月 ；MOHリスク評価

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**神経学的所見**

意識レベル: 清明 / JCS ＿ / GCS ＿
髄膜刺激症状: 項部硬直 有 / 無、Kernig徴候 有 / 無
局所徴候: 麻痺 有 / 無、Barre徴候 有 / 無
脳神経: 眼瞼下垂 有 / 無、瞳孔不同 有 / 無
小脳症状: 指鼻試験 正常 / 測定障害

**局所**

側頭動脈: 正常 / 怒張 / 圧痛 ；GCA疑い
圧痛点: 副鼻腔叩打痛 有 / 無、大後頭神経圧痛 有 / 無

**除外項目(Negative ROS)**

雷鳴頭痛の否定、神経脱落症状なし、項部硬直なし

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 片側（稀に両側）の後頭部～頭頂部の電撃的・突き刺すような痛み。
- **圧痛点:** 大後頭神経出口（Arnold's point）に強い圧痛。
- **感覚障害:** 支配領域のピリピリ感（アロディニア）。
- **除外:** 帯状疱疹（皮疹なし）、椎骨動脈解離（持続痛・突然発症）。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 保存療法（局所安静、温熱、NSAIDs、ビタミンB12）。
- 難治例には神経ブロック（局所麻酔薬注射）。

#### 処方・処置 (選択)

**内服治療**

- ロキソプロフェンNa錠60mg 1回1錠 1日3回 毎食後 14日分
- レバミピド錠100mg 1回1錠 1日3回 毎食後 14日分
- タリージェ錠5mg 1回1錠 1日2回 朝夕食後 14日分 *— 7日以上かけて効果をみながら増量*
- メチコバール錠500μg 1回1錠 1日3回 毎食後 14日分

**神経ブロック**

- キシロカイン注射液1% 5mL 1本 圧痛点に局所注射

> 💡 **Dr.Advice**
>
> **[SNOOP4 (Red Flags)]** 以下のサインは二次性頭痛（危険）を示唆する。一つでもあれば、当院で抱えずCT/MRIのある施設へ紹介せよ。
>
> **[雷鳴頭痛の定義]** 「突然」とは「何時何分」と言えるレベルの急激さであり、「バットで殴られたような」と表現される。痛みのピークが1分以内に完成する場合は、CTが正常でもルンバール（髄液検査）が必要なSAHの可能性がある。
>
> **[緑内障発作]** 「頭が痛い」と言って来院しても、実は「眼圧上昇（緑内障発作）」であることがある。特に高齢女性の「片頭痛＋吐き気＋眼痛＋充血」は要注意。ペンライトで瞳孔を確認し、散瞳・対光反射消失があれば即眼科へ。
>
> **[帯状疱疹の先行痛]** 後頭神経痛と診断した数日後に、同部位に水疱が出現し「帯状疱疹」だったケースが多い。初診時に「もし赤いブツブツが出たら、すぐに再診してください（抗ウイルス薬が必要です）」と必ず伝えておくこと。これがリスクマネジメントである。
>
> **[椎骨動脈解離]** 「美容院でのシャンプー後」「カイロプラクティック後」などに発症した急激な後頸部痛は、椎骨動脈解離を疑う。痛み止めで様子を見ず、MRI(MRA)を考慮せよ。
