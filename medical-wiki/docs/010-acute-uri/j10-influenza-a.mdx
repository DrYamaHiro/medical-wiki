<h3 className="soap-s">主訴・ROS</h3>

主訴: 発熱 / 咽頭痛 / 咳嗽 / 鼻汁 / 喀痰 / 頭痛 / 倦怠感 / 関節痛 / 悪寒 / 呼吸困難

**呼吸器系**

咳嗽: 乾性 / 湿性 / 犬吠様, 喀痰: 粘液性 / 膿性 / 血痰 / なし
呼吸困難: なし / あり: 安静時・労作時, 喘鳴: なし / あり

**耳鼻・その他**

鼻汁: 水様 / 膿性, 咽頭痛: 嚥下痛 / 自発痛

**背景**

発症時期: ＿
周囲の感染状況: 家族 / 職場 / 学校 に インフル / コロナ / 感冒 あり
ワクチン接種歴: インフル 済 / 未, コロナ 済 / 未

<h3 className="soap-o">所見・検査</h3>

咽頭: 発赤 / 扁桃肥大 / 膿栓・白苔 / 口蓋垂偏位
頸部リンパ節: 腫脹なし / あり: 圧痛

**胸部**

呼吸音: 清明 / 喘鳴 / Crackles / Rhonchi / 減弱

**迅速検査**

- 新型コロナウイルス抗原精密測定
- A群ベータ溶連菌抗原定性
- アデノウイルス抗原精密測定

**画像 (必要時)**

- 胸部単純X線撮影

<h3 className="soap-a">評価</h3>

- **診断根拠:** 急激な発熱、悪寒、関節痛、咳嗽。迅速抗原検査 A型陽性。
- **重症度:** 軽症 / 中等症 / ハイリスク群
- **鑑別:** COVID-19、細菌性肺炎。

<h3 className="soap-p">方針・処方</h3>

- 抗インフルエンザ薬の投与（発症48時間以内の場合）。
- 異常行動についての注意喚起（特に小児・未成年）。
- 就業・登校禁止期間の指導。
- 経口・吸入が困難な状態（嘔吐・意識障害）の場合は、高次医療機関へ紹介する（当院ラピアクタなし）。

#### 処方・アクション (以下から選択)

**抗ウイルス薬**

- オセルタミビルカプセル75mg 1回1カプセル 1日2回 朝夕食後 5日分
- ゾフルーザ錠20mg 2錠 1回服用 (体重80kg未満)
- イナビル吸入粉末剤20mg 2キット 1回吸入
- リレンザディスク5mg 1日2回 5日間 吸入

**感冒薬セット(4剤)**

- アセトアミノフェン錠500mg 1回1錠 1日3回 毎食後 5日分 *— 解熱鎮痛*
- ムコダイン錠500mg 1回1錠 1日3回 毎食後 5日分 *— 去痰*
- メジコン錠15mg 1回2錠 1日3回 毎食後 5日分 *— 鎮咳*
- トラネキサム酸錠250mg 1回2錠 1日3回 毎食後 5日分 *— 抗炎症*

> 💡 **Dr.Advice**
>
> **[RedFlag]** 喉頭蓋炎の兆候（流涎、含み声、呼吸困難、吸気性喘鳴）がある場合は直ちに高次医療機関へ搬送せよ。
>
> **[鑑別]** Centor Criteria（発熱・咳なし・前頸部リンパ節圧痛・扁桃白苔）を確認し、細菌性（溶連菌）の可能性を見極めよ。
>
> **[設備制約]** 当院にはCTおよびファイバースコープがないため、喉頭蓋炎や深頸部膿瘍が疑われる場合は、身体所見のみでトリアージし、早期に専門医へ紹介せよ。
>
> **[治療戦略]** 48時間を超えている場合は抗ウイルス薬の効果は限定的である。ハイリスク群以外は対症療法も選択肢となる。
>
> **[NSAIDs注意]** インフルエンザ脳症のリスクを考慮し、アセトアミノフェンを第一選択とせよ。
