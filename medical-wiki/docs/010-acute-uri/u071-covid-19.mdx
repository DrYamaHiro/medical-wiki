<h3 className="soap-s">主訴・ROS</h3>

主訴: 発熱 / 咽頭痛 / 咳嗽 / 鼻汁 / 喀痰 / 頭痛 / 倦怠感 / 関節痛 / 悪寒 / 呼吸困難

**呼吸器系**

咳嗽: 乾性 / 湿性 / 犬吠様, 喀痰: 粘液性 / 膿性 / 血痰 / なし
呼吸困難: なし / あり: 安静時・労作時, 喘鳴: なし / あり

**耳鼻・その他**

鼻汁: 水様 / 膿性, 咽頭痛: 嚥下痛 / 自発痛

**背景**

発症時期: ＿
周囲の感染状況: 家族 / 職場 / 学校 に インフル / コロナ / 感冒 あり
ワクチン接種歴: インフル 済 / 未, コロナ 済 / 未

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃, SpO2: ＿% (Room Air)

**頭頸部**

咽頭: 発赤 / 扁桃肥大 / 膿栓・白苔 / 口蓋垂偏位
頸部リンパ節: 腫脹なし / あり: 圧痛

**胸部**

呼吸音: 清明 / 喘鳴 / Crackles / Rhonchi / 減弱
心音: 整 / 雑音なし

**皮膚**

発疹: 有 / 無

**迅速検査**

インフルエンザ抗原検査 A陽性 / B陽性 / 陰性
新型コロナウイルス抗原検査 陽性 / 陰性
A群ベータ溶連菌抗原検査 陽性 / 陰性
アデノウイルス抗原検査 陽性 / 陰性

**画像 (必要時)**

胸部単純X線検査
所見: 異常なし / 浸潤影 / 気管支壁肥厚
AI解析結果: 異常なし / 異常所見あり: ＿

<h3 className="soap-ns">診察前対応</h3>

**迅速検査**

- インフルエンザ抗原精密測定
- 新型コロナウイルス抗原精密測定
- A群ベータ溶連菌抗原定性
- アデノウイルス抗原精密測定

**画像 (必要時)**

- 胸部単純X線撮影

<h3 className="soap-a">評価</h3>

- **診断根拠:** 発熱、咽頭痛、咳嗽。迅速抗原検査 陽性。
- **重症度:** 軽症 / 中等症I / 中等症II / 重症
- **リスク因子:** 高齢、肥満、基礎疾患、ワクチン未接種。

<h3 className="soap-p">方針・処方</h3>

- 感染対策の徹底と自宅療養指導。
- 重症化リスクに応じた抗ウイルス薬の検討。

#### 処方・アクション

**重症化リスクあり・抗ウイルス薬**

- ゾコーバ錠125mg 初日3錠 2日目以降1錠 1日1回 5日分 (併用禁忌確認必須)
- ラゲブリオカプセル200mg 1回4カプセル 1日2回 5日分

**感冒薬セット(4剤)**

- アセトアミノフェン錠500mg 1回1錠 1日3回 毎食後 5日分
- ムコダイン錠500mg 1回1錠 1日3回 毎食後 5日分
- メジコン錠15mg 1回2錠 1日3回 毎食後 5日分
- トラネキサム酸錠250mg 1回2錠 1日3回 毎食後 5日分

**文書**

> 💡 **Dr.Advice**
>
> **[RedFlag]** 喉頭蓋炎の兆候（流涎、含み声、呼吸困難、吸気性喘鳴）がある場合は直ちに高次医療機関へ搬送せよ。
>
> **[鑑別]** Centor Criteria（発熱・咳なし・前頸部リンパ節圧痛・扁桃白苔）を確認し、細菌性（溶連菌）の可能性を見極めよ。
>
> **[設備制約]** 当院にはCTおよびファイバースコープがないため、喉頭蓋炎や深頸部膿瘍が疑われる場合は、身体所見のみでトリアージし、早期に専門医へ紹介せよ。
>
> **[併用禁忌]** ゾコーバは併用禁忌薬が多いため、必ずお薬手帳を確認せよ。妊婦には禁忌。
>
> **[RedFlag]** SpO2 93%以下、呼吸困難、持続する胸痛は入院適応を考慮し直ちに連携病院へ相談せよ。当院にはCTがないため、肺病変の評価は胸部X線に限定される。
