<h3 className="soap-s">主訴・ROS</h3>

主訴: 麻痺 / しびれ / 脱力 / 言語障害 / めまい / 激しい頭痛

**神経症状**

麻痺部位: 右 / 左 / 上肢 / 下肢 / 顔面
言語: 呂律が回らない / 言葉が出ない / 理解できない
嚥下: 飲み込みにくい / ムセる
その他: めまい / ふらつき / 視野欠損 / 複視 / 意識消失

**発症状況 (OPQRST)**

発症様式: 突然 / 徐々に / 朝起きたら[Wake-up stroke]
発症時刻: ＿時＿分 最終健常確認時刻

**リスク・背景**

既往歴: 高血圧 / 糖尿病 / 脂質異常症 / 喫煙 / 心房細動 / 脳卒中既往
内服薬: 抗凝固薬 / 抗血小板薬 / なし

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿ (整/不整), SpO2: ＿%, GCS: ＿ (JCS ＿)

**神経所見 (FAST)**

Face (顔面麻痺): 有 / 無
Arm (上肢麻痺): Barre徴候 下垂あり / なし
Speech (構語障害): 有 / 無
Time (発症時刻): ＿

**その他**

眼症状: 共同偏視 / 瞳孔不同 / 対光反射消失
小脳症状: 指鼻試験 正常 / 拙劣, 継ぎ足歩行 可能 / 不可
髄膜刺激症状: 項部硬直 有 / 無 ；SAH疑い
心電図: 正常洞調律 / 心房細動 / その他不整脈

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 脳局所症状が出現し、24時間以内（多くは1時間以内）に完全に消失したもの。
- **ABCD2スコア:** ＿点 (Age, BP, Clinical features, Duration, Diabetes)
- **リスク:** 発症後48時間以内の脳梗塞移行リスク極めて大。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 【緊急】当院では画像評価ができないため、症状が消失していても「脳梗塞の前兆」として直ちに専門病院へ紹介する。
- 帰宅させてはならない。

#### アクション

**紹介・搬送**

- **紹介先:** 脳神経外科・救急外来
- 抗血小板薬の投与は、出血の除外ができないため行わない。

> 💡 **Dr.Advice**
>
> **[設備制約・絶対搬送]** 当院にはCT/MRIがないため、脳梗塞か脳出血かの鑑別は不可能である。脳卒中を疑う所見があれば、ルート確保等は行わず、直ちに救急要請または専門病院へ紹介せよ。
>
> **[禁忌]** 画像診断なしにバイアスピリン等の抗血小板薬を投与することは、脳出血であった場合に増悪させるため「絶対禁忌」である。何も飲ませずに送ること。
>
> **[症状消失]** 「治ったから大丈夫」ではない。TIAは不安定なプラークが存在する証明であり、いつ本発作（脳梗塞）が起きてもおかしくない状態である。必ず専門医へ送れ。
