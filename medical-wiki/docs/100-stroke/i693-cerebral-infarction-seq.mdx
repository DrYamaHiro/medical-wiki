<h3 className="soap-s">主訴・ROS</h3>

**神経症状**

麻痺部位: 右 / 左 / 上肢 / 下肢 / 顔面
言語: 呂律が回らない / 言葉が出ない / 理解できない
嚥下: 飲み込みにくい / ムセる
その他: めまい / ふらつき / 視野欠損 / 複視 / 意識消失

**発症状況 (OPQRST)**

発症様式: 突然 / 徐々に / 朝起きたら[Wake-up stroke]
発症時刻: ＿時＿分 最終健常確認時刻

**リスク・背景**

既往歴: 高血圧 / 糖尿病 / 脂質異常症 / 喫煙 / 心房細動 / 脳卒中既往
内服薬: 抗凝固薬 / 抗血小板薬 / なし

小脳症状: 指鼻試験 正常 / 拙劣, 継ぎ足歩行 可能 / 不可

<h3 className="soap-a">評価</h3>

- **診断根拠:** 脳梗塞の既往（他院情報）。
- **病型:** ラクナ / アテローム / 心原性
- **後遺症:** 麻痺 / 構語障害 / 嚥下障害 / なし
- **リスク管理:** LDL目標&lt;70、血圧&lt;130/80。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 抗血小板療法または抗凝固療法の継続。
- 危険因子（高血圧・糖尿病・脂質異常症）の厳格な管理。
- 胃粘膜保護（抗血小板薬内服時）。

#### 処方・アクション (選択)

**抗血小板薬 (いずれか1剤)**

- バイアスピリン錠100mg 1回1錠 1日1回 朝食後 28日分

- クロピドグレル錠75mg 1回1錠 1日1回 朝食後 28日分

- プレタールOD錠100mg 1回2錠 1日2回 朝夕食後 28日分

**胃粘膜保護 (アスピリン併用時推奨)**

- タケキャブ錠10mg 1回1錠 1日1回 朝食後 28日分

#### 指導・計画 (必須)

- **患者指導:** 抗血小板薬の自己中断は再発リスクを高めるため厳禁。

> 💡 **Dr.Advice**
>
> **[設備制約・絶対搬送]** 当院にはCT/MRIがないため、脳梗塞か脳出血かの鑑別は不可能である。脳卒中を疑う所見があれば、ルート確保等は行わず、直ちに救急要請または専門病院へ紹介せよ。
>
> **[禁忌]** 画像診断なしにバイアスピリン等の抗血小板薬を投与することは、脳出血であった場合に増悪させるため「絶対禁忌」である。何も飲ませずに送ること。
>
> **[自動車運転]** 脳卒中後の運転再開には、道路交通法に基づき公安委員会での手続き（診断書提出や臨時適性検査）が必要である。自己判断での運転は「無免許運転」扱いとなるリスクがあるため、必ず専門の評価を受けるよう指導せよ。
>
> **[NSAIDs併用注意]** バイアスピリンやクロピドグレル内服中の患者に対し、安易にロキソプロフェン等のNSAIDsを漫然投与すると、消化管出血のリスクが倍増する。疼痛管理はアセトアミノフェンや湿布を優先せよ。
>
> **[服薬継続]** 「症状がないから治った」と誤解し服薬を自己中断することが、脳卒中再発の最大の要因である。「一生飲み続けることで、再発率を下げている保険」であることを繰り返し説明せよ。
>
> **[脈の確認]** 脳梗塞既往者（特に原因不明やラクナ梗塞とされた例）において、再診のたびに検脈を行い、心房細動（脈不整）が新たに出現していないか確認せよ。見つかれば抗凝固療法への変更が必要となる。
