<h3 className="soap-s">主訴・ROS</h3>

主訴: 検診異常 / 口渇 / 多飲 / 多尿 / 体重減少 / だるさ

**代謝・内分泌**

口渇・多飲: なし / あり, 多尿・夜間尿: なし / あり ＿回
体重変化: なし / 増加 / 減少 ＿kg

**合併症関連 (ROS)**

眼: 視力低下 / 霧視 / 飛蚊症
神経: 手足のしびれ / こむら返り / 立ちくらみ / 便通異常
心・腎: むくみ / 息切れ / 胸痛

**生活習慣・背景**

食事: 規則的 / 間食多い / 糖質過多 / アルコール
運動: なし / 定期的, 喫煙: なし / あり
家族歴: 糖尿病 父 / 母 / 兄弟
妊娠歴: 巨大児出産 / 妊娠糖尿病
他院治療歴: 無 / 有: 薬剤名 ＿

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BMI: ＿ (身長 ＿cm / 体重 ＿kg), 腹囲: ＿cm

**神経所見**

アキレス腱反射: 消失 / 低下 / 正常
振動覚(内果): ＿秒 / 低下 / 正常
Touch Test: 感覚鈍麻 / 正常

**フットチェック**

足背動脈: 触知 / 減弱, 爪白癬・鶏眼・胼胝 有 / 無

**院内迅速検査**

尿: 糖(＿) 蛋白(＿) 潜血(＿) ケトン(＿)
血糖: ＿ mg/dL, HbA1c: ＿ %

**外注検査**

CPR(インスリン分泌能), 尿中微量アルブミン

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 随時血糖≧200 or 空腹時≧126 or HbA1c≧6.5%。
- **病態:** インスリン抵抗性主体 / 分泌不全主体
- **合併症:** 網膜症(無/単純/増殖)、腎症(1/2/3/4/5期)、神経障害(有/無)。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 食事・運動療法のみ / 薬物療法開始 / 増量・追加 / 変更
- シックデイ、検査・手術時の休薬指導。

#### 処方 (選択)

**SGLT2阻害薬 (心・腎保護/体重減少)**

- ジャディアンス錠10mg 1回1錠 1日1回 朝食後 28日分
- フォシーガ錠10mg 1回1錠 1日1回 朝食後 28日分

**DPP-4阻害薬 (安全性高い)**

- ジャヌビア錠50mg 1回1錠 1日1回 朝食後 28日分
- トラゼンタ錠5mg 1回1錠 1日1回 朝食後 28日分 *— 腎機能低下時も同用量*

**ビグアナイド薬 (肥満・抵抗性改善)**

- メトグルコ錠250mg 1回2錠 1日2回 朝夕食後 28日分

**GLP-1受容体作動薬 (体重減少・食欲抑制)**

- リベルサス錠3mg 1回1錠 1日1回 起床時空腹時 28日分 *— 導入用*
- リベルサス錠7mg 1回1錠 1日1回 起床時空腹時 28日分 *— 維持用*
[ID: Inj-DM-GLP1-Mun2.5] マンジャロ2.5mg
- マンジャロ皮下注2.5mg 1キット 週1回皮下注 *— 導入用*
- マンジャロ皮下注5mg 1キット 週1回皮下注

**配合剤**

- イニシンク配合錠 1回1錠 1日1回 朝食後 28日分 *— DPP4/Met*

#### 指導・計画

- **栄養指導:** 適正カロリー / 糖質制限 / 減塩
- **再診目安:** 4週間後 (採血・尿検査)

> 💡 **Dr.Advice**
>
> **[1型除外]** 「急激な発症」「著明な体重減少」「ケトン体陽性」「若年」「痩せ型」は1型糖尿病（インスリン依存状態）を疑う。見逃すとDKAで死に至るため、必ずCPRと抗体検査を行え。
>
> **[眼科受診]** 糖尿病網膜症は自覚症状なく進行し、失明原因の上位である。当院では眼底検査ができないため、初診時に必ず眼科への対診依頼 [ID: Doc-Gen-Referral] を行え。
>
> **[処方日数]** オンライン診療において、基礎疾患情報（お薬手帳、健診結果、紹介状等）がない初診患者の場合は「7日分を上限」とする。情報ありなら通常処方可。
>
> **[シックデイ対策]** 発熱・下痢・嘔吐などで食事が摂れない時は、SGLT2阻害薬（フォシーガ等）とビグアナイド薬（メトグルコ等）は必ず休薬するよう指導せよ（ケトアシドーシス・乳酸アシドーシス予防）。
