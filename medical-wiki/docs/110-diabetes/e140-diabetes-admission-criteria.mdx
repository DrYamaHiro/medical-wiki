<h3 className="soap-s">主訴・ROS</h3>

主訴: 検診異常 / 口渇 / 多飲 / 多尿 / 体重減少 / だるさ

**代謝・内分泌**

口渇・多飲: なし / あり, 多尿・夜間尿: なし / あり ＿回
体重変化: なし / 増加 / 減少 ＿kg

**合併症関連 (ROS)**

眼: 視力低下 / 霧視 / 飛蚊症
神経: 手足のしびれ / こむら返り / 立ちくらみ / 便通異常
心・腎: むくみ / 息切れ / 胸痛

**生活習慣・背景**

食事: 規則的 / 間食多い / 糖質過多 / アルコール
運動: なし / 定期的, 喫煙: なし / あり
家族歴: 糖尿病 父 / 母 / 兄弟
妊娠歴: 巨大児出産 / 妊娠糖尿病
他院治療歴: 無 / 有: 薬剤名 ＿

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BMI: ＿ (身長 ＿cm / 体重 ＿kg), 腹囲: ＿cm

**神経所見**

アキレス腱反射: 消失 / 低下 / 正常
振動覚(内果): ＿秒 / 低下 / 正常
Touch Test: 感覚鈍麻 / 正常

**フットチェック**

足背動脈: 触知 / 減弱, 爪白癬・鶏眼・胼胝 有 / 無

**院内迅速検査**

尿: 糖(＿) 蛋白(＿) 潜血(＿) ケトン(＿)
血糖: ＿ mg/dL, HbA1c: ＿ %

**外注検査**

CPR(インスリン分泌能), 尿中微量アルブミン

<h3 className="soap-a">評価</h3>

- **判断根拠:** 以下のいずれかに該当する場合は、原則として入院管理が可能な専門病院へ紹介する。

**絶対的適応（緊急入院）**

1. 昏睡・意識障害がある（DKA / HHS 疑い）。
2. 尿ケトン体陽性 (++)以上、かつ消化器症状（悪心・嘔吐・腹痛）あり。
3. 重症感染症の合併（壊疽、肺炎、腎盂腎炎）。
4. 著しい高血糖（随時血糖 ≧ 400mg/dL）かつ脱水所見あり。

**相対的適応（待機的入院・教育入院）**

1. 外来でのインスリン導入が困難（手技習得不能、認知機能低下）。
2. 頻回な低血糖（特に無自覚性低血糖）を繰り返す。
3. 血糖コントロールが極めて不良（HbA1c ≧ 10%程度）。

<h3 className="soap-p">方針・処方</h3>

- **緊急時:** ルート確保（生食）、酸素投与を行い、救急搬送または緊急紹介。
- **待機時:** 地域連携室を通じて教育入院を予約。

#### 処置・紹介

**救急処置**

- 生理食塩液 500mL 点滴静注 *— 脱水補正*

**紹介**

- **紹介先:** 糖尿病専門医・救急科（DKA疑い・高血糖緊急症）

> 💡 **Dr.Advice**
>
> **[1型除外]** 「急激な発症」「著明な体重減少」「ケトン体陽性」「若年」「痩せ型」は1型糖尿病（インスリン依存状態）を疑う。見逃すとDKAで死に至るため、必ずCPRと抗体検査を行え。
>
> **[眼科受診]** 糖尿病網膜症は自覚症状なく進行し、失明原因の上位である。当院では眼底検査ができないため、初診時に必ず眼科への対診依頼 [ID: Doc-Gen-Referral] を行え。
>
> **[処方日数]** オンライン診療において、基礎疾患情報（お薬手帳、健診結果、紹介状等）がない初診患者の場合は「7日分を上限」とする。情報ありなら通常処方可。
>
> **[DKAのサイン]** 「糖尿病患者の腹痛・嘔吐」は、胃腸炎ではなくまずDKA（ケトアシドーシス）を疑え。クスマウル呼吸（深大呼吸）や呼気のアセトン臭（果実臭）があれば待ったなしの緊急事態である。
>
> **[ソフトクライテリア]** HbA1cが10%を超えていても、元気で食事が摂れていれば必ずしも即入院ではない。しかし「痩せてきた（異化亢進）」「喉が渇いて仕方ない（脱水）」という症状があれば、入院による急速是正が望ましい。
