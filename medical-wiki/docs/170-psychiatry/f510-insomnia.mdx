<h3 className="soap-s">主訴・ROS</h3>

**不眠のタイプ**

入眠障害: 30分以上 / 1時間以上 かかる
中途覚醒: 1回 / 2回以上 / その後眠れない
早朝覚醒: 起きたい時間より2時間以上早い
熟眠障害: ぐっすり眠った感じがしない

**精神・身体症状**

気分: 落ち込み / 不安感 / イライラ / 意欲低下
身体: 動悸 / 息苦しさ / 喉のつかえ / 頭痛
食欲: 低下 / 過食, 体重変化: ＿kg

**生活習慣・背景**

就床時刻: ＿時、起床時刻: ＿時
昼寝: なし / あり ＿分
嗜好品: カフェイン＿杯 / エナジードリンク / 寝酒あり
ストレス要因: 仕事 / 家庭 / 介護 / 金銭 / 特になし

<h3 className="soap-o">所見・検査</h3>

表情: 暗い / 不安げ / 緊張 / 涙ぐむ
身体所見: 甲状腺腫大 有 / 無

**除外検査 (外注)**

甲状腺スクリーニング
- TSH, FT3, FT4 ；機能亢進/低下の除外
末梢血一般(CBC)
- Hb ；貧血の除外

<h3 className="soap-ns">診察前対応</h3>

**除外検査 (外注)**

- 甲状腺スクリーニング
- 末梢血一般(CBC)

<h3 className="soap-a">評価</h3>

- **診断根拠:** 入眠障害、中途覚醒、早朝覚醒、熟眠障害。
- **重症度(AIS):** ＿/24点 6点以上で治療対象
- **原因:** ストレス / 生活リズム / 嗜好品 / 身体症状 / 薬剤性
- **フェーズ:** 導入・調整 / 維持 / 減薬・離脱

<h3 className="soap-p">方針・処方</h3>

- 睡眠衛生指導（起床時間の固定、カフェイン制限、寝床でのスマホ禁止）。
- 【オンライン初診】向精神薬の処方は不可。非向精神薬（オレキシン受容体拮抗薬、メラトニン受容体作動薬等）を選択し、基礎疾患不明時は7日分上限。
- 【対面/再診】必要に応じてベンゾジアゼピン系等を検討。
- **減薬期:** 睡眠が安定したら、漸減（1/4錠ずつ）や隔日投与へ移行し、漢方への置換も検討。

#### 処方・処置 (選択)

**初診オンライン可 (非向精神薬)**

**対面・再診専用 (向精神薬・オンライン初診不可)**

**減薬・切り替えセット**

[ID: Rx-Psych-Sleep-Day2.5] デエビゴ錠2.5mg 1回1錠 就寝直前 28日分 ；低用量へ減量

> 💡 **Dr.Advice**
>
> **[睡眠衛生]** 薬を出す前に「カフェインは午後控える」「寝酒はやめる（中途覚醒の原因）」「眠くなるまで布団に入らない」といった睡眠衛生指導を行うことが、薬物療法の効果を高める。
>
> **[うつ・双極性除外]** 不眠は「うつ病」の必発症状である。「気分が沈む」「死にたいと思う」などの精神症状がないか必ず確認せよ。また、過去に「異常に元気だった時期（躁）」がないかも確認が必要（抗うつ薬禁忌の可能性）。
>
> **[隠れうつ病 (2質問法)]** 「眠れない」はうつ病の必発症状である。以下の2点を必ず確認せよ（感度96%）。
>
> **[自殺リスク (死ぬ病気)]** 精神疾患は、癌や心筋梗塞と同様に「放置すれば死ぬ病気（自殺）」である。「消えてしまいたい、死にたいと思うことはありますか？」と直球で聞くことを躊躇うな。聞くことが自殺を誘発することはない。計画性がある場合は即座に専門医へ送れ。
>
> **[双極性障害 (躁転注意)]** うつ状態の患者に抗うつ薬を投与して「躁転（躁状態）」させると、人生を破壊する（散財、社会的信用失墜）リスクがある。「過去に異常に元気で、眠らなくても平気だった時期はないか」を必ず確認し、疑わしければ抗うつ薬には手を出さず専門医へ送れ。
>
> **[身体疾患の除外]** 「動悸・不眠」は甲状腺機能亢進症、「だるさ・うつ」は甲状腺機能低下症や貧血、「夜眠れない」は睡眠時無呼吸(SAS)やむずむず脚(RLS)の可能性がある。これらを見逃して向精神薬を出しても治らない。
>
> **[オンライン初診処方制限]** オンライン診療において、お薬手帳、健診結果、紹介状等で基礎疾患が把握できない初診患者の場合、処方は「7日分を上限」とせよ。
>
> **[まず非薬物療法]** 「薬を飲めば治る」ではない。「眠くなるまで布団に入らない（睡眠制限療法）」「朝の光を浴びる」といった行動療法を併用しないと、薬の効果は半減する。
>
> **[薬剤選択のロジック]** 現在は「デエビゴ（レンボレキサント）」などのオレキシン拮抗薬が第一選択である。依存性がなく、入眠から維持までカバーできる。これで効果不十分な場合や、スパッと寝たいという強い希望がある場合に限り、BZ系/非BZ系を「頓服」や「短期間」で追加する。
>
> **[デエビゴの悪夢]** オレキシン拮抗薬はREM睡眠を増やすため、悪夢を見ることがある。「夢を見るのは脳が処理している証拠」と説明するか、続くなら減量を検討せよ。
>
> **[多剤併用 (Polypharmacy)]** 「眠れないから」といって、ベンゾジアゼピン系を2種類以上重ねる（例：レンドルミン＋サイレース）のは、効果が増すより副作用（転倒・ふらつき・健忘）のリスクが跳ね上がるため推奨されない。併用するなら「作用機序の異なる薬」を組み合わせよ（例：デエビゴ［ベース］＋ ルネスタ［入眠頓用］）。
>
> **[高齢者への注意]** 高齢者に長時間作用型（ベンザリン等）や強力なBZ系（サイレース等）を出すと、夜間トイレ時の転倒骨折や、翌日のふらつき・誤嚥の原因となる。原則としてオレキシン系か、少量のリルマザホン（リスミー）等に留めよ。
>
> **[出口戦略]** 「ずっと飲み続ける薬ではありません」。調子が良ければ「半分に割る」「1日おきにする」など、具体的な減らし方を提示し、ゴール（卒業）を共有せよ。
