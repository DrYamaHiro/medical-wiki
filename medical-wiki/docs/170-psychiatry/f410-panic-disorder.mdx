<h3 className="soap-s">主訴・ROS</h3>

**不眠のタイプ**

入眠障害: 30分以上 / 1時間以上 かかる
中途覚醒: 1回 / 2回以上 / その後眠れない
早朝覚醒: 起きたい時間より2時間以上早い
熟眠障害: ぐっすり眠った感じがしない

**精神・身体症状**

気分: 落ち込み / 不安感 / イライラ / 意欲低下
身体: 動悸 / 息苦しさ / 喉のつかえ / 頭痛
食欲: 低下 / 過食, 体重変化: ＿kg

**生活習慣・背景**

就床時刻: ＿時、起床時刻: ＿時
昼寝: なし / あり ＿分
嗜好品: カフェイン＿杯 / エナジードリンク / 寝酒あり
ストレス要因: 仕事 / 家庭 / 介護 / 金銭 / 特になし

表情: 暗い / 不安げ / 緊張 / 涙ぐむ

- TSH, FT3, FT4 ；機能亢進/低下の除外

<h3 className="soap-a">評価</h3>

- **診断根拠:** 予期しないパニック発作（動悸、発汗、震え、息苦しさ、死ぬ恐怖）の反復。
- **予期不安:** 有 / 無, 広場恐怖: 有 / 無
- **除外診断:** 甲状腺機能亢進症、不整脈、喘息、過換気症候群（身体因性）。

<h3 className="soap-p">方針・処方</h3>

- SSRIによる薬物療法（第一選択）。
- 発作時の頓服（抗不安薬）をお守りとして処方。
- 認知行動療法的なアプローチ（暴露療法など）。

> 💡 **Dr.Advice**
>
> **[睡眠衛生]** 薬を出す前に「カフェインは午後控える」「寝酒はやめる（中途覚醒の原因）」「眠くなるまで布団に入らない」といった睡眠衛生指導を行うことが、薬物療法の効果を高める。
>
> **[うつ・双極性除外]** 不眠は「うつ病」の必発症状である。「気分が沈む」「死にたいと思う」などの精神症状がないか必ず確認せよ。また、過去に「異常に元気だった時期（躁）」がないかも確認が必要（抗うつ薬禁忌の可能性）。
>
> **[隠れうつ病 (2質問法)]** 「眠れない」はうつ病の必発症状である。以下の2点を必ず確認せよ（感度96%）。
>
> **[自殺リスク (死ぬ病気)]** 精神疾患は、癌や心筋梗塞と同様に「放置すれば死ぬ病気（自殺）」である。「消えてしまいたい、死にたいと思うことはありますか？」と直球で聞くことを躊躇うな。聞くことが自殺を誘発することはない。計画性がある場合は即座に専門医へ送れ。
>
> **[双極性障害 (躁転注意)]** うつ状態の患者に抗うつ薬を投与して「躁転（躁状態）」させると、人生を破壊する（散財、社会的信用失墜）リスクがある。「過去に異常に元気で、眠らなくても平気だった時期はないか」を必ず確認し、疑わしければ抗うつ薬には手を出さず専門医へ送れ。
>
> **[身体疾患の除外]** 「動悸・不眠」は甲状腺機能亢進症、「だるさ・うつ」は甲状腺機能低下症や貧血、「夜眠れない」は睡眠時無呼吸(SAS)やむずむず脚(RLS)の可能性がある。これらを見逃して向精神薬を出しても治らない。
>
> **[オンライン初診処方制限]** オンライン診療において、お薬手帳、健診結果、紹介状等で基礎疾患が把握できない初診患者の場合、処方は「7日分を上限」とせよ。
>
> **[死なない保証]** パニック発作は「死ぬほどの恐怖」を感じるが、医学的にこれで死ぬことはない。「心臓は止まらないし、窒息もしない。10分～30分で必ず治まる」と繰り返し保証し、安心させよ。
>
> **[ペーパーバッグ法禁止]** 過換気発作に対して「紙袋を口に当てる（ペーパーバッグ法）」は、低酸素血症による死亡事故の報告があり、現在は推奨されない。「口をすぼめて、ゆっくり息を吐く（吸う時間の倍かける）」呼吸法を指導せよ。
>
> **[カフェイン]** カフェインはパニック発作の強力な誘発因子である。コーヒー、エナジードリンク、緑茶の摂取を控えさせよ。
