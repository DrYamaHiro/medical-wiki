<h3 className="soap-s">主訴・ROS</h3>

悩み: 薄毛 / 肥満 / 疲労 / 肌荒れ / シミ / しわ
希望治療: AGA / ダイエット / プラセンタ / 美容内服 / 点滴

**確認事項**

既往歴: 糖尿病、膵炎、甲状腺疾患、肝障害、心疾患
内服薬: 抗凝固薬 / 糖尿病薬 / ホルモン剤
妊娠・授乳の可能性: 無 / 有 ；多くの自費薬で禁忌

<h3 className="soap-o">所見・検査</h3>

身長: ＿cm, 体重: ＿kg, BMI: ＿
血圧: ＿/＿ mmHg ；処方前必須

**評価**

脱毛パターン: Hamilton-Norwood / Ludwig
皮膚所見: 肝斑 / 老人性色素斑 / ニキビ痕

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- 診断根拠：パターン化した脱毛、軟毛化。
- 性別：男性 / 女性

<h3 className="soap-p">方針・処方</h3>

- 男性(AGA)：フィナステリドまたはデュタステリド（5α還元酵素阻害薬）。
- 女性(FAGA)：スピロノラクトン（抗男性ホルモン作用）。
- 女性への男性用薬（フィナステリド/デュタステリド）処方は絶対禁忌（胎児毒性）。

**AGA治療薬 (男性専用・自費)**

- **効能:** 5α還元酵素(II型)阻害。ヘアサイクルの正常化、抜け毛の抑制（進行遅延）。
- **効能:** 5α還元酵素(I型・II型)阻害。フィナステリドより強力な発毛効果・抜け毛抑制。

**FAGA治療薬 (女性用・自費)**

- スピロノラクトン2.5mg 1回1錠 1日1回 28日分
- **効能:** 抗アルドステロン作用・抗アンドロゲン作用により、抜け毛の原因となる男性ホルモンの働きを抑える。

> 💡 **Dr.Advice**
>
> **[混合診療の禁止]** 原則として、同日に「保険診療（風邪薬処方など）」と「自費診療（美容薬・点滴）」を併用することは混合診療とみなされ禁止されている。どちらか一方にするか、別日に受診するよう案内せよ。
>
> **[同意書]** 自由診療は保険適用外であるため、効果・副作用・費用について十分な説明を行い、必ず書面での同意を取得すること。
>
> **[副作用]** 性機能障害（リビドー減退）、肝機能障害。女性用スピロノラクトンは利尿作用による頻尿・血圧低下に注意。
