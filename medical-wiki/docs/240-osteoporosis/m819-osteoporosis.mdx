<h3 className="soap-s">主訴・ROS</h3>

検診異常: 骨密度低下 / YAM値指摘
身長低下: なし / あり: 若い頃より ＿cm 低下 ；2cm以上はリスク
背部痛: なし / あり: いつの間にか骨折疑い

**リスク因子(FRAX)**

年齢: ＿歳, 性別: 女 / 男, 閉経年齢: ＿歳 早発閉経 / 卵巣摘出
骨折歴: なし / あり: 部位・時期
家族歴: なし / 大腿骨近位部骨折 / 脊椎圧迫骨折
生活歴: ステロイド使用 / 喫煙 / アルコール多飲

<h3 className="soap-o">所見・検査</h3>

バイタル: 身長 ＿cm, 体重 ＿kg, BMI ＿
姿勢: 正常 / 円背, 壁際直立時の後頭部・壁距離 0cm / &gt;0cm

**骨密度 (他施設DEXA法等の結果)**

腰椎(L2-4): YAM ＿ %, T-score ＿
大腿骨頚部: YAM ＿ %, T-score ＿

**採血 (外注)**

骨代謝マーカー
- TRACP-5b(骨吸収), P1NP(骨形成), 25(OH)VitD
- Ca, P, ALP, Cre, eGFR

<h3 className="soap-ns">診察前対応</h3>

**採血 (外注)**

- 骨代謝マーカー

<h3 className="soap-a">評価</h3>

- **診断根拠:** 脆弱性骨折の既往、または骨密度YAM値&lt;70%。
- **骨代謝回転:** 高回転型 / 低回転型
- **除外:** 続発性骨粗鬆症（甲状腺機能亢進症、副甲状腺機能亢進症など）。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 経口薬による治療および管理を中心に行う。自己注射製剤やデノスマブ（プラリア）が必要な場合は専門医へ紹介。

#### 処方・処置

**メイン薬剤 (1つ選択)**

- アレンドロン酸錠35mg 1回1錠 週1回 起床時 4週分 *— 標準BP製剤*
- ボナロン経口ゼリー35mg 1回1包 週1回 起床時 4週分 *— 嚥下不安時*
- ビビアント錠20mg 1回1錠 1日1回 朝食後 28日分 *— 若年閉経後*

**併用薬 (必要時追加)**

- エディロールカプセル0.75μg 1回1カプセル 1日1回 朝食後 28日分 *— VitD不足・Ca吸収改善*

**紹介・連携**

- 歯科受診（口腔内チェック依頼）
- 整形外科（専門的治療・注射製剤検討依頼）

#### 指導・計画

- **患者指導:** BP製剤は起床直後に服用し、30分間は横にならず飲食も避けること（食道潰瘍予防・吸収確保）。
- **再診目安:** 4週間後。

> 💡 **Dr.Advice**
>
> **[身長低下]** 若い頃より「2cm以上」縮んでいれば椎体骨折の可能性が高い。「4cm以上」ならほぼ確実である。痛みがなくても胸腰椎X線を撮像せよ。
>
> **[YAM値]** 診断基準はYAM 70%未満（または80%未満＋危険因子）である。同年代比較(Age Match)ではなく、常に「若年成人平均(YAM)」を基準に判断せよ。
>
> **[設備制約]** 当院にはDEXA（骨密度測定装置）がないため、確定診断および詳細な経過評価には近隣のDEXA保有施設へ検査を依頼せよ。
>
> **[剤形選択]** 錠剤の服用が困難、または起床時のルール遵守が難しい場合は、ゼリー製剤を選択せよ。
>
> **[プラリア]** 当院ではプラリア（デノスマブ）の取り扱いはありませんが、骨折リスクが非常に高い患者や、経口薬の継続が困難な患者には極めて有効な選択肢です。6ヶ月に1回の皮下注射で済むため、経口ビスホスホネート製剤が生活スタイルに合わない患者や、胃腸障害で内服できない患者の治療継続を守るメリットがあります。「内服の飲み忘れが多い」「胃の不快感で続けられない」「重度の骨密度低下（YAM &lt;60%以下）」など、院内での経口薬管理に限界を感じる場合は、治療機会を損失させないよう、速やかにプラリア等の注射製剤が可能な整形外科・専門医へ紹介してバトンタッチしてください。プラリアは、投与を中止すると骨密度が急速に低下し、多発椎体骨折を起こす「リバウンド現象」のリスクがあります。紹介先を検討する際は、長期的な通院が可能か、あるいは中止時にビスホスホネート製剤へスイッチする出口戦略を共有できる施設を選定してください。
