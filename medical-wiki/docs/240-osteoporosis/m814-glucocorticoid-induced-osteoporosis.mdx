<h3 className="soap-s">主訴・ROS</h3>

検診異常: 骨密度低下 / YAM値指摘
身長低下: なし / あり: 若い頃より ＿cm 低下 ；2cm以上はリスク
背部痛: なし / あり: いつの間にか骨折疑い

**リスク因子(FRAX)**

年齢: ＿歳, 性別: 女 / 男, 閉経年齢: ＿歳 早発閉経 / 卵巣摘出
骨折歴: なし / あり: 部位・時期
家族歴: なし / 大腿骨近位部骨折 / 脊椎圧迫骨折
生活歴: ステロイド使用 / 喫煙 / アルコール多飲

<h3 className="soap-o">所見・検査</h3>

バイタル: 身長 ＿cm, 体重 ＿kg, BMI ＿
姿勢: 正常 / 円背, 壁際直立時の後頭部・壁距離 0cm / &gt;0cm

**骨密度 (他施設DEXA法等の結果)**

腰椎(L2-4): YAM ＿ %, T-score ＿
大腿骨頚部: YAM ＿ %, T-score ＿

**採血 (外注)**

骨代謝マーカー
- TRACP-5b(骨吸収), P1NP(骨形成), 25(OH)VitD
- Ca, P, ALP, Cre, eGFR

<h3 className="soap-ns">診察前対応</h3>

**採血 (外注)**

- 骨代謝マーカー

<h3 className="soap-a">評価</h3>

- **リスク評価:** プレドニゾロン換算 5mg/日以上を3ヶ月以上使用中または予定。
- **管理ガイドライン:** スコアに基づき介入。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 骨折リスクが高いため、早期から予防投与を開始する。

#### 処方・処置

**予防投与セット**

- アレンドロン酸錠35mg 1回1錠 週1回 起床時 4週分
- エディロールカプセル0.75μg 1回1カプセル 1日1回 朝食後 28日分

> 💡 **Dr.Advice**
>
> **[身長低下]** 若い頃より「2cm以上」縮んでいれば椎体骨折の可能性が高い。「4cm以上」ならほぼ確実である。痛みがなくても胸腰椎X線を撮像せよ。
>
> **[YAM値]** 診断基準はYAM 70%未満（または80%未満＋危険因子）である。同年代比較(Age Match)ではなく、常に「若年成人平均(YAM)」を基準に判断せよ。
>
> **[設備制約]** 当院にはDEXA（骨密度測定装置）がないため、確定診断および詳細な経過評価には近隣のDEXA保有施設へ検査を依頼せよ。
>
> **[早期介入]** ステロイドによる骨量減少は「投与開始直後」が最も急速である。骨密度が正常でも、高リスク例では直ちにBP製剤を開始せよ。
