<h3 className="soap-s">主訴・ROS</h3>

**排尿症状 (IPSS/OABSS)**

頻尿: なし / 昼間 ＿回 / 夜間 ＿回, 尿勢: 良好 / 低下 / 途切れる / 腹圧排尿
切迫感: なし / あり: 我慢できない / 漏れる, 残尿感: なし / あり

**疼痛・不快感**

部位: 会陰部 / 睾丸 / 下腹部 / 鼠径部 / 射精時, 排尿時痛: なし / あり

**男性機能・全身**

勃起不全: なし / あり, 性欲: あり / 低下
全身症状: 意欲低下 / 不眠 / イライラ / ほてり

**既往・背景**

既往歴: 糖尿病 / 脳卒中 / 脊椎疾患 / 尿路結石 / なし
内服薬: 抗コリン薬 / 睡眠薬 / 抗血小板薬 / なし

<h3 className="soap-o">所見・検査</h3>

腹部: 膀胱充満 なし / あり

**直腸診 (DRE)**

前立腺: 正常大 / 腫大: 鶏卵大 / 鵞卵大 / テニスボール大
硬度: 弾性軟 / 弾性硬 / 石様硬, 圧痛: なし / あり, 表面: 平滑 / 結節あり, 正中溝: 触知可 / 消失

**検査**

尿検査・尿沈渣
- **定性:** 蛋白(＿) 糖(＿) 潜血(＿) 白血球(＿)
- **沈渣:** RBC(＿) WBC(＿) ；感染・血尿除外
腫瘍マーカーセット(PSA含む)
- **PSA:** ＿ ng/mL (外注)
スコア評価: IPSS ＿点 (QOL ＿点), OABSS ＿点

<h3 className="soap-ns">診察前対応</h3>

**検査**

- 尿定性・尿沈渣
- 腫瘍マーカーセット(PSA含む)

<h3 className="soap-a">評価</h3>

- **診断根拠:** 尿勢低下、頻尿、残尿感。DREで弾性硬の腫大。
- **重症度(IPSS):** 軽症0-7 / 中等症8-19 / 重症20-35
- **除外:** 前立腺癌（PSA高値、石様硬結節）。
- **制約:** 正確な体積測定や残尿測定は不可。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** α1遮断薬またはPDE5阻害薬による薬物療法。前立腺腫大が著しい場合は5α還元酵素阻害薬の併用。尿閉リスクが高い場合や薬物抵抗性は泌尿器科紹介。

#### 処方・処置

**α1遮断薬 (1つ選択)**

- ハルナールD錠0.2mg 1回1錠 1日1回 朝食後 14日分 *— 標準・高齢者向け*
- ユリーフ錠4mg 1回1錠 1日2回 朝夕食後 14日分 *— 強力・射精障害注意*

**PDE5阻害薬 (血管拡張)**

- ザルティア錠5mg 1回1錠 1日1回 朝食後 28日分 *— OAB併用可・ニトロ禁忌*

**5α還元酵素阻害薬 (縮小)**

- アボルブカプセル0.5mg 1回1カプセル 1日1回 28日分 *— 効果発現に半年*

#### 指導・計画

- **患者指導:** アボルブ服用中は献血不可。PSA値が半減するため解釈に注意が必要であることを説明。

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院にはエコーがないため、正確な前立腺体積や残尿量の測定はできない。DRE（デジタル直腸診）による推定と、問診（IPSS）による重症度判定で治療方針を決定する。
>
> **[PSA解釈]** 4.0ng/mL以上は専門医紹介の目安だが、前立腺肥大症や前立腺炎、射精直後でも上昇する。「癌のマーカーですが、良性の病気でも上がります」と事前に説明し、不安を煽りすぎないよう配慮せよ。
>
> **[残尿リスク]** 頻尿を訴える高齢男性に、安易に抗コリン薬（頻尿止め）を出すと、隠れていた残尿が増加し「尿閉」に至ることがある。エコーで残尿確認ができない当院では、男性への抗コリン薬投与は原則として泌尿器科紹介後、あるいは慎重に行うべきである。
>
> **[アボルブ]** 前立腺を小さくするが、PSA値を約50%低下させる。PSA評価時は「測定値を2倍」して判定する必要があることを忘れるな。
