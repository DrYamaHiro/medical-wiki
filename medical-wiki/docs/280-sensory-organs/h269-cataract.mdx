<h3 className="soap-s">主訴・ROS</h3>

**眼症状**

充血: 右 / 左 / 両眼, 眼脂: 膿性 / 水様 / 粘液性
自覚症状: 掻痒感 / 異物感 / 乾燥感 / 流涙
視力・視野: 低下なし / 急激な視力低下 / 霧視 / 視野欠損
眼痛: なし / 表面 / 奥 / 動かすと痛い

**耳症状**

耳痛: なし / あり, 耳漏: なし / 膿性 / 血性
難聴: なし / 急激 / 徐々 / 低音が聞こえにくい
その他: 耳閉感 / 耳鳴り

**経過・背景**

発症時期: ＿
コンタクトレンズ使用: 有 / 無
既往歴: 緑内障 / 白内障 / 中耳炎 / アレルギー

<h3 className="soap-o">所見・検査</h3>

**眼**

結膜充血: 有 / 無, 眼瞼腫脹: 有 / 無
角膜・瞳孔: 角膜混濁 / 瞳孔不同 / 対光反射迅速 / 対光反射消失

**耳**

外耳道: 発赤 / 耳垢栓塞 / 異常なし
鼓膜所見: 正常 / 発赤 / 混濁 / 液貯留 / 穿孔
耳介牽引痛: 有 / 無 ；外耳炎サイン

**検査**

標準純音聴力検査
- **結果:** 実施せず / 正常 / 感音難聴 / 伝音難聴

<h3 className="soap-ns">診察前対応</h3>

**検査**

- 標準純音聴力検査

<h3 className="soap-a">評価</h3>

- **診断根拠:** 視力低下（矯正不能）、霧視、羞明。水晶体の混濁。

<h3 className="soap-p">方針・処方</h3>

- 治療戦略: 進行予防点眼薬。完治には手術しかないことを説明。生活に支障（運転免許更新等）が出たら眼科紹介。

#### 処方・処置

**進行抑制**

- カリーユニ点眼液0.005% 1本 1日3回 両眼

**紹介**

#### 指導・計画

- **患者指導:** 目薬で濁りが取れる（治る）わけではなく、進行を遅らせる薬であることを説明。
- **再診目安:** 3ヶ月後（定期処方）。

> 💡 **Dr.Advice**
>
> **[RedFlag - 眼]** 「急激な視力低下」「激しい眼痛（緑内障発作疑い）」「視野欠損」「眼球打撲」は専門医による緊急対応が必要。処方で粘らず即日紹介せよ。
>
> **[RedFlag - 耳]** 「突然聞こえなくなった（突発性難聴）」は時間との勝負（Time is Hearing）。ステロイド治療のゴールデンタイムを逃さないため、即座に聴力検査を行い、異常があれば直ちに専門医へ送れ。
>
> **[ステロイド点眼の禁忌]** 角膜ヘルペスや真菌感染にステロイドを点眼すると、劇的に悪化し失明のリスクがある。細隙灯で角膜を確認できない環境では、安易にステロイド点眼処方は避けるべきである。
