<h3 className="soap-s">主訴・ROS</h3>

主訴: 鼻汁 / 鼻閉 / くしゃみ / 鼻内掻痒感 / 後鼻漏 / 嗅覚障害 / 鼻出血

**鼻症状詳細**

鼻汁: 水様 / 粘性 / 膿性 / 血性
鼻閉: 片側 / 両側 / 交代性
嗅覚障害: 有 / 無

**背景**

発症時期・季節性: ＿
アレルゲン曝露: ハウスダスト / ペット / 花粉
出血 (鼻出血時): 左 / 右, 止血にかかった時間: ＿分
市販点鼻薬の使用: なし / あり: 長期連用

<h3 className="soap-o">所見・検査</h3>

**前鼻鏡**

粘膜: 蒼白浮腫 / 発赤腫脹 / 乾燥 / びらん
鼻汁: 水様 / 膿性 / 粘液性
鼻中隔: 弯曲なし / 右凸 / 左凸, 穿孔 有 / 無
キーゼルバッハ部位: 出血点あり / 痂皮 / 血管怒張

**検査**

副鼻腔圧痛: 有 / 無
単純X線 (Waters法): 上顎洞透過性低下 / ニボー像 / 正常

<h3 className="soap-a">評価</h3>

- **診断根拠:** 慢性的な片側性の鼻閉。前鼻鏡で鼻中隔の弯曲を確認。
- **合併症:** 肥厚性鼻炎、副鼻腔炎（換気不全による）。

<h3 className="soap-p">方針・処方</h3>

- 点鼻血管収縮薬（一時的）や点鼻ステロイドによる保存療法。
- 改善乏しい場合や睡眠時無呼吸の原因となる場合は手術適応（耳鼻科紹介）。

#### 処方・アクション (選択)

**保存療法セット**

- アラミスト点鼻液27.5μg 1瓶 1日1回 各鼻腔 噴霧
- プリビナ液0.05% 1本 1日数回 鼻腔内噴霧 *— 長期連用禁忌(血管収縮剤)*

**紹介**

- **紹介先:** 耳鼻咽喉科（鼻中隔矯正術の相談）

> 💡 **Dr.Advice**
>
> **[点鼻薬性鼻炎]** 市販の血管収縮薬入り点鼻薬（ナザール等）を長期連用していないか必ず確認せよ。これが原因で鼻閉が悪化している場合、使用中止させなければ治らない。
>
> **[設備制約]** 当院にはCTおよびファイバースコープがないため、X線で診断困難な副鼻腔炎や、上咽頭・深部の観察が必要な場合は耳鼻咽喉科へ紹介せよ。
>
> **[オンライン初診処方制限]** オンライン診療において、お薬手帳、健診結果、紹介状等で基礎疾患が把握できない初診患者の場合、処方は「7日分を上限」とせよ。
>
> **[手術]** 骨軟骨の構造異常であるため、薬物療法は対症療法に過ぎない。根治には手術（鼻中隔矯正術）が必要であることを説明し、適切なタイミングで専門医へ送れ。
