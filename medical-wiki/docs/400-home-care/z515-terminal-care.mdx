<h3 className="soap-a">評価</h3>

- 診断根拠：治癒困難な悪性腫瘍。予後予測（月単位 / 週単位 / 日単位）。
- PPS (Palliative Performance Scale)：＿ %
- 苦痛症状：疼痛 / 呼吸困難 / 嘔気 / 倦怠感 / 譫妄

<h3 className="soap-p">方針・処方</h3>

- 疼痛コントロール（WHO方式がん疼痛治療法）。院外処方にて麻薬を手配。
- 予期せぬ急変への対応（DNAR確認）。
- 副作用対策（便秘・嘔気）の徹底。

**疼痛管理・麻薬セット (院外処方)**

- <span className="med-id">[Rx-Home-Nar-Fen1]</span> フェントステープ1mg 1枚 1日1回 24時間毎貼付 ；貼付剤(安定時)
- <span className="med-id">[Rx-Home-Nar-Nar2]</span> ナルサス錠2mg 1回1錠 1日1回 朝食後 ；内服(安定時)
- <span className="med-id">[Rx-Home-Nar-Oxy5]</span> オキシコンチンTR錠5mg 1回1錠 1日2回 朝夕食後 ；内服

**レスキュー (突出痛用)**

- <span className="med-id">[Rx-Home-Nar-Ops]</span> オプソ内服液5mg 1包 疼痛時 1日4回まで
- <span className="med-id">[Rx-Home-Nar-Anp]</span> アンペック坐剤10mg 1個 疼痛時 1日4回まで
- <span className="med-id">[Rx-Home-Nar-Abstr]</span> アブストラル舌下錠100μg 1錠 疼痛時 1日4回まで

**副作用対策 (必須)**

- <span className="med-id">[Rx-GI-Const-Mag]</span> 酸化マグネシウム錠330mg 1回2錠 1日3回 ；便秘（必発）
- <span className="med-id">[Rx-GI-Sym-Nau]</span> ナウゼリン錠10mg 1回1錠 1日3回 ；嘔気（初期）
- <span className="med-id">[Rx-Home-Nar-Nov]</span> ノバミン錠5mg 1回1錠 1日3回 ；嘔気・不穏

> 💡 **Dr.Advice**
>
> **[初回契約]** 24時間対応体制、緊急時の連絡先、費用負担について書面を用いて説明し、同意書を取得すること。
>
> **[多職種連携]** ケアマネージャー、訪問看護師、薬局と情報を共有し、初回のカンファレンスを行うこと。
>
> **[ACP]** 初回から「死」の話をするのは繊細さが求められるが、「もしもの時、どうされたいか」を早めに共有しておくことが、家族の迷いを減らす。
>
> **[便秘対策]** オピオイドを開始する際は、必ず下剤を併用せよ（オピオイド誘発性便秘は耐性ができない）。
>
> **[レスキュー]** 1日定時投与量の10～20%（または1/6量）をレスキュー1回量として設定する。レスキューの使用回数が増えたら、ベースアップを検討せよ。
