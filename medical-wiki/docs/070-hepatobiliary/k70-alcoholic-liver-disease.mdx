自覚症状: 皮膚掻痒感 / 尿の色が濃い / 悪心 / 食欲不振

**リスク・背景**

飲酒歴: なし / あり: ＿ ml × ＿ 日/週 [種類: ビール/日本酒/焼酎]
体重変化: なし / 増加 / 減少 ＿kg
既往歴: 輸血歴 / タトゥー / 胆石 / 糖尿病 / 脂質異常症

肝・脾: 触知せず / 肝腫大 / 脾腫
圧痛: なし / 右季肋部 / 心窩部, Murphy徴候: 陰性 / 陽性

**皮膚・眼**

<h3 className="soap-a">評価</h3>

- **診断根拠:** 過剰なアルコール摂取歴（純エタノール60g/日以上）。γGTP優位の上昇。AST&gt;ALTパターン。
- **禁酒意思:** あり / なし

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 断酒・節酒指導が治療の全てである。
- 肝庇護薬の投与、重症例（肝不全徴候あり）は専門医紹介。

#### 処方・処置 (選択)

**肝庇護薬**

- <span className="med-id">[Rx-Hep-Urs]</span> ウルソ(肝庇護)
- ウルソ錠100mg 1回1錠 1日3回 毎食後 28日分

**注射 (AST/ALT著高時)**

- <span className="med-id">[Inj-Hep-Pro]</span> 強力ネオミノファーゲンシー
- 強力ネオミノファーゲンシー静注20mL 1管 静脈内注射

#### 指導・計画 (必須)

- **患者指導:** 断酒の必要性を説明。
- **再診目安:** 2-4週間後。

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院には腹部エコーおよびCTがない。肝臓の形状評価、胆石の有無、胆管拡張の確認ができないため、身体所見で異常（黄疸、Murphy徴候、腫瘤）があれば、迷わず画像診断可能な施設へ紹介せよ。
>
> **[FIB-4 Index]** 脂肪肝を見たら必ずFIB-4 indexを算出せよ。スコア2.67以上の場合は肝硬変への進展リスクが高いため、肝臓専門医へ紹介すること。

</div>

> **[説得]** 「肝臓が悲鳴を上げている」と数値を具体的に示して警告せよ。γGTPは断酒により速やかに改善するため、モチベーション維持の指標にしやすい。
