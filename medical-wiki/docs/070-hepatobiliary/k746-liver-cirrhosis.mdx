主訴: 検診異常 / 右季肋部痛 / 心窩部痛 / 背部痛 / 黄疸 / 倦怠感

**消化器・肝胆膵**

腹痛: なし / 右季肋部 / 心窩部 / 背部, 性状: 鈍痛 / 疝痛 / 持続的
自覚症状: 皮膚掻痒感 / 尿の色が濃い / 悪心 / 食欲不振

**リスク・背景**

飲酒歴: なし / あり: ＿ ml × ＿ 日/週 [種類: ビール/日本酒/焼酎]
体重変化: なし / 増加 / 減少 ＿kg
既往歴: 輸血歴 / タトゥー / 胆石 / 糖尿病 / 脂質異常症

**腹部**

肝・脾: 触知せず / 肝腫大 / 脾腫
圧痛: なし / 右季肋部 / 心窩部, Murphy徴候: 陰性 / 陽性

**皮膚・眼**

黄疸: なし / 眼球結膜 / 皮膚

<h3 className="soap-a">評価</h3>

- **診断根拠:** 画像上の肝変形・萎縮（他院情報）、血小板減少、PT延長、Alb低下。
- **重症度:** Child-Pugh A / B / C
- **合併症:** 腹水 / 食道静脈瘤 / 肝性脳症 / こむら返り

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 専門医との併診（画像フォロー依頼）。
- 栄養療法（BCAA）、便秘管理（アンモニア対策）、こむら返り対策。

#### 処方 (選択)

**BCAA製剤 (Alb低下時)**

- <span className="med-id">[Rx-Hep-Cir-Liv]</span> リーバクト
- リーバクト配合顆粒 1回1包 1日3回 食後 28日分

**肝性脳症予防 (難治性便秘)**

- <span className="med-id">[Rx-Hep-Cir-Rif]</span> リフキシマ
- リフキシマ錠200mg 1回2錠 1日3回 食後 28日分

**こむら返り対策**

- <span className="med-id">[Rx-Hep-Cir-Car]</span> レボカルニチン
- レボカルニチン塩化物錠300mg 1回1錠 1日3回 毎食後 28日分
- <span className="med-id">[Rx-Kampo-068]</span> 芍薬甘草湯(68)
- ツムラ芍薬甘草湯(68)エキス顆粒 2.5g 1回1包 頓用 10回分

**排便管理 (必須)**

- <span className="med-id">[Rx-GI-Const-Mag]</span> 酸化マグネシウム
- 酸化マグネシウム錠330mg 1回1-2錠 1日3回 毎食後 28日分

#### 指導・計画 (必須)

- **患者指導:** 便秘はアンモニア上昇の最大要因であるため、排便コントロールを徹底するよう指導。

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院には腹部エコーおよびCTがない。肝臓の形状評価、胆石の有無、胆管拡張の確認ができないため、身体所見で異常（黄疸、Murphy徴候、腫瘤）があれば、迷わず画像診断可能な施設へ紹介せよ。
>
> **[FIB-4 Index]** 脂肪肝を見たら必ずFIB-4 indexを算出せよ。スコア2.67以上の場合は肝硬変への進展リスクが高いため、肝臓専門医へ紹介すること。

</div>

> 💡 **Dr.Advice**
>
> **[アンモニア]** 便秘はアンモニア上昇（肝性脳症）の最大要因である。酸化マグネシウム等で「1日2回」の排便を維持するよう厳重に指導せよ。
