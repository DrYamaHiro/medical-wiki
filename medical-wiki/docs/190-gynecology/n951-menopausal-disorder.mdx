<h3 className="soap-s">主訴・ROS</h3>

主訴: ホットフラッシュ / のぼせ / 多汗 / 不眠 / イライラ / 下腹部痛 / 腰痛 / 不正出血 / 帯下異常 / 外陰部痒み / 乳房しこり

**更年期症状**

のぼせ・発汗: 有 / 無, 動悸: なし / あり, 関節痛・手指こわばり: なし / あり
精神: 安定 / 不安 / 抑うつ / 易怒性
睡眠: 良好 / 入眠障害 / 中途覚醒 / 早朝覚醒

**婦人科症状**

下腹部痛: なし / あり, 腰痛: なし / あり, 不正出血: なし / あり
帯下: 増量 / 酒粕状 / 悪臭 / 黄色 / 血性, 外陰部掻痒感: 有 / 無

**乳房**

しこり: なし / あり, 疼痛・発赤・熱感: なし / あり, 授乳中: はい / いいえ

**月経歴・背景**

最終月経: ＿月＿日 順調 / 不順 / 閉経 ＿歳
周期: ＿日, 日数: ＿日, 量: 多 / 普通 / 少
妊娠の可能性: なし / あり / 不明
既往歴: 子宮筋腫 / 内膜症 / 乳癌 / 血栓症

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**腹部診察**

視診: 平坦 / 膨満, 触診: 軟 / 筋性防御 / 腫瘤触知
圧痛: なし / 下腹部正中 / 左右付属器領域
CVA叩打痛: 有 / 無 ；腎盂腎炎除外

**乳房視触診**

皮膚所見: 正常 / 発赤 / 熱感, 腫瘤: なし / あり, 腋窩リンパ節: 腫脹なし / あり

**検査**

女性ホルモンセット(E2, FSH, LH, PRL)
貧血・鉄代謝(Hb, Ferritin)
甲状腺スクリーニング(TSH, FT4) ；更年期との鑑別

<h3 className="soap-ns">診察前対応</h3>

**検査**

- 女性ホルモンセット(E2, FSH, LH, PRL)
- 貧血・鉄代謝(Hb, Ferritin)
- 甲状腺スクリーニング(TSH, FT4)

<h3 className="soap-a">評価</h3>

- **診断根拠:** 閉経周辺期(45-55歳頃)。ホットフラッシュ、多汗、不眠、イライラ、関節痛。
- **重症度(SMIスコア):** ＿点
- **鑑別・除外:** 甲状腺機能異常(動悸・汗)、うつ病、高血圧。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 第一選択として漢方療法。症状が強い場合は禁忌を確認しHRTを検討。プラセンタ療法も選択肢。

#### 処方・処置 (選択)

**漢方療法 (第一選択)**

- ツムラ加味逍遙散(24)エキス顆粒 2.0g 1回1包 1日3回 食前 28日分 *— 精神症状・不定愁訴*
- ツムラ桂枝茯苓丸(25)エキス顆粒 2.5g 1回1包 1日3回 食前 28日分 *— のぼせ・赤ら顔*
- ツムラ当帰芍薬散(23)エキス顆粒 2.5g 1回1包 1日3回 食前 28日分 *— 冷え・貧血*

**HRT (ホルモン補充療法)**

- メノエイドコンビパッチ 1回1枚 週2回 腹部に貼付 *— 子宮あり・配合剤*
- ル・エストロジェル0.06% 1回1プッシュ 1日1回 腕に塗布 *— 子宮なし・E2単独*

**プラセンタ注射 (保険)**

- メルスモン注 1管 皮下注射

#### 指導・計画 (必須)

- **患者指導:** HRTのメリット（骨粗鬆症予防・血管保護）とリスク（乳癌等）を説明。定期的な検診を推奨。
- **再診目安:** 4週間後。

> 💡 **Dr.Advice**
>
> **[妊娠の絶対除外]** 閉経前の女性の「腹痛」「吐き気」「不正出血」を見たら、本人が否定しても医学的には「妊娠（特に異所性妊娠）」を常に疑う必要がある。当院では尿中hCGキットの常備がないため、疑わしければ問診で強く確認するか、近隣薬局で検査薬を購入して確認するよう促す、あるいは直ちに婦人科へ紹介するフローを徹底せよ。
>
> **[内診・エコー不可の限界]** 当院には内診台および経腟エコーがないため、子宮筋腫、卵巣嚢腫、子宮体癌などの器質的疾患の診断は不可能である。「薬で様子を見る」のは、あくまで機能性疾患（更年期・PMS等）と推定される場合に限り、不正出血が続く場合や腹痛が強い場合は、必ず婦人科専門医へ紹介せよ。
>
> **[HRTのリスク説明]** HRTは「乳癌リスクをわずかに上げる（長期投与時）」可能性がある一方、「骨粗鬆症予防」「脂質改善」「血管保護」のメリットがある。「定期的な乳がん検診・子宮がん検診を受けること」を条件に処方せよ。
>
> **[甲状腺との鑑別]** 更年期の「動悸・汗・イライラ」は甲状腺機能亢進症（バセドウ病）と酷似している。また「だるさ・むくみ」は低下症（橋本病）と似る。初診時に一度はTSH/FT4を測定し、見逃しを防ぐのが鉄則である。
