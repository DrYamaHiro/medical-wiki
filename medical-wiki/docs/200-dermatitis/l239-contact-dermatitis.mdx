<h3 className="soap-a">評価</h3>

- **診断根拠:** 原因物質に触れた部位に一致した紅斑、丘疹、小水疱。境界明瞭。
- **原因推測:** 金属 / 植物 / 化粧品 / 湿布 / 毛染め / マスク

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 原因の特定と回避、およびステロイドによる消炎。

#### 処方・アクション (選択)

**外用ステロイド**

- <span className="med-id">[Rx-Der-Ste-S]</span> アンテベート軟膏0.05% 1本 1日2回 塗布 ；体幹・四肢
- <span className="med-id">[Rx-Der-Ste-M]</span> リンデロンVG軟膏0.12% 1本 1日2回 塗布 ；顔面・頸部など

**内服痒み止め**

- <span className="med-id">[Rx-Gen-Ent-Olo]</span> アレロック錠5mg 1回1錠 1日2回 朝夕食後 7日分

#### 指導・計画 (必須)

- **患者指導:** 疑わしい原因物質の徹底回避を指示。
- **再診目安:** 1週間後

> 💡 **Dr.Advice**
>
> **[KOH即日不可の対応]** 当院ではKOH（真菌検査）が外注となるため、その場での白癬確定診断ができない。典型的な湿疹に見えても、ステロイドを塗って悪化する場合は白癬の可能性がある。「もし薬を塗って赤みや痒みが増したら、すぐに中止して再診してください」と必ず伏線を張れ。疑わしければ、初回は抗真菌薬もステロイドも使わず、保湿剤や非ステロイド系（スタデルム等）で検査結果を待つのも賢明な戦略である。
>
> **[FTU (Finger Tip Unit)]** 外用薬の効果が出ない最大の理由は「塗布量不足」である。チューブから「人差し指の先から第一関節まで（約0.5g）」出した量が、「大人の手のひら2枚分」の面積を塗る適量である。「ティッシュが張り付くくらい」たっぷり塗るよう指導せよ。
>
> **[パッチテスト]** 原因不明で繰り返す場合はパッチテストが必要だが、急性期（かぶれている最中）に行うと反応が強く出すぎる（Angry Back）ため、落ち着いてから専門医へ紹介して実施せよ。
