<h3 className="soap-s">主訴・ROS</h3>

主訴: 発疹 / 痒み / 疼痛 / 腫瘤 / 脱毛 ＿日前から出現
経過: 急速拡大 / 再発 / 慢性 / 徐々に拡大
自覚症状: 激痒 / 軽度痒み / 痛み / 無症状 / 赤み / ブツブツ / 水疱
部位: 顔面 / 手指 / 体幹 / 下肢 / 全身

**誘因・背景**

接触歴: 植物 / 金属 / 化粧品 / 虫 / 動物 / 新しい薬 / 新しい洗剤・柔軟剤 / 庭仕事 / ペット
既往歴: アトピー / 蕁麻疹 / 糖尿病 / 喘息 / 鼻炎 / 花粉症

<h3 className="soap-o">所見・検査</h3>

**皮疹性状**

原発疹: 紅斑 / 丘疹 / 水疱 / 膿疱 / 結節 / 膨疹
続発疹: 痂皮 / 鱗屑 / びらん / 潰瘍 / 苔癬化 / 掻破痕 / 色素沈着 / 乾燥
配列・分布: 左右対称 / 片側性 / 帯状 / 孤立 / 集簇 / 汎発 / デルマトーム / 接触部位 / 脂漏部位

**その他**

爪: 混濁 / 肥厚 / 変形, 毛髪: 脱毛 / 牽引痛

**検査**

真菌鏡検(KOH): 外注提出 / 未実施 / 陽性 / 陰性 ※外注
真菌鏡検(KOH法)
ダーモスコピー: 実施 / 未実施

<h3 className="soap-ns">診察前対応</h3>

**検査**

- 真菌鏡検(KOH法)

<h3 className="soap-a">評価</h3>

- **状態:** 肉眼的な紅斑・湿潤は消失。皮膚は平滑化（ツルツル）。
- **コントロール指標:** TARC: ＿ pg/mL ※目標値: 成人&lt;450

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** プロアクティブ療法へ移行（再燃予防）。

#### 処方・アクション (選択)

**非ステロイド外用・JAK阻害**

**ステロイド間歇塗布用**

#### 指導・計画 (必須)

- 患者指導: 保湿剤の継続と、症状がない部位への間歇的なステロイド塗布の意義を説明。
- **再診目安:** 4週間後

> 💡 **Dr.Advice**
>
> **[KOH即日不可の対応]** 当院ではKOH（真菌検査）が外注となるため、その場での白癬確定診断ができない。典型的な湿疹に見えても、ステロイドを塗って悪化する場合は白癬の可能性がある。「もし薬を塗って赤みや痒みが増したら、すぐに中止して再診してください」と必ず伏線を張れ。疑わしければ、初回は抗真菌薬もステロイドも使わず、保湿剤や非ステロイド系（スタデルム等）で検査結果を待つのも賢明な戦略である。
>
> **[FTU (Finger Tip Unit)]** 外用薬の効果が出ない最大の理由は「塗布量不足」である。チューブから「人差し指の先から第一関節まで（約0.5g）」出した量が、「大人の手のひら2枚分」の面積を塗る適量である。「ティッシュが張り付くくらい」たっぷり塗るよう指導せよ。
>
> **[プロアクティブ療法]** 「火事（炎症）は消えたが、炭はまだ熱い」。見た目がきれいでも、週2回（例：土日）薬を塗り続けることで、再発率を劇的に下げられることを説明し、治療継続のモチベーションを高めよ。
