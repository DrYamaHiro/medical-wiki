<h3 className="soap-s">主訴・ROS</h3>

主訴: 労作時息切れ / 動悸 / 胸痛 / 浮腫 / 下肢痛

**心臓症状**

動悸: 脈が飛ぶ / 突然始まる / 常に速い / ドンとする
胸痛: 圧迫感 / 絞扼感 / 鋭痛 / 放散痛:肩・顎
呼吸困難: 労作時 / 安静時 / 夜間発作性 / 起坐呼吸

**末梢血管・下肢**

間欠性跛行: なし / あり: ＿mで出現
下肢症状: 冷感 / 色調変化 / 血管怒張 / 左右差あり
浮腫: なし / 下肢 / 顔面 / 片側 / 両側 / 急激

**リスク・背景**

既往歴: 心筋梗塞 / 弁膜症 / 高血圧 / 糖尿病 / 喫煙
内服薬: 抗凝固薬 / 抗血小板薬 / 利尿薬

<h3 className="soap-o">所見・検査</h3>

バイタル: BP ＿/＿ (左右差: 有 / 無), HR ＿ (整/不整), SpO2 ＿%

**胸部**

頸静脈怒張: 有 / 無
心音: 整 / 不整 / 雑音: 収縮期・拡張期 / III音・IV音
呼吸音: 清明 / 湿性ラ音 / 喘鳴

**末梢・下肢**

下肢浮腫: なし / Pitting / Non-pitting
動脈拍動(足背・後脛骨): 良好 / 減弱 / 消失
Homan's徴候: 陰性 / 陽性、皮膚: 色素沈着 / 潰瘍 / 静脈瘤

**検査**

心電図: 正常洞調律 / ST変化 / 異常Q波 / 不整脈 / LVH
胸部X線: CTR ＿%, 肺鬱血 / 胸水 / 異常なし
採血(外注): BNP, Dダイマー, 电解質, 腎機能

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 労作時の胸部圧迫感、安静で軽快。関連痛（肩・顎）。
- **心電図:** ST低下 / 正常

<h3 className="soap-p">方針・処方</h3>

- カテーテル検査の要否検討のため、循環器内科へ紹介。
- 抗血小板薬、発作時ニトロの処方（紹介までのつなぎ）。

#### 処方・処置 (選択)

**虚血性心疾患セット**

- バイアスピリン錠100mg 1回1錠 1日1回 朝食後 28日分 *— 抗血小板*
- ミオコールスプレー0.3mg 1本 発作時1回1噴霧 *— 発作時*
- ニトロペン舌下錠0.3mg 1回1錠 発作時舌下 10回分 *— 発作時*

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院には心エコー、ホルター心電図、ABI測定機器がない。心雑音、不整脈、脈拍欠損、重症心不全を認めた場合は、詳細評価のため循環器専門医へ紹介せよ。
>
> **[BNP解釈]** 息切れの鑑別にはBNPが有用だが、外注のため結果は後日となる。急性期のトリアージは身体所見とSpO2、X線で行え。
>
> **[ニトロ指導]** 「胸が痛くなったら迷わず使う」「座って使う（起立性低血圧注意）」「5分おきに3回使っても治まらなければ救急車」の3点を徹底指導せよ。
