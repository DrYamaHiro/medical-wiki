<h3 className="soap-s">主訴・ROS</h3>

主訴: 検診異常[脂質・尿酸・肥満] / 痛風発作 / 体重増加

**リスク因子・生活習慣**

家族歴: 心筋梗塞 / 脳卒中 / 脂質異常症 / FH疑い
痛風発作歴: なし / あり: ＿年前, 部位 ＿
生活習慣: 喫煙 / 飲酒 / 運動不足 / 糖質過多 / 揚げ物 / 早食い
アルコール: 毎日 / 週＿回 / ＿合

**代謝・内分泌ROS**

自覚症状: なし / 全身倦怠感 / 口渇 / こむら返り
体重変化: なし / 増加 ＿kg / 減少 ＿kg
既往歴: 糖尿病 / 高血圧 / 脂肪肝 / 尿路結石

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BMI: ＿, 腹囲: ＿cm

**身体所見**

眼瞼黄色腫 有 / 無、角膜輪 有 / 無
アキレス腱肥厚 有 / 無: 幅 ＿mm ；9mm以上でFH濃厚
痛風結節 有 / 無、甲状腺腫大 有 / 無

**検査 (外注)**

全身スクリーニング (脂質・血糖・肝腎・尿酸含む)
甲状腺スクリーニング (TSH/FT3/FT4) ；二次性除外

<h3 className="soap-ns">診察前対応</h3>

**検査 (外注)**

- 全身スクリーニング
- 甲状腺スクリーニング

<h3 className="soap-a">評価</h3>

- **診断根拠:** LDL-C ≧ 140 mg/dL。
- **久山町スコア リスク区分:** 低リスク / 中リスク / 高リスク
- **管理目標値:** LDL &lt;160 / &lt;140 / &lt;120 mg/dL ※二次予防は &lt;70

<h3 className="soap-p">方針・処方</h3>

- 治療戦略 (Step Up/Down):
- 開始: 生活改善3-6ヶ月で未達ならスタチン開始。高リスク群は直ちに開始。
- 増量(Step Up): ストロングスタチンへ変更、最大量へ増量、またはエゼチミブ併用。
- 減量(Step Down): 目標値を連続して20%以上下回り、生活習慣が安定している場合に検討。
- 中止: 超高齢者(75歳以上)のフレイル進行、余命予後不良、横紋筋融解症などの重篤な副作用出現時。

#### 処方 (選択)

**ストロングスタチン**

[ID: Rx-Dys-Sta-Ros2.5] ロスバスタチン2.5mg
- ロスバスタチン錠2.5mg 1回1錠 1日1回 朝食後 28日分
- アトルバスタチン錠5mg 1回1錠 1日1回 朝食後 28日分

**併用薬**

- ゼチーア錠10mg 1回1錠 1日1回 朝食後 28日分

> 💡 **Dr.Advice**
>
> **[二次性脂質異常症の除外]** 「薬を出す前に原因を治せ」。
>
> **[久山町スコアの活用]** 「高いから下げる」ではなく「リスクがあるから下げる」。患者に「今後10年で心筋梗塞や脳卒中になる確率」を数値で示すことで、服薬アドヒアランスが劇的に向上する。アプリやWebの計算ツールを診察室で開き、患者と一緒に画面を見ながら入力することを推奨する。
>
> **[設備制約と紹介基準]** 当院には頸動脈エコーやABIがないため、血管の「詰まり具合」を直接見ることができない。久山町スコア「高リスク」群や、糖尿病・喫煙が重なる症例は、一度は近隣施設へ「動脈硬化評価（頸動脈エコー・ABI・脈波）」を依頼し、ベースラインを把握しておくのが安全である。
>
> **[オンライン初診処方制限]** オンライン診療において、お薬手帳、健診結果、紹介状等で基礎疾患が把握できない初診患者の場合、処方は「7日分を上限」とせよ。
>
> **[75歳以上の一次予防]** 高齢者への一次予防スタチン投与のエビデンスは限定的である。「薬を増やしたくない」という希望があれば無理強いせず、食事療法中心で経過を見るのも妥当な判断である（ポリファーマシー回避）。ただし、二次予防（脳梗塞・心筋梗塞後）は年齢に関わらず必須。
>
> **[女性のLDL]** 閉経後女性はエストロゲン低下によりLDLが急上昇する。男性と同じ基準で即投薬する前に、まずはホルモン変化の影響であることを説明し、リスク因子（喫煙・高血圧）がなければ少し様子を見ても良いケースがある。
