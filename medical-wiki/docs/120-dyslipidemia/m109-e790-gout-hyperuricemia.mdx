<h3 className="soap-s">主訴・ROS</h3>

主訴: 検診異常[脂質・尿酸・肥満] / 痛風発作 / 体重増加

**リスク因子・生活習慣**

家族歴: 心筋梗塞 / 脳卒中 / 脂質異常症 / FH疑い
痛風発作歴: なし / あり: ＿年前, 部位 ＿
生活習慣: 喫煙 / 飲酒 / 運動不足 / 糖質過多 / 揚げ物 / 早食い
アルコール: 毎日 / 週＿回 / ＿合

**代謝・内分泌ROS**

自覚症状: なし / 全身倦怠感 / 口渇 / こむら返り
体重変化: なし / 増加 ＿kg / 減少 ＿kg
既往歴: 糖尿病 / 高血圧 / 脂肪肝 / 尿路結石

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BMI: ＿, 腹囲: ＿cm

**身体所見**

眼瞼黄色腫 有 / 無、角膜輪 有 / 無
アキレス腱肥厚 有 / 無: 幅 ＿mm ；9mm以上でFH濃厚
痛風結節 有 / 無、甲状腺腫大 有 / 無

**検査 (外注)**

全身スクリーニング (脂質・血糖・肝腎・尿酸含む)
甲状腺スクリーニング (TSH/FT3/FT4) ；二次性除外

<h3 className="soap-ns">診察前対応</h3>

**検査 (外注)**

- 全身スクリーニング
- 甲状腺スクリーニング

<h3 className="soap-a">評価</h3>

- **診断根拠:** 尿酸値&gt;7.0mg/dL。痛風発作の既往、関節炎所見。
- **関節炎:** 有 / 無 部位: ＿

<h3 className="soap-p">方針・処方</h3>

- 治療戦略:
- 発作中: 尿酸降下薬を開始しない（変動により発作増悪するため）。
- 寛解期: 発作がおさまってから維持療法開始。
- 目標尿酸値: 6.0mg/dL以下。

#### 処方 (選択)

**痛風発作時**

- コルヒチン錠0.5mg 1回1錠 発作予兆時 頓用 5回分
- ロキソプロフェンNa錠60mg 1回1錠 (疼痛時) 1日3回まで 10回分
- レバミピド錠100mg 1回1錠 (疼痛時) 10回分
- プレドニン錠5mg 1回6錠 1日1回 朝食後 3-5日分 *— NSAIDs無効・腎障害時*

**尿酸降下療法 (維持期)**

- フェブリク錠10mg 1回1錠 1日1回 朝食後 28日分 *— 開始用量*
- フェブリク錠20mg 1回1錠 1日1回 朝食後 28日分 *— 維持用量*
- フェブリク錠40mg 1回1錠 1日1回 朝食後 28日分

**酸性尿改善**

- ウラリット配合散 1回1g 1日3回 毎食後 28日分

> 💡 **Dr.Advice**
>
> **[二次性脂質異常症の除外]** 「薬を出す前に原因を治せ」。
>
> **[久山町スコアの活用]** 「高いから下げる」ではなく「リスクがあるから下げる」。患者に「今後10年で心筋梗塞や脳卒中になる確率」を数値で示すことで、服薬アドヒアランスが劇的に向上する。アプリやWebの計算ツールを診察室で開き、患者と一緒に画面を見ながら入力することを推奨する。
>
> **[設備制約と紹介基準]** 当院には頸動脈エコーやABIがないため、血管の「詰まり具合」を直接見ることができない。久山町スコア「高リスク」群や、糖尿病・喫煙が重なる症例は、一度は近隣施設へ「動脈硬化評価（頸動脈エコー・ABI・脈波）」を依頼し、ベースラインを把握しておくのが安全である。
>
> **[オンライン初診処方制限]** オンライン診療において、お薬手帳、健診結果、紹介状等で基礎疾患が把握できない初診患者の場合、処方は「7日分を上限」とせよ。
>
> **[コルヒチン・カバー]** 尿酸降下薬（フェブリク等）の開始初期は、尿酸結晶が溶け出すことで逆に発作（動員発作）が起きやすくなる。治療開始後3-6ヶ月間は、コルヒチン0.5mg/日を予防的に併用する（Colchicine Cover）か、予兆を感じたらすぐ飲めるよう頓服を持たせることで、ドロップアウトを防げる。
>
> **[発作中の禁忌]** 「痛いから尿酸の薬をください」と来院した患者に、いきなりフェブリクを出してはならない。血中尿酸値が急激に下がると、関節内の結晶が剥がれ落ちて発作が劇的に悪化・長期化する。「今は嵐が過ぎるのを待つ時期です。痛み止めだけでしのぎ、2週間後に数値を下げる治療を始めましょう」と説明せよ。
>
> **[尿路結石の合併]** 痛風患者は尿が酸性に傾いており（酸性尿）、尿酸結石ができやすい。尿検査でpH&lt;6.0の場合は、水分摂取指導に加え、クエン酸製剤（ウラリット等）による尿アルカリ化を検討せよ。ただしpH&gt;7.0にしすぎるとリン酸Ca結石のリスクとなるため調整が必要。
>
> **[尿酸と心血管リスク]** 尿酸は単に関節が痛くなる物質ではなく、血管内皮障害を引き起こす「動脈硬化のリスク因子」である。痛風発作がない高尿酸血症（無症候性）であっても、高血圧やCKDを合併している場合は、将来の心血管イベント抑制を見据えて治療介入する意義がある。
>
> **[フルクトース（果糖）の害]** 「ビールは控えている」という患者でも、清涼飲料水や果物の過剰摂取（果糖）が尿酸値を上げている場合がある。「ジュースや果物を摂りすぎていませんか？」という問診が、生活指導の突破口になることが多い。
