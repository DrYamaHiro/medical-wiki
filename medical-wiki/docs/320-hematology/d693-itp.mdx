<h3 className="soap-a">評価</h3>

- **診断根拠:** 血小板数の単独減少(&lt;10万)。白血球・赤血球は正常。
- **症状:** 無症状 / 点状出血 / 紫斑 / 鼻出血 / 歯肉出血
- **H.Pylori:** 陽性 / 陰性 / 未検
- **除外:** 偽性血小板減少（EDTA依存性）、肝硬変、薬剤性、SLE。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 血小板数 &gt;3万かつ無症状なら経過観察。ピロリ菌陽性なら除菌療法（第一選択）。血小板数 &lt;2-3万または出血症状ありなら専門医へ即時紹介。

#### 処方・処置

**ピロリ除菌 (適応時)**

- <span className="med-id">[Rx-GI-Pylori-1st]</span> ボノサップ(一次除菌)
- ボノサップパック400 1シート 1日2回 朝夕食後 7日分

**検査・紹介**

- <span className="med-id">[Lab-GI-HP-Ab]</span> ピロリ菌検査
- 血清ヘリコバクター・ピロリ抗体 (外注)
- <span className="med-id">[Doc-Gen-Referral]</span> 診療情報提供書 (血液内科へ紹介：重症時・ステロイド導入検討)

#### 指導・計画

- **再診目安:** 2週間後（血小板数確認）。

> 💡 **Dr.Advice**
>
> **[貧血は症状]** 「貧血」は病名ではなく「状態」である。鉄剤を出して終わりにしてはならない。閉経後女性や男性の貧血は、消化管癌（胃癌・大腸癌）のサインである可能性が極めて高いため、必ず内視鏡検査を強く推奨せよ。
>
> **[異食症]** 「氷をガリガリ食べるのが止められない」は鉄欠乏性貧血に特異的な症状である。鉄剤投与で劇的に消失する。
>
> **[偽性血小板減少]** 検診で「血小板が低い」と言われたが全く無症状の場合、採血管内の凝固（EDTA依存性）による検査エラーの可能性がある。再検時も低ければ紹介せよ。
>
> **[ピロリ除菌]** ピロリ菌陽性のITP患者の約半数は、除菌だけで血小板数が回復する。ステロイドや摘脾を行う前に、必ずピロリ菌チェックを行うのが現代のスタンダードである。
