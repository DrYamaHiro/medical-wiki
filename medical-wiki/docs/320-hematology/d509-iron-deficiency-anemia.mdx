<h3 className="soap-s">主訴・ROS</h3>

**貧血症状**

自覚症状: 労作時息切れ / 動悸 / めまい / 立ちくらみ / 易疲労感 / 耳鳴り
特異的症状: 氷食症[Pica] / 爪の変形 / 舌痛 / 味覚障害
神経症状: なし / 手足のしびれ / 歩行時のふらつき ；B12欠乏疑い

**出血・栄養歴**

月経: 順調 / 過多月経 / 凝血塊あり / 期間延長
消化管: なし / 黒色便 / 血便 / 胃切除歴
食事・嗜好: 偏食 / ベジタリアン / アルコール多飲

<h3 className="soap-o">所見・検査</h3>

眼瞼結膜: 正常 / 蒼白
口腔: 舌乳頭萎縮 / 平滑舌 / 口角炎
爪: 正常 / 扁平 / 匙状爪[Koilonychia]

**循環器・神経**

脈拍: ＿ bpm, 心雑音: なし / 収縮期駆出性雑音
神経所見: 異常なし / 振動覚低下 / Romberg徴候陽性

**院内迅速検査**

末梢血一般(CBC)
- Hb ＿ g/dL, MCV ＿ fL, Plt ＿ x10^4/μL
- **判定:** 小球性&lt;80 / 正球性80-100 / 大球性&gt;100

**外注検査**

貧血・鉄代謝
- 血清鉄(Fe), UIBC, フェリチン
ビタミン欠乏性貧血
- ビタミンB12, 葉酸 ；大球性の場合
網状赤血球数

<h3 className="soap-ns">診察前対応</h3>

**院内迅速検査**

- 末梢血一般(CBC)

**外注検査**

- 貧血・鉄代謝
- ビタミン欠乏性貧血

<h3 className="soap-a">評価</h3>

- **診断根拠:** Hb低値、MCV低値(&lt;80)、MCHC低値。フェリチン低値(&lt;12ng/mL)。
- **原因:** 月経過多 / 消化管出血 / 成長期・妊娠 / 偏食 / 術後吸収障害
- **除外:** 慢性疾患に伴う貧血（フェリチン正常～高値）、サラセミア。

<h3 className="soap-p">方針・処方</h3>

- 治療戦略: 鉄剤内服（フェリチン正常化まで継続）。副作用（嘔気・便秘）対策。原因検索（内視鏡・婦人科エコー）の実施・紹介。静注が必要な場合は他院へ紹介。

#### 処方・処置

**鉄剤 (1つ選択)**

- フェロ・グラデュメット錠105mg 1回1錠 1日1回 朝食後 28日分 *— 徐放性・第一選択*
- インクレミンシロップ5% 10mL 1回1包 1日1回 朝食後 28日分 *— 小児・錠剤困難時*
- クエン酸第一鉄Na錠50mg 1回1錠 1日1-2回 食後 28日分 *— 低用量から開始したい時*

**副作用対策 (併用)**

- 酸化マグネシウム錠330mg 1回1錠 1日3回 毎食後 28日分 *— 便秘対策*
- ナウゼリン錠10mg 1回1錠 1日3回 食前 5日分 *— 悪心時*

**紹介**

#### 指導・計画

- **患者指導:** 便が黒くなることを説明。お茶での服用も臨床上は問題ない。
- **再診目安:** 1ヶ月後（Hb上昇確認）。

> 💡 **Dr.Advice**
>
> **[貧血は症状]** 「貧血」は病名ではなく「状態」である。鉄剤を出して終わりにしてはならない。閉経後女性や男性の貧血は、消化管癌（胃癌・大腸癌）のサインである可能性が極めて高いため、必ず内視鏡検査を強く推奨せよ。
>
> **[異食症]** 「氷をガリガリ食べるのが止められない」は鉄欠乏性貧血に特異的な症状である。鉄剤投与で劇的に消失する。
>
> **[原因検索]** 「貧血は病名ではなく状態」である。鉄剤で数値が良くなっても、背後に胃癌や大腸癌が隠れている可能性がある。閉経後女性や男性の貧血は、必ず消化管精査を行え。
>
> **[高齢者の貧血]** 高齢者の鉄欠乏性貧血は「消化管癌」と同義と考えよ。便潜血が陰性でも、貧血があるなら内視鏡（上部・下部）を行う正当な理由になる。
