<h3 className="soap-s">主訴・ROS</h3>

**睡眠関連**

いびき: 指摘あり / 自分の音で起きる / 動画で確認
無呼吸の指摘: あり / なし
日中の過眠: 運転中 / 会議中 / 食後 / テレビ視聴中
ESS(眠気スコア): ＿/24点 (11点以上で病的過眠)
中途覚醒: ＿ 回/晩, 起床時の症状: 頭痛 / 口渇 / 熟眠感欠如

**下肢症状 (RLS疑い)**

脚の不快感: 夕方～夜間に増悪 / じっとしていると悪化 / 動くと楽になる
感覚: ムズムズする / 虫が這うよう / 痛痒い

**背景**

睡眠時間: ＿時間 (就床 ＿:＿ / 起床 ＿:＿)
体重変化: ＿kg増/＿年
既往歴: 高血圧 / 糖尿病 / 心房細動 / 脳卒中
職業: 運転業務 / 高所作業 / 精密作業 ；事故リスク確認

<h3 className="soap-o">所見・検査</h3>

身体計測: 身長 ＿cm, 体重 ＿kg, BMI ＿
頸囲: ＿cm (男性&gt;38cm, 女性&gt;35cmでリスク増)
バイタル: BP: ＿/＿ mmHg

**咽頭所見**

口蓋扁桃肥大: Mackenzie I / II / III / IV
軟口蓋低位: 有 / 無, 舌根沈下: 有 / 無
Mallampati分類: Class I / II / III / IV

**下肢**

静脈瘤: 有 / 無, 貧血眼瞼: 有 / 無

**検査 (外注予定)**

携帯型睡眠時無呼吸検査（簡易検査）
在宅睡眠時無呼吸検査（OCST/フルPSG）

<h3 className="soap-ns">診察前対応</h3>

**検査 (外注予定)**

- 携帯型睡眠時無呼吸検査（簡易検査）
- 在宅睡眠時無呼吸検査（OCST/フルPSG）

<h3 className="soap-a">評価</h3>

- **診断根拠:** URGEの4徴候を満たす。
U (Urge): 脚を動かしたい衝動
R (Rest): 安静時に悪化
G (Gyration): 動かすと軽快
E (Evening): 夕方～夜間に増悪
- **検査:** フェリチン値 ＿ ng/mL &lt;50 は鉄欠乏として治療対象
- **原因:** 特発性 / 鉄欠乏性 / 腎不全 / 薬剤性 / 妊娠

<h3 className="soap-p">方針・処方</h3>

- 治療戦略:
- 原因治療（鉄欠乏なら鉄剤、原因薬の中止）。
- 非薬物療法（カフェイン・アルコール制限、マッサージ）。
- 薬物療法（ドパミンアゴニスト、α2δリガンド）。

#### 処方 (選択)

**鉄欠乏治療 (フェリチン&lt;50)**

- フェロ・グラデュメット錠105mg 1回1錠 1日1回 朝食後 28日分

**RLS治療薬**

- ビ・シフロール錠0.125mg 1回1錠 1日1回 夕食後 28日分 *— ドパミン作動薬*

- ガバペン錠200mg 1回1錠 1日1回 夕食後 28日分 *— α2δリガンド*

- クロナゼパム錠0.5mg 1回1錠 1日1回 就寝前 28日分 *— BZ系・頓用*

> 💡 **Dr.Advice**
>
> **[Mallampati分類]** 開口時に口蓋垂がどれだけ見えるか。Class III（口蓋垂基部のみ）やIV（全く見えない）は気道狭窄を示唆し、SASの予測因子となる。
>
> **[職業運転手]** トラック、バス、タクシー等の運転業務従事者の場合、SASによる居眠り事故は社会的責任を問われる重大事案となる。「仕事を守るために検査が必要です」と説得し、未治療のまま放置させないこと。
>
> **[鉄補充の重要性]** フェリチン&lt;50ng/mLであれば、貧血（Hb低下）がなくても脳内鉄欠乏により症状が出る。鉄剤投与が第一選択となる。
>
> **[Augmentation (症状増悪)]** ドパミン作動薬（ビ・シフロール等）を長期・高用量使用すると、逆に症状が悪化・早期化（Augmentation）することがある。この場合は増量せず、α2δリガンド（ガバペン/レグナイト）への変更を検討せよ。
