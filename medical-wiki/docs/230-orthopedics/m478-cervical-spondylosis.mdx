<h3 className="soap-s">主訴・ROS</h3>

部位: 腰 / 頸 / 肩 / 膝 / 手首 / 足底 右 / 左 / 両側
疼痛スケール(NRS): ＿/10
受傷機転: なし / 重いものを持った / 転倒 / 寝違え / スポーツ
増悪因子: 前屈 / 後屈 / 起立時 / 歩行時 / 夜間 / 起床時
しびれ・放散痛: なし / あり: 上肢・下肢

**Red Flags確認**

排尿・排便障害: なし / あり ；馬尾症状
全身症状: なし / 発熱 / 体重減少 / 癌の既往
安静時・夜間痛: なし / あり ；感染・腫瘍疑い

<h3 className="soap-o">所見・検査</h3>

歩行様式: 正常 / 跛行 / 前傾姿勢
圧痛点: 棘突起 / 脊柱起立筋 / 関節裂隙 / 腱付着部
可動域(ROM): 正常 / 屈曲・伸展制限 / 疼痛制限あり

**神経学的所見**

徒手筋力(MMT): 正常 / 低下: ＿
感覚障害: なし / あり: デルマトーム ＿
腱反射(PTR/ATR): 正常 / 低下 / 亢進

**誘発テスト**

腰椎: SLR - / + ＿°, FNS - / +
頸椎: Spurling - / +, Jackson - / +
その他: Finkelstein - / +, Drop arm - / +

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 頸部～肩甲骨内側の疼痛、こり感。
- **神経根症状:** なし / あり: 上肢放散痛・Spurling陽性
- **脊髄症状:** なし / あり: 巧緻運動障害・歩行障害

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 保存療法（薬物、枕調整、姿勢指導）。脊髄症状（Myelopathy）があれば手術適応の可能性があるため即紹介。

#### 処方・処置

**頸椎症治療セット (併用可)**

- テルネリン錠1mg 1回1錠 1日3回 毎食後 14日分 *— 筋緊張緩和*
- メチコバール錠500μg 1回1錠 1日3回 毎食後 28日分
- ロキソプロフェンNa錠60mg 1回1錠 1日3回 毎食後 14日分
- レバミピド錠100mg 1回1錠 1日3回 毎食後 14日分
- ツムラ葛根湯(1)エキス顆粒 2.5g 1回1包 1日3回 食前 7日分 *— 肩こり*

#### 指導・計画

- **患者指導:** 枕の高さ調整（タオル等で隙間を埋める程度）。
- **再診目安:** 2週間後。

> 💡 **Dr.Advice**
>
> **[Red Flags]** 「安静にしていても痛い（内臓痛・腫瘍・化膿性脊椎炎）」「熱がある」「尿が出にくい（馬尾症候群）」は緊急事態である。当院にはMRIがないため、これらの徴候があれば鎮痛剤で様子を見ず、直ちに専門病院へ紹介せよ。
>
> **[画像診断の適応]** Red Flagsがない単純な急性腰痛（ぎっくり腰）や寝違えであれば、初診時のX線撮影は医学的に必須ではない（治療方針が変わらないため）。患者が「骨折がないか心配」している場合や、高齢者（圧迫骨折リスク）の場合に撮影を行え。
>
> **[脊髄症のサイン]** 「手袋をしたような感覚」「階段が怖い」「箸が使いにくい」は脊髄圧迫の危険信号（Myelopathy）。マッサージや牽引は禁忌となりうるため、MRI精査を急げ。
>
> **[枕]** 「高い枕」は頸椎の前屈を強要し症状を悪化させる。「玄関マットやバスタオルを折り畳んで、首の隙間を埋める程度の高さ」を推奨せよ。
