<h3 className="soap-s">主訴・ROS</h3>

部位: 腰 / 頸 / 肩 / 膝 / 手首 / 足底 右 / 左 / 両側
疼痛スケール(NRS): ＿/10
受傷機転: なし / 重いものを持った / 転倒 / 寝違え / スポーツ
増悪因子: 前屈 / 後屈 / 起立時 / 歩行時 / 夜間 / 起床時
しびれ・放散痛: なし / あり: 上肢・下肢

**Red Flags確認**

排尿・排便障害: なし / あり ；馬尾症状
全身症状: なし / 発熱 / 体重減少 / 癌の既往
安静時・夜間痛: なし / あり ；感染・腫瘍疑い

<h3 className="soap-o">所見・検査</h3>

歩行様式: 正常 / 跛行 / 前傾姿勢
圧痛点: 棘突起 / 脊柱起立筋 / 関節裂隙 / 腱付着部
可動域(ROM): 正常 / 屈曲・伸展制限 / 疼痛制限あり

**神経学的所見**

徒手筋力(MMT): 正常 / 低下: ＿
感覚障害: なし / あり: デルマトーム ＿
腱反射(PTR/ATR): 正常 / 低下 / 亢進

**誘発テスト**

腰椎: SLR - / + ＿°, FNS - / +
頸椎: Spurling - / +, Jackson - / +
その他: Finkelstein - / +, Drop arm - / +

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 腰部の運動時痛、脊柱起立筋の圧痛。下肢への放散痛や麻痺なし。
- **病態:** 筋・筋膜性 / 椎間関節性 / 仙腸関節性
- **除外:** Red Flags（発熱、癌既往、骨折リスク、膀胱直腸障害）なし。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 安静は推奨しない（動ける範囲で動く）。保存療法（鎮痛剤、筋弛緩薬、湿布）。コルセットは疼痛緩和目的に限定。

#### 処方・処置

**鎮痛薬 (選択)**

- ロキソプロフェンNa錠60mg 1回1錠 1日3回 毎食後 7日分
- レバミピド錠100mg 1回1錠 1日3回 毎食後 7日分
- セレコキシブ錠200mg 1回1錠 1日2回 朝夕食後 7日分 *— 胃腸障害時など*
- ツムラ芍薬甘草湯(68)エキス顆粒 2.5g 1回1包 頓用 10回分 *— 筋痙攣・ぎっくり腰*

**筋弛緩薬 (併用)**

- ミオナール錠50mg 1回1錠 1日3回 毎食後 7日分

**外用薬**

- ロキソプロフェンNaテープ100mg 7枚入 2袋 1日1回 患部に貼付

#### 指導・計画

- **患者指導:** 安静にしすぎず、痛みのない範囲で動くよう指導。入浴は急性期(2日)は避ける。
- **再診目安:** 1週間後。

> 💡 **Dr.Advice**
>
> **[Red Flags]** 「安静にしていても痛い（内臓痛・腫瘍・化膿性脊椎炎）」「熱がある」「尿が出にくい（馬尾症候群）」は緊急事態である。当院にはMRIがないため、これらの徴候があれば鎮痛剤で様子を見ず、直ちに専門病院へ紹介せよ。
>
> **[画像診断の適応]** Red Flagsがない単純な急性腰痛（ぎっくり腰）や寝違えであれば、初診時のX線撮影は医学的に必須ではない（治療方針が変わらないため）。患者が「骨折がないか心配」している場合や、高齢者（圧迫骨折リスク）の場合に撮影を行え。
>
> **[湿布の枚数制限]** 湿布薬の上限は原則「1処方につき計70枚（10袋）」まで。超える場合は医学的理由の記載が必要となる。
>
> **[入浴]** 急性期（受傷直後～2日）は炎症を助長するため長湯は避ける。慢性期は温めて血流を良くすることが推奨される。
