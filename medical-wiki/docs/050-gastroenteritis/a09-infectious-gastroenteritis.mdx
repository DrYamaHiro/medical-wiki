<h3 className="soap-s">主訴・ROS</h3>

主訴: 腹痛 / 悪心・嘔吐 / 下痢 / 便秘 / 血便 / 吐血 / 心窩部不快感

**消化器系**

腹痛部位: 心窩部 / 右季肋部 / 臍周囲 / 右下腹部 / 左下腹部 / 全体
腹痛性状: 鈍痛 / 鋭痛 / 波がある / 持続的
便性状: 水様 / 泥状 / 粘血便 / 黒色便 / 兎糞状, 回数: ＿ 回/日
食事関連: 食前 / 食後 / 空腹時 / 夜間

**全身・背景**

発熱: 有 / 無, 倦怠感: 有 / 無
食事摂取: 可能 / 不可: 水分のみ / 絶食中
生食歴: 牡蠣 / 鶏肉 / 焼肉 / 刺身 / なし ＿日前
周囲の流行: 家族 / 職場 / 学校
ストレス: 有 / 無, アルコール: 多飲 / 機会飲酒
内服薬: NSAIDs / ステロイド / 抗血小板薬

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**全身**

脱水所見: 口腔乾燥 有 / 無, ツルゴール 正常 / 低下
眼瞼結膜: 貧血なし / 蒼白

**腹部**

視診: 平坦 / 膨満
聴診: 腸雑音 正常 / 亢進 / 減弱 / 消失 / 金属音
触診: 軟 / 筋性防御 / 板状硬, 腫瘤 有 / 無
圧痛点: 心窩部 / McBurney / Murphy / 全体 / なし
反跳痛: 有 / 無, CVA叩打痛: 有 / 無

**直腸診（必要な場合）**

タール便 有 / 無

**除外項目(Negative ROS)**

板状硬無, 反跳痛無, 拍動性腫瘤無

<h3 className="soap-a">評価</h3>

- **診断根拠:** 急性の嘔吐、下痢、腹痛。周囲の流行や生食歴。
- **重症度:** 脱水の有無で判断。
- **鑑別:** 虫垂炎、胆嚢炎（※当院ではエコー不可のため、身体所見で疑えば紹介）。

<h3 className="soap-p">方針・処方</h3>

- 対症療法が基本。整腸剤、制吐剤。
- 止痢薬（下痢止め）は毒素排出を遅らせるため原則使用しない。
- 脱水補正（経口補水液または点滴）。

#### 処方・アクション (選択)

**胃腸炎・基本セット**

- ビオフェルミン錠 1回1錠 1日3回 毎食後 5日分 *— 整腸剤*
- ナウゼリン錠10mg 1回1錠 1日3回 食前 5日分 *— 吐き気止め*
- アセトアミノフェン錠500mg 1回1錠 1日3回 毎食後 5日分 *— 腹痛時*
- ツムラ五苓散(17)エキス顆粒 2.5g 1回1包 1日3回 食前 5日分 *— 水様性下痢・嘔気*

**点滴（脱水時）**

- ラクテック注 250mL 点滴静注

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院にはCT、エコー、上部消化管内視鏡がない。吐血、黒色便、激しい腹痛、腹膜刺激症状がある場合は、診断がつかなくても直ちに専門病院へ紹介せよ。
>
> **[トリアージ]** 高齢者の心窩部痛はACS（心筋梗塞）の可能性があるため、必ず心電図を確認せよ。
>
> **[食事指導]** 「絶食」は必ずしも必要ない。水分が摂れるなら、うどんやお粥など消化の良いものを少量ずつ摂取させることで回復が早まる。
