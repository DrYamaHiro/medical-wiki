<h3 className="soap-s">主訴・ROS</h3>

主訴: 腹痛 / 悪心・嘔吐 / 下痢 / 便秘 / 血便 / 吐血 / 心窩部不快感

**消化器系**

腹痛部位: 心窩部 / 右季肋部 / 臍周囲 / 右下腹部 / 左下腹部 / 全体
腹痛性状: 鈍痛 / 鋭痛 / 波がある / 持続的
便性状: 水様 / 泥状 / 粘血便 / 黒色便 / 兎糞状, 回数: ＿ 回/日
食事関連: 食前 / 食後 / 空腹時 / 夜間

**全身・背景**

発熱: 有 / 無, 倦怠感: 有 / 無
食事摂取: 可能 / 不可: 水分のみ / 絶食中
生食歴: 牡蠣 / 鶏肉 / 焼肉 / 刺身 / なし ＿日前
周囲の流行: 家族 / 職場 / 学校
ストレス: 有 / 無, アルコール: 多飲 / 機会飲酒
内服薬: NSAIDs / ステロイド / 抗血小板薬

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**全身**

脱水所見: 口腔乾燥 有 / 無, ツルゴール 正常 / 低下
眼瞼結膜: 貧血なし / 蒼白

**腹部**

視診: 平坦 / 膨満
聴診: 腸雑音 正常 / 亢進 / 減弱 / 消失 / 金属音
触診: 軟 / 筋性防御 / 板状硬, 腫瘤 有 / 無
圧痛点: 心窩部 / McBurney / Murphy / 全体 / なし
反跳痛: 有 / 無, CVA叩打痛: 有 / 無

**直腸診（必要な場合）**

タール便 有 / 無

**除外項目(Negative ROS)**

板状硬無, 反跳痛無, 拍動性腫瘤無

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 急激な心窩部痛、悪心・嘔吐。
- **原因:** アルコール / NSAIDs / ストレス / アニサキス疑い
- **除外:** 急性膵炎（アミラーゼ採血）、胆石（エコー不可のため身体所見で疑い紹介）、ACS。

<h3 className="soap-p">方針・処方</h3>

- 胃の安静（消化の良い食事）。
- 酸分泌抑制薬と粘膜保護薬の投与。
- アニサキス疑い（生魚摂取歴＋激痛）なら即紹介（当院内視鏡なし）。

#### 処方・アクション

**急性胃炎治療セット**

- タケキャブ錠20mg 1回1錠 1日1回 朝食後 14日分
- ファモチジン錠20mg 1回1錠 1日2回 朝夕食後 7日分
- レバミピド錠100mg 1回1錠 1日3回 毎食後 14日分
- ナウゼリン錠10mg 1回1錠 1日3回 食前 5日分

**アニサキス疑い**

- **紹介先:** 消化器内科（緊急内視鏡摘出依頼）

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院にはCT、エコー、上部消化管内視鏡がない。吐血、黒色便、激しい腹痛、腹膜刺激症状がある場合は、診断がつかなくても直ちに専門病院へ紹介せよ。
>
> **[トリアージ]** 高齢者の心窩部痛はACS（心筋梗塞）の可能性があるため、必ず心電図を確認せよ。
>
> **[アニサキス]** 「刺身を食べて数時間後の激痛」はアニサキスを強く疑う。当院では摘出できないため、診断がついた時点で即座に内視鏡可能な施設へ送れ。
