<h3 className="soap-s">主訴・ROS</h3>

主訴: 腹痛 / 悪心・嘔吐 / 下痢 / 便秘 / 血便 / 吐血 / 心窩部不快感

**消化器系**

腹痛部位: 心窩部 / 右季肋部 / 臍周囲 / 右下腹部 / 左下腹部 / 全体
腹痛性状: 鈍痛 / 鋭痛 / 波がある / 持続的
便性状: 水様 / 泥状 / 粘血便 / 黒色便 / 兎糞状, 回数: ＿ 回/日
食事関連: 食前 / 食後 / 空腹時 / 夜間

**全身・背景**

発熱: 有 / 無, 倦怠感: 有 / 無
食事摂取: 可能 / 不可: 水分のみ / 絶食中
生食歴: 牡蠣 / 鶏肉 / 焼肉 / 刺身 / なし ＿日前
周囲の流行: 家族 / 職場 / 学校
ストレス: 有 / 無, アルコール: 多飲 / 機会飲酒
内服薬: NSAIDs / ステロイド / 抗血小板薬

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

**全身**

脱水所見: 口腔乾燥 有 / 無, ツルゴール 正常 / 低下
眼瞼結膜: 貧血なし / 蒼白

**腹部**

視診: 平坦 / 膨満
聴診: 腸雑音 正常 / 亢進 / 減弱 / 消失 / 金属音
触診: 軟 / 筋性防御 / 板状硬, 腫瘤 有 / 無
圧痛点: 心窩部 / McBurney / Murphy / 全体 / なし
反跳痛: 有 / 無, CVA叩打痛: 有 / 無

**直腸診（必要な場合）**

タール便 有 / 無

**除外項目(Negative ROS)**

板状硬無, 反跳痛無, 拍動性腫瘤無

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 汚染食品摂取後の急激な嘔吐・下痢・腹痛・発熱。
- **原因菌:** サルモネラ（卵・肉）、腸炎ビブリオ（魚介）、黄色ブドウ球菌（おにぎり・弁当）。

<h3 className="soap-p">方針・処方</h3>

- 原則は対症療法（排菌を促すため止痢薬は避ける）。
- 重症例（高熱、血便）では抗菌薬投与を検討するが、ニューキノロン（レボフロキサシン）の使用は慎重に。

#### 処方・アクション

**対症療法**

- ビオフェルミンR錠 1回1錠 1日3回 毎食後 5日分
- アセトアミノフェン錠500mg 1回1錠 1日3回 毎食後 5日分

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院にはCT、エコー、上部消化管内視鏡がない。吐血、黒色便、激しい腹痛、腹膜刺激症状がある場合は、診断がつかなくても直ちに専門病院へ紹介せよ。
>
> **[トリアージ]** 高齢者の心窩部痛はACS（心筋梗塞）の可能性があるため、必ず心電図を確認せよ。
>
> **[止痢薬禁忌]** 感染性胃腸炎に対し、強力な下痢止め（ロペミン等）を使用すると、毒素が腸内に貯留し重症化する恐れがあるため、原則使用禁止とせよ。
>
> **[抗菌薬の害]** サルモネラ腸炎に抗菌薬を投与すると、排菌期間が延びる（保菌者化）リスクがある。免疫不全や菌血症リスクがない限り、漫然と投与しないこと。
