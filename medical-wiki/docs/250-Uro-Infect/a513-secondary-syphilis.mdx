<h3 className="soap-s">主訴・ROS</h3>

**排尿症状**

排尿時痛: なし / 排尿終わり / 最初から / 全体
頻尿: なし / あり: ＿回/日, 残尿感: なし / あり, 尿意切迫感: なし / あり
尿の性状: 混濁 / 血尿 / 悪臭

**局所症状**

尿道分泌物: なし / 膿性[ドロっとした黄色] / 漿液性[サラサラ] / 下着の汚れ
疼痛: 下腹部 / 側腹部 / 陰部 / 睾丸 / 咽頭痛

**経過・背景**

症状出現: ＿日前から
性交渉歴: なし / あり: ＿日前・パートナー/風俗
最終月経: ＿ 妊娠の可能性: 有 / 無

<h3 className="soap-o">所見・検査</h3>

バイタル: BT: ＿℃

**腹部・局所**

圧痛: 下腹部 / CVA叩打痛[右・左] / なし
外陰部: 異常なし / 発赤 / 腫脹 / 分泌物 / 水疱 / 潰瘍

**院内迅速検査**

尿検査・尿沈渣
- **定性:** 蛋白(＿) 糖(＿) 潜血(＿) 白血球(＿) 亜硝酸塩(＿)
- **沈渣:** RBC(＿) WBC(＿) 細菌(＿)

**外注検査**

尿培養 (※複雑性・腎盂腎炎疑いのみ)
性感染症(尿) ；淋菌・クラミジアPCR
性感染症(咽頭) ；うがい液PCR

<h3 className="soap-ns">診察前対応</h3>

**院内迅速検査**

- 尿定性・尿沈渣

**外注検査**

- 尿培養
- 性感染症(尿)
- 性感染症(咽頭)

<h3 className="soap-a">評価</h3>

- **診断根拠:** 感染から3ヶ月～3年。バラ疹、手掌・足底紅斑（梅毒性乾癬）、扁平コンジローマ、梅毒性脱毛。RPR高値。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 第1期と同様、早期梅毒として治療（ステルイズ単回）。皮疹は治療後一時的に増悪（JH反応）後に消退することを説明。

#### 処方・処置

**第一選択 (筋注)**

- ステルイズ水性懸濁筋注240万単位 1シリンジ 臀部筋肉注射

**代替 (内服)**

- サワシリンカプセル250mg 1回2カプセル 1日3回 毎食後 28日分

#### 指導・計画

- **届出:** 診断後7日以内に発生届提出。

> 💡 **Dr.Advice**
>
> **[排尿痛の鑑別]** 排尿終末時痛・残尿感・頻尿なら「膀胱炎」、排尿初期痛・尿道分泌物なら「尿道炎」、発熱・背部叩打痛(CVA)なら「腎盂腎炎」を疑う。
>
> **[設備制約]** 当院には腹部エコーおよびCTがないため、結石のサイズ同定や腎膿瘍の評価はできない。高熱、激痛、背部痛を伴う場合は、診断がつかなくても早期に泌尿器科または総合病院へ紹介せよ。
>
> **[STD検査]** 淋菌・クラミジアは「尿（初尿）」で検査する。膀胱炎疑いの「中間尿」とは採尿タイミングが異なるため、指示出しの際は看護師へ明確に伝えること。
>
> **[梅毒届出]** 梅毒は全数把握疾患（5類感染症）である。診断後7日以内に保健所へ発生届を提出せよ。
>
> **[The Great Imitator]** バラ疹は「薬疹」「ウイルス性発疹」「アトピー」等と誤診されやすい。「手足の裏に発疹があるか」を確認し、あれば梅毒を強く疑え。
>
> **[感染力最強]** 第2期の皮疹（特に扁平コンジローマや粘膜疹）には大量のトレポネーマが含まれており、感染力が極めて高い。素手での接触は厳禁。
