<h3 className="soap-s">主訴・ROS</h3>

**めまい (TiTrATE)**

Timing (発症・持続): 突然 / 徐々 / 秒単位 / 分単位 / 時間単位 / 一日中
Triggers (誘因): 頭位変換 / 起立 / ストレス / 特になし
And
Targeted Exam (随伴症状): 難聴、耳鳴り、耳閉感、頭痛、しびれ
性状: 回転性 / 浮動性 / 前失神 / 立ちくらみ

**末梢神経・しびれ**

部位: 手袋靴下型 / デルマトーム / 多発単神経
性状: ジンジン / ピリピリ / 感覚脱失 / 筋力低下

**背景**

既往歴: 糖尿病、高血圧、脂質異常症、癌（化学療法歴）、飲酒

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿ (左右差・起立試験), HR: ＿, SpO2: ＿%

**神経所見 (中枢性除外)**

眼振: 水平回旋混合 / 垂直 / 方向交代性 / 注視方向性 / なし
小脳症状: 指鼻試験 正常 / 測定障害, 体幹失調 起立歩行可 / 不可
脳神経: 顔面神経麻痺、構語障害、複視
HINTS: Head Impulse 正常 / 異常, Nystagmus, Test of Skew

**耳**

聴力検査: 正常 / 感音難聴 / 低音障害
鼓膜所見: 正常 / 内陥 / 混濁

**誘発テスト**

Dix-Hallpike / Supine Roll test: 陽性 / 陰性

<h3 className="soap-ns">診察前対応</h3>

*（データなし）*

<h3 className="soap-a">評価</h3>

- **診断根拠:** 先行する感冒症状の後、突発する激しい回転性めまいが「数日間」持続。難聴・耳鳴りなし。
- **所見:** 患側へのHead Impulse Test陽性（異常）。健側向きの水平回旋混合性眼振。
- **除外:** 小脳梗塞（HINTSで中枢性を否定）。

<h3 className="soap-p">方針・処方</h3>

- 急性期（～3日）は鎮静・制吐・補液でしのぐ。
- ステロイド投与（発症早期なら前庭機能回復の可能性あり）。
- 早期離床とリハビリテーション（寝たきりは回復を遅らせる）。

#### 処方・処置 (選択)

**急性期治療**

- プレドニン錠5mg 1回6錠 1日1回 朝食後 3日分 *— 漸減*
- ナウゼリン錠10mg 1回1錠 1日3回 食前 5日分 *— 制吐剤*
- ラクテック注 250mL 点滴静注 *— 嘔吐による脱水補正*
- プリンペラン注射液10mg 1管 静脈内注射 ；嘔気時

> 💡 **Dr.Advice**
>
> **[難聴の有無]** 「聞こえはおかしくないですか？」と執拗に確認せよ。めまいに難聴を伴う場合は「突発性難聴（めまい伴う）」または「AICA領域の脳梗塞」の可能性があり、治療方針や緊急度が全く異なる。
