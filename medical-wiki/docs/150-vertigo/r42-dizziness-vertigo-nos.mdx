Timing (発症・持続): 突然 / 徐々 / 秒単位 / 分単位 / 時間単位 / 一日中
Triggers (誘因): 頭位変換 / 起立 / ストレス / 特になし
And
Targeted Exam (随伴症状): 難聴、耳鳴り、耳閉感、頭痛、しびれ
性状: 回転性 / 浮動性 / 前失神 / 立ちくらみ

**末梢神経・しびれ**

部位: 手袋靴下型 / デルマトーム / 多発単神経
性状: ジンジン / ピリピリ / 感覚脱失 / 筋力低下

**背景**

既往歴: 糖尿病、高血圧、脂質異常症、癌（化学療法歴）、飲酒

<h3 className="soap-o">所見・検査</h3>

バイタル: BP: ＿/＿ (左右差・起立試験), HR: ＿, SpO2: ＿%

**神経所見 (中枢性除外)**

眼振: 水平回旋混合 / 垂直 / 方向交代性 / 注視方向性 / なし
小脳症状: 指鼻試験 正常 / 測定障害, 体幹失調 起立歩行可 / 不可
脳神経: 顔面神経麻痺、構語障害、複視
HINTS: Head Impulse 正常 / 異常, Nystagmus, Test of Skew

**耳**

聴力検査: 正常 / 感音難聴 / 低音障害
鼓膜所見: 正常 / 内陥 / 混濁

**誘発テスト**

Dix-Hallpike / Supine Roll test: 陽性 / 陰性

<h3 className="soap-a">評価</h3>

- **診断根拠:** めまい・ふらつきを訴えるが、眼振や神経所見に乏しく、特異的疾患の基準を満たさないもの。
- **除外:** 脳血管障害、不整脈、起立性低血圧、貧血、薬剤性（降圧薬・睡眠薬）。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 対症療法 / 漢方療法 / 経過観察
- PPPD（持続性知覚性姿勢誘発めまい）の可能性を考慮。

#### 処方 (選択)

**対症療法**

- <span className="med-id">[Rx-Neuro-Vert-Ceph]</span> セファドール
- セファドール錠25mg 1回1錠 1日3回 毎食後 7日分
- <span className="med-id">[Rx-Neuro-Vert-Beta]</span> メリスロン
- メリスロン錠6mg 1回2錠 1日3回 毎食後 7日分
- <span className="med-id">[Rx-Kampo-039]</span> 苓桂朮甘湯(39)
- ツムラ苓桂朮甘湯(39)エキス顆粒 2.5g 1回1包 1日3回 食前 7日分 *— 立ちくらみ・ふらつき*
- <span className="med-id">[Rx-Kampo-030]</span> 真武湯(30)
- ツムラ真武湯(30)エキス顆粒 2.5g 1回1包 1日3回 食前 7日分 *— 高齢者の冷え・浮動感*

> **[PPPD]** 3ヶ月以上続く浮動性めまいで、器質的異常がない場合はPPPD（心因性とは異なる機能性疾患）の可能性がある。「脳が揺れに対して過敏になっている状態」と説明し、SSRI（ジェイゾロフト等）や認知行動療法が有効な場合があるため、難治例は専門医へ。
