月齢: ＿ヶ月/歳 1M / 3-4M / 6-7M / 9-10M / 1歳半 / 3歳
出生歴: ＿週 ＿g分娩 経腟 / 帝切、合併症 無 / 有

心配事: 発達 / 体重 / 皮膚 / 陰部 / 臍 / 予防接種

<h3 className="soap-o">所見・検査</h3>

**身体計測**

身長: ＿cm ( SD), 体重: ＿kg ( SD)
頭囲: ＿cm ( SD), 胸囲: ＿cm ( SD)
カウプ指数: ＿ 15-18正常 / &gt;20肥満傾向 / &lt;13やせ
成長曲線: 曲線に沿っている / 逸脱 / 横ばい

**診察**

大泉門: 開 / 閉 / 膨隆 / 陥凹、赤色反射: 有 / 無
口腔: 歯数 ＿本 / 鵞口瘡 / 上皮真珠
心音・呼吸音: 異常なし、腹部腫瘤: 無
股関節開排: 制限なし / あり・クリック音
外陰部: 精巣下降 / 陰唇癒合 / 鼠径ヘルニア

**発達確認**

追視・あやし笑い・定頸 (3-4M)
寝返り・お座り・持ち替え (6-7M)
つかまり立ち・ハイハイ・パラシュート反射 (9-10M)
独歩・有為語・指差し・積み木 (1歳半)

<h3 className="soap-a">評価</h3>

- 発育評価：順調 / 要経過観察 / 精査必要
- 発達評価：月齢相当 / 境界域 / 遅滞疑い

<h3 className="soap-p">方針・処方</h3>

- 栄養・育児指導。
- 事故防止指導（月齢に合わせた具体例）。
- 予防接種スケジュールの確認と予約。
- 次回の健診時期の提示。

**文書・オーダー**

- ヘパリン類似物質ローション0.3% 1本 1日2回 塗布 *— スキンケア指導時*

**紹介**

- 整形外科へ紹介（股関節脱臼疑い・開排制限）
- 小児外科・泌尿器科へ紹介（停留精巣・ヘルニア）
- 小児循環器科へ紹介（心雑音精査）

> 💡 **Dr.Advice**
>
> **[成長曲線]** 「平均より小さい」こと自体は問題ではない。「成長曲線（パーセンタイル枠）に沿って増えているか」が重要。曲線から下方に逸脱していく（Cross down）場合は、栄養不足や器質的疾患を疑え。
>
> **[母親のメンタル]** 子供だけでなく、母親の表情や言動（産後うつ、育児ノイローゼの兆候）にも注意を払い、必要なら地域保健師と連携せよ。
>
> **[安全確認]** 肥満児や痩せ型児では、年齢/体重別セットをそのまま適用すると過量・過小投与になるリスクがある。必ず mg/kg での用量確認を行え。
>
> **[股関節脱臼]** 3-4ヶ月健診で最も見逃してはならない疾患。「開排制限（開きが硬い）」「大腿皮膚溝の非対称（しわの左右差）」があれば、躊躇なく整形外科へ紹介せよ。
>
> **[停留精巣]** 「お風呂に入ると降りてくる（移動性精巣）」なら経過観察可だが、常に陰嚢内に触れない場合は手術適応（1歳前後～）となるため、泌尿器科へ紹介する。
>
> **[臍ヘルニア]** 圧迫療法（綿球固定）で早期治癒が期待できる。1-2ヶ月健診で発見したら積極的に圧迫を指導せよ（皮膚トラブルに注意）。
