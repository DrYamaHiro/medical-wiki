<h3 className="soap-s">主訴・ROS</h3>

主訴: 痒み / 痛み / 水疱 / 膿疱 / しこり / イボ
部位: 足指 / 爪 / 口唇 / 体幹 / 陰部 / ＿
経過: ＿日前から出現, 急速に拡大 / 繰り返している / 再発

**皮膚・アレルギー**

自覚症状: 激痒 / 軽度痒み / 痛み / 無症状
既往・背景: 糖尿病 / アトピー / 免疫抑制 / 帯状疱疹 / なし
接触歴: 家族に水虫 / プール / ジム / 性交渉 / 動物 / なし

<h3 className="soap-o">所見・検査</h3>

**皮疹性状**

原発疹: 趾間浸軟 / 小水疱 / 角化 / 膿疱 / 結節 / 膨疹
続発疹: 痂皮 / びらん / 潰瘍 / 鱗屑
分布・配列: 帯状配列 / 集簇性 / 左右対称 / 孤立 / 汎発
局所所見: 圧痛あり / 波動あり / 中心臍窩あり

**検査**

真菌鏡検(KOH法) (外注提出) ；当院即日不可
細菌培養同定・感受性 (外注提出)
単純ヘルペスウイルス抗原検査 (迅速)
水痘帯状疱疹ウイルス抗原検査 (迅速)

<h3 className="soap-ns">診察前対応</h3>

**検査**

- 真菌鏡検(KOH法)
- 細菌培養同定・感受性
- 単純ヘルペスウイルス抗原定性
- 水痘帯状疱疹ウイルス抗原定性

<h3 className="soap-a">評価</h3>

- **診断根拠:** 境界不明瞭な紅斑、熱感、腫脹、圧痛。発熱。
- **重症度:** 軽症 / 中等症:全身症状あり / 重症:壊死性筋膜炎疑い
- **侵入門:** 足白癬 / 外傷 / 潰瘍 / 虫刺され / なし

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 適切な抗菌薬投与と局所の挙上・冷却。

#### 処方・アクション (選択)

**抗菌薬・鎮痛**

- ロキソプロフェンNa錠60mg 1回1錠 1日3回 毎食後 5日分
- レバミピド錠100mg 1回1錠 1日3回 毎食後 5日分

#### 指導・計画 (必須)

- **患者指導:** 患部を心臓より高く保ち、冷却すること。悪化時は直ちに再診。
- **再診目安:** 2-3日後（効果判定）。

> 💡 **Dr.Advice**
>
> **[KOH即日不可の対応]** 当院ではKOH（真菌検査）が外注となるため、その場での白癬確定診断ができない。典型的な湿疹に見えても、ステロイドを塗って悪化する場合は白癬の可能性がある。「もし薬を塗って赤みや痒みが増したら、すぐに中止して再診してください」と必ず伏線を張れ。疑わしければ、初回は抗真菌薬もステロイドも使わず、保湿剤や非ステロイド系（スタデルム等）で検査結果を待つのも賢明な戦略である。
>
> **[細菌培養の重要性]** 蜂窩織炎や化膿性粉瘤では、切開排膿時や抗生剤開始前に必ず培養を採取せよ。経験的治療が無効だった場合、この培養結果が「次の手（耐性菌対応）」を決める命綱となる。
>
> **[壊死性筋膜炎の除外]** 「紫斑・水疱形成」「激痛（または知覚麻痺）」「握雪感（ガス産生）」「ショックバイタル」は、緊急手術が必要な壊死性筋膜炎のサインである。1時間単位で進行するため、疑わしければ直ちに救急搬送せよ。
>
> **[水虫の治療]** 足白癬の亀裂から菌が入ることが多い。しかし、蜂窩織炎の急性期に水虫薬を塗るとかぶれやすく、紛らわしい。「赤みが引いてから水虫の治療を始めましょう」と指導せよ。
