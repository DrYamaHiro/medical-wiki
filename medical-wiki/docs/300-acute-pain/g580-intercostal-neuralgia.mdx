<h3 className="soap-s">主訴・ROS</h3>

**痛みの分析 (OPQRST)**

Onset (発症様式): 突然 / 急性 / 徐々 / ある動作で
Palliative/Provocative (増悪・寛解): 体動 / 食事 / 深呼吸 / 安静時
Quality (性状): 締め付けられる / 裂けるよう / 鈍痛 / 差し込む
Region (部位): 胸部 / 心窩部 / 右季肋部 / 右下腹部 / 背部
Severity (強度): 冷や汗をかく激痛 / 会話可能 / 波がある
Time (経過): ＿分/時間 前から持続

**随伴症状**

冷汗 / 嘔気・嘔吐 / 呼吸困難 / 意識消失 / 下血 / なし

**リスク因子**

高血圧 / 糖尿病 / 脂質異常症 / 喫煙 / 心疾患の既往 / なし

<h3 className="soap-o">所見・検査</h3>

バイタル: BP ＿/＿ (左右差: 有 / 無), HR ＿, SpO2 ＿%, BT ＿℃
顔貌: 苦悶様 / 蒼白 / 冷汗 / 安定

**身体所見**

胸部: 異常なし / 心雑音 / 呼吸音異常 / 頸静脈怒張
腹部: 平坦 / 膨満, 腹膜刺激症状 なし / 筋性防御 / 反跳痛
圧痛点: なし / McBurney / Murphy / 全体
CVA叩打痛: なし / 右 / 左

尿検査
- 潜血(＿) ；結石・解離の鑑別
胸部単純X線検査
腹部単純X線検査
- **所見:** Free air / 結石影 / 縦隔拡大 / 異常なし

**除外項目(Negative ROS)**

頭痛時: 項部硬直なし / 麻痺・しびれなし / 雷鳴頭痛否定

腹痛時: 板状硬なし / 反跳痛なし / 拍動性腫瘤なし

<h3 className="soap-a">評価</h3>

- **診断根拠:** 肋骨に沿った片側性の発作性疼痛。姿勢や咳で誘発されることがある。圧痛点（Valleix点）。
- **除外診断:** 帯状疱疹（皮疹なし）、胸膜炎（深呼吸で増悪）、心疾患、骨折。

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 対症療法（NSAIDs、ビタミンB12）。帯状疱疹の前駆痛の可能性を説明し、発疹が出たら再診するよう指導。

#### 処方・処置

**内服治療セット**

- <span className="med-id">[Rx-Gen-Pain-Lox-Set-14]</span> ロキソプロフェン・レバミピドセット(定期)
- ロキソプロフェンNa錠60mg 1回1錠 1日3回 毎食後 14日分
- レバミピド錠100mg 1回1錠 1日3回 毎食後 14日分
- <span className="med-id">[Rx-Neuro-Vit-Met]</span> メチコバール
- メチコバール錠500μg 1回1錠 1日3回 毎食後 14日分
- <span className="med-id">[Rx-Der-HZ-Mir-10]</span> タリージェ
- タリージェ錠5mg 1回1錠 1日2回 朝夕食後 14日分 *— 7日以上かけて効果をみながら増量*

#### 指導・計画

- **患者指導:** 後から発疹が出る可能性があるため、出現時は直ちに受診するよう指導。

<div className="so-dr-advice">

> 💡 **Dr.Advice**
>
> **[Killer Chest Pain]** 胸痛・背部痛で絶対に見逃してはならない5大疾患（ACS、大動脈解離、肺塞栓、緊張性気胸、食道破裂）。当院にはCT/Echoがないため、少しでも疑わしければ「帰宅させない」覚悟が必要。
>
> **[Killer Abdominal Pain]** 腹部大動脈瘤破裂、上腸間膜動脈閉塞症、消化管穿孔、異所性妊娠。特に高齢者の「腹痛＋ショックバイタル」は血管系イベントを強く疑え。
>
> **[設備制約]** トロポニンTやDダイマーの迅速検査、およびエコー・CTがない。心電図とX線で異常がなくても、症状が強ければ「見逃し」を避けるために救急搬送を選択せよ。

</div>

<div className="ap-advice-spacer"></div>

> 💡 **Dr.Advice**
>
> **[帯状疱疹]** 「肋間神経痛ですね」と診断した数日後に発疹が出て、「誤診された」とトラブルになるケースが多い。「今は水疱がないですが、もし赤いブツブツが出たら帯状疱疹ですので、すぐに来てください」と必ず伏線を張れ。
