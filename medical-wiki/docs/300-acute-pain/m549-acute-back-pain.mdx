<h3 className="soap-s">主訴・ROS</h3>

**痛みの分析 (OPQRST)**

Onset (発症様式): 突然 / 急性 / 徐々 / ある動作で
Palliative/Provocative (増悪・寛解): 体動 / 食事 / 深呼吸 / 安静時
Quality (性状): 締め付けられる / 裂けるよう / 鈍痛 / 差し込む
Region (部位): 胸部 / 心窩部 / 右季肋部 / 右下腹部 / 背部
Severity (強度): 冷や汗をかく激痛 / 会話可能 / 波がある
Time (経過): ＿分/時間 前から持続

**随伴症状**

冷汗 / 嘔気・嘔吐 / 呼吸困難 / 意識消失 / 下血 / なし

**リスク因子**

高血圧 / 糖尿病 / 脂質異常症 / 喫煙 / 心疾患の既往 / なし

<h3 className="soap-o">所見・検査</h3>

バイタル: BP ＿/＿ (左右差: 有 / 無), HR ＿, SpO2 ＿%, BT ＿℃
顔貌: 苦悶様 / 蒼白 / 冷汗 / 安定

**身体所見**

胸部: 異常なし / 心雑音 / 呼吸音異常 / 頸静脈怒張
腹部: 平坦 / 膨満, 腹膜刺激症状 なし / 筋性防御 / 反跳痛
圧痛点: なし / McBurney / Murphy / 全体
CVA叩打痛: なし / 右 / 左

**院内迅速検査**

心電図（12誘導）
- **所見:** 正常 / ST変化 / 異常Q波
末梢血一般(CBC), CRP
- WBC ＿, CRP ＿, 血糖 ＿
尿検査
- 潜血(＿) ；結石・解離の鑑別
胸部単純X線検査
腹部単純X線検査
- **所見:** Free air / 結石影 / 縦隔拡大 / 異常なし

**除外項目(Negative ROS)**

頭痛時: 項部硬直なし / 麻痺・しびれなし / 雷鳴頭痛否定
胸背部痛時: 圧痛なし / 皮疹なし / 血圧左右差なし
腹痛時: 板状硬なし / 反跳痛なし / 拍動性腫瘤なし

<h3 className="soap-ns">診察前対応</h3>

**院内迅速検査**

- 心電図（12誘導）
- 末梢血一般(CBC), CRP
- 尿定性
- 胸部単純X線撮影
- 腹部単純X線撮影

<h3 className="soap-a">評価</h3>

- **診断根拠:** 背部の急激な疼痛。
- 鑑別疾患（整形以外）:
- 大動脈解離（裂ける痛み、移動痛）
- 尿管結石（側腹部～鼡径部への放散、血尿）
- 急性膵炎（アルコール歴、前屈位で軽減）
- 圧迫骨折（体動時痛、叩打痛）

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** Red Flags（解離・膵炎・化膿性脊椎炎）の除外。尿検査（血尿）、X線（結石・骨折）。整形疾患または尿路結石と断定できれば鎮痛剤で経過観察。

#### 処方・処置

**鎮痛 (頓用・選択)**

- ボルタレンサポ50mg 1回1個 激痛時頓用 5回分 *— 結石・整形痛に有効*
- ロキソプロフェンNa錠60mg 1回1錠 (疼痛時) 10回分
- レバミピド錠100mg 1回1錠 (疼痛時) 10回分

**結石排石 (併用)**

- ハルナールD錠0.2mg 1回1錠 1日1回 朝食後 14日分 *— 尿管拡張(適応外)*
- ウロカルン錠225mg 1回2錠 1日3回 毎食後 14日分

**紹介**

#### 指導・計画

- **再診目安:** 疼痛持続時。

> 💡 **Dr.Advice**
>
> **[Killer Chest Pain]** 胸痛・背部痛で絶対に見逃してはならない5大疾患（ACS、大動脈解離、肺塞栓、緊張性気胸、食道破裂）。当院にはCT/Echoがないため、少しでも疑わしければ「帰宅させない」覚悟が必要。
>
> **[Killer Abdominal Pain]** 腹部大動脈瘤破裂、上腸間膜動脈閉塞症、消化管穿孔、異所性妊娠。特に高齢者の「腹痛＋ショックバイタル」は血管系イベントを強く疑え。
>
> **[設備制約]** トロポニンTやDダイマーの迅速検査、およびエコー・CTがない。心電図とX線で異常がなくても、症状が強ければ「見逃し」を避けるために救急搬送を選択せよ。
>
> **[叩打痛]** 肋骨脊柱角(CVA)の叩打痛があれば、尿路結石や腎盂腎炎を強く示唆する。脊椎自体の叩打痛は圧迫骨折や脊椎炎を示唆する。叩く場所を意識して鑑別せよ。
>
> **[D-dimerなし]** 当院ではD-dimerが即日出ないため、解離や肺塞栓の除外には使えない。疑わしければ画像診断ができる施設へ送るのが鉄則である。
