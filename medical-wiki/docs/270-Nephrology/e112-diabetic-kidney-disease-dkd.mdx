<h3 className="soap-s">主訴・ROS</h3>

検診異常: 蛋白尿 / 潜血 / eGFR低下 / クレアチニン上昇

**自覚症状**

浮腫: なし / 下肢 / 眼瞼 / 夕方に増悪
尿異常: 泡立ち / 夜間頻尿
全身症状: なし / 全身倦怠感 / 掻痒感 / 食欲不振 ；尿毒症症状

**リスク因子**

既往歴: 糖尿病 / 高血圧 / 脂質異常症 / 痛風 / なし
家族歴: 透析患者 / 多発性嚢胞腎 / なし
内服歴: なし / NSAIDs長期連用

<h3 className="soap-o">所見・検査</h3>

バイタル: BP ＿/＿ mmHg
身体所見: 浮腫 なし / 下肢pitting / 全身, 眼瞼結膜 正常 / 蒼白

尿検査
- 蛋白(＿) 潜血(＿) 糖(＿)

- 尿沈渣、尿蛋白/Cr比(随時)
- **血液:** Cre, eGFR, BUN, K, P, Ca, Alb, Hb
- **画像:** 当院では腎エコー・CT不可

<h3 className="soap-a">評価</h3>

- **診断根拠:** 糖尿病歴、網膜症合併（一般的）、微量アルブミン尿～顕性蛋白尿。
- **病期:** 第2期:微量Alb / 第3期:顕性Alb / 第4期:腎不全

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 第2期での早期介入が重要。RAAS阻害薬＋SGLT2阻害薬＋MRA（ケレンディア）の多剤併用による強力な腎保護を行う。

#### 処方・処置

**DKD治療セット (併用)**

- <span className="med-id">[Rx-DM-SGLT2-Dap]</span> フォシーガ
- フォシーガ錠10mg 1回1錠 1日1回 朝食後 28日分
- <span className="med-id">[Rx-DM-Nep-Ker]</span> ケレンディア
- ケレンディア錠10mg 1回1錠 1日1回 朝食後 28日分 *— MRA(要K確認)*
- <span className="med-id">[Rx-HT-ARB-Tel40]</span> テルミサルタン
- テルミサルタン錠40mg 1回1錠 1日1回 朝食後 28日分

**早期発見検査 (外注)**

- <span className="med-id">[Lab-DM-Full]</span> 糖尿病詳細セット
- HbA1c, 血糖, 尿中微量アルブミン, 尿中ケトン体, CPR

#### 指導・計画

- **再診目安:** 4週間後。

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院にはエコーがないため、腎後性腎不全（水腎症）や腎萎縮の評価ができない。前立腺肥大症の既往がある男性や、急激なeGFR低下例は、閉塞機転の除外のため一度は画像診断可能な施設へ紹介せよ。
>
> **[検査タイムラグ]** 血液検査（Cre, K）の結果は「後日」となる。「今日現在の腎機能」は分からないため、著しい体調不良（尿毒症疑い）や乏尿がある場合は、結果を待たずに総合病院へ送る判断が必要。

</div>

> 💡 **Dr.Advice**
>
> **[ケレンディア]** 糖尿病性腎症に対し、従来のMRA（スピロノラクトン等）より高カリウム血症リスクが低く、腎保護効果が証明された薬剤。ACE/ARB+SGLT2でも蛋白尿が残る場合に追加せよ。
