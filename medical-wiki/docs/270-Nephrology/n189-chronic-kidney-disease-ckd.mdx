<h3 className="soap-s">主訴・ROS</h3>

検診異常: 蛋白尿 / 潜血 / eGFR低下 / クレアチニン上昇

**自覚症状**

浮腫: なし / 下肢 / 眼瞼 / 夕方に増悪
尿異常: 泡立ち / 夜間頻尿
全身症状: なし / 全身倦怠感 / 掻痒感 / 食欲不振 ；尿毒症症状

**リスク因子**

既往歴: 糖尿病 / 高血圧 / 脂質異常症 / 痛風 / なし
家族歴: 透析患者 / 多発性嚢胞腎 / なし
内服歴: なし / NSAIDs長期連用

<h3 className="soap-o">所見・検査</h3>

バイタル: BP ＿/＿ mmHg
身体所見: 浮腫 なし / 下肢pitting / 全身, 眼瞼結膜 正常 / 蒼白

**院内迅速検査**

尿検査
- 蛋白(＿) 潜血(＿) 糖(＿)

**外注検査 (結果は後日)**

- 尿沈渣、尿蛋白/Cr比(随時)
- **血液:** Cre, eGFR, BUN, K, P, Ca, Alb, Hb
- **画像:** 当院では腎エコー・CT不可

<h3 className="soap-ns">診察前対応</h3>

**院内迅速検査**

- 尿定性

<h3 className="soap-a">評価</h3>

- **診断根拠:** eGFR&lt;60 または 尿異常(蛋白尿等)が3ヶ月以上持続。
- **ステージ:** G1 / 2 / 3a / 3b / 4 / 5
- **原因:** 糖尿病性 / 腎硬化症 / 慢性糸球体腎炎 / 不明
- **尿蛋白区分:** A1:正常 / 2:軽度 / 3:高度

<h3 className="soap-p">方針・処方</h3>

- **治療戦略:** 血圧管理(130/80未満・RAAS阻害薬)。腎保護(SGLT2阻害薬導入)。合併症管理(腎性貧血・高K血症・アシドーシス)。減塩指導(6g/日未満)。

#### 処方・処置

**腎保護薬 (併用推奨)**

- フォシーガ錠10mg 1回1錠 1日1回 朝食後 28日分 *— 第一選択*
- テルミサルタン錠40mg 1回1錠 1日1回 朝食後 28日分 *— 尿蛋白抑制*

**腎性貧血 (内服のみ)**

- ダーブロック錠2mg 1回1錠 1日1回 朝食後 28日分 *— HIF-PH阻害薬*
- エベレンゾ錠20mg 1回1錠 週3回(月水金など) 28日分

**高カリウム血症 (K&gt;5.5)**

- ロケルマ懸濁用散剤5g 1回1包 1日1回 (水に懸濁) 28日分 *— 即効性*
- カリメート経口液20% 1包 1日2回～3回 食後 28日分 *— 標準*

**代謝性アシドーシス**

- 炭酸水素ナトリウム錠500mg 1回2錠 1日3回 毎食後 28日分

**紹介**

#### 指導・計画

- **再診目安:** 4週間後。

> 💡 **Dr.Advice**
>
> **[設備制約]** 当院にはエコーがないため、腎後性腎不全（水腎症）や腎萎縮の評価ができない。前立腺肥大症の既往がある男性や、急激なeGFR低下例は、閉塞機転の除外のため一度は画像診断可能な施設へ紹介せよ。
>
> **[検査タイムラグ]** 血液検査（Cre, K）の結果は「後日」となる。「今日現在の腎機能」は分からないため、著しい体調不良（尿毒症疑い）や乏尿がある場合は、結果を待たずに総合病院へ送る判断が必要。
>
> **[注射不可]** 当院ではESA注射（ネスプ等）の採用がないため、腎性貧血治療は内服のHIF-PH阻害薬で行う。HIF-PH阻害薬の使用が難しい場合（悪性腫瘍合併等）は、ESA投与のために他院へ紹介する必要がある。
>
> **[SGLT2阻害薬]** RAS阻害薬に次ぐCKD治療の基本薬。投与初期にeGFRが10～20%低下すること（Initial Dip）があるが、中止せず継続することで長期的には腎保護に働く。
>
> **[高K血症薬の使い分け]** ロケルマは即効性があり味が良いが、Naと交換するため「浮腫・心不全」に注意。カリメートは標準的だが「便秘・腸管穿孔」リスクがある。心機能低下例にはカリメート、それ以外はロケルマが第一選択となりつつある。
