[Vertigo] めまい・末梢神経 (初診)

<S: 主訴・ROS>:
    【めまい (TiTrATE)】
    Timing (発症・持続): ( 突然 / 徐々 / 秒単位 / 分単位 / 時間単位 / 一日中 )
    Triggers (誘因): ( 頭位変換 / 起立 / ストレス / 特になし )
    And
    Targeted Exam (随伴症状): 難聴、耳鳴り、耳閉感、頭痛、しびれ
    性状: ( 回転性 / 浮動性 / 前失神 / 立ちくらみ )
    【末梢神経・しびれ】
    部位: ( 手袋靴下型 / デルマトーム / 多発単神経 )
    性状: ( ジンジン / ピリピリ / 感覚脱失 / 筋力低下 )
    【背景】
    既往歴: 糖尿病、高血圧、脂質異常症、癌（化学療法歴）、飲酒

<O: 所見・検査>:
    バイタル: BP: ＿/＿ (左右差・起立試験), HR: ＿, SpO2: ＿%
    【神経所見 (中枢性除外)】
    眼振: ( 水平回旋混合 / 垂直 / 方向交代性 / 注視方向性 / なし )
    小脳症状: 指鼻試験 ( 正常 / 測定障害 ), 体幹失調 ( 起立歩行可 / 不可 )
    脳神経: 顔面神経麻痺、構語障害、複視
    HINTS: Head Impulse ( 正常 / 異常 ), Nystagmus, Test of Skew
    【耳】
    聴力検査: ( 正常 / 感音難聴 / 低音障害 )
    鼓膜所見: ( 正常 / 内陥 / 混濁 )
    【誘発テスト】
    Dix-Hallpike / Supine Roll test: ( 陽性 / 陰性 )


▶ Dr.Advice [中枢性のRed Flags (3D)]:
    ・Diplopia (複視): 複視が突然出現したら脳幹・小脳障害を強く疑え。
    ・Dysarthria/Dysphagia (構語・嚥下障害): 突然の構語障害は脳卒中のレッドフラッグ。
    ・Disequilibrium (体幹失調・歩行不能): 起立・歩行が全くできないめまいは末梢性ではなく中枢性の証拠。追加で「突然発症・頭痛・意識障害・血圧異常」があればただちに救急搬送。
▶ Dr.Advice [HINTS plus]:
    ・Head Impulse Test (頭位衝動試験) 「異常 = 末梢性」: 急速な頭の動かしに対し眼球が遅れて追いかけるキャッチアップ眼球運動(Catch-up saccade)があれば「末梢性（前庭神経炎など）を示す」。
    ・Nystagmus (眼振パターン): 方向固定性の水平眼振 → 末梢性。方向交代性や垂直成分入り → 中枢性。
    ・Test of Skew (遮閉試験): 垂直方向の眼球偏倚(Skew deviation)があれば中枢性。
    ・HINTS "plus": 聴力低下(急性・一側性)が加わった場合も中枢性（特に後下小脳動脈梗塞）を否定できない。
    ・臨床的意義: HINTSはMRIを超える感度で中枢性を検出できるとされる（ただし習熟が必要）。急性前庭症候群でHINTSが「末梢性パターン」でない場合は緊急MRIを考慮せよ。
