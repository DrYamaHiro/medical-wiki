[Cardiovascular] 循環器・血管疾患 (再診)

<S: 主訴・ROS>:
    自覚症状: ( 息切れ改善 / 動悸なし / 胸痛なし / 変化なし )
    夜間就寝: ( 枕1つで可 / 苦しくて起きる )
    【生活管理】
    体重: ＿kg (前回比 ＿kg), 塩分摂取: ( 遵守 / 外食多い )
    【血管・出血】
    歩行距離の変化: ( 延長 / 不変 / 短縮 )
    弾性ストッキング: ( 着用中 / 未着用 )
    出血傾向: ( なし / 鼻出血 / 歯肉出血 / 皮下出血 )

<O: 所見・検査>:
    バイタル: BP ＿/＿, HR ＿
    頸静脈怒張、下肢浮腫の改善度。
    下肢皮膚: 爪の色、小さな傷や潰瘍の有無（フットケア確認）。


▶ Dr.Advice [体重モニタリング]: 心不全管理において体重は最も鋭敏な指標である。「数日で2kg以上の増加」は体液貯留（急性増悪）のサインであり、早期の利尿薬調整が必要。
