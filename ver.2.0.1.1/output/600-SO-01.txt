[Emergency] 救命救急・CPA対応 (Code Blue)

<S: 状況>:
    発見状況: ( 待合室で倒れた / 診察中に意識消失 / 処置中に急変 )
    目撃者の有無: ( あり / なし )
    最終健常確認時刻: ＿時＿分
    既往歴: ( 心疾患 / アレルギー / 糖尿病 / 脳卒中 )

<O: 身体所見・モニタリング>:
    意識: ( JCS 300 / GCS 3 / 反応なし )
    呼吸: ( 無呼吸 / 死戦期呼吸 / 努力様呼吸 )
    脈拍: ( 橈骨動脈触知不可 / 頸動脈触知不可 / 10秒確認 )
    瞳孔: ( 散瞳 / 対光反射消失 )
    心電図波形: ( VF / pVT / PEA / Asystole )
    SpO2: ( 測定不能 / 低下 )


▶ Dr.Advice [Code Blue]: 「誰も一人で蘇生はできない」。発見時は直ちに大声で人を集め、役割分担（リーダー、胸骨圧迫、気道、ルート、タイムキーパー、記録、家族対応）を明確にせよ。
▶ Dr.Advice [中断の最小化]: 胸骨圧迫の中断は「10秒以内」厳守。ルート確保や挿管のために圧迫を止めてはならない。
