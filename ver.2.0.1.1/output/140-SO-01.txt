[Headache] 頭痛・てんかん・神経痛 (初診)

<S: 主訴・ROS>:
    【頭痛 (OPQRST)】
    発症様式: ( 突然[雷鳴] / 急性 / 徐々 / いつもの頭痛 )
    増悪・寛解: ( 体動 / 入浴 / アルコール / 光・音 / 安静 )
    性状: ( ズキズキ / 締め付けられる / 殴られたような / 電撃的 )
    部位: ( 片側 / 両側 / 後頭部 / 目の奥 / 全体 )
    強度: NRS ＿/10 ( 生活に支障あり / 寝込む )
    経過: ＿時間/日 持続、頻度 ＿回/月
    【随伴症状】
    悪心・嘔吐、閃輝暗点、流涙・鼻汁、肩こり
    しびれ、麻痺、言葉が出にくい、発熱
    【てんかん・発作】
    意識消失 ( 有 / 無 )、痙攣 ( 全身 / 一部 )、前兆
    目撃情報: ( 白目 / 泡を吹く / 硬直 / 脱力 )
    【背景】
    既往歴: ( 片頭痛 / 高血圧 / 癌 / 副鼻腔炎 / 脳卒中 )
    市販薬使用: ( なし / あり: ＿錠/月 ) ；MOHリスク評価

<O: 所見・検査>:
    バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃
    【神経学的所見】
    意識レベル: ( 清明 / JCS ＿ / GCS ＿ )
    髄膜刺激症状: 項部硬直 ( 有 / 無 )、Kernig徴候 ( 有 / 無 )
    局所徴候: 麻痺 ( 有 / 無 )、Barre徴候 ( 有 / 無 )
    脳神経: 眼瞼下垂 ( 有 / 無 )、瞳孔不同 ( 有 / 無 )
    小脳症状: 指鼻試験 ( 正常 / 測定障害 )
    【局所】
    側頭動脈: ( 正常 / 怒張 / 圧痛 ) ；GCA疑い
    圧痛点: 副鼻腔叩打痛 ( 有 / 無 )、大後頭神経圧痛 ( 有 / 無 )
    【除外項目(Negative ROS)】
    雷鳴頭痛の否定、神経脱落症状なし、項部硬直なし


▶ Dr.Advice [SNOOP4 (Red Flags)]: 以下のサインは二次性頭痛（危険）を示唆する。一つでもあれば、当院で抱えずCT/MRIのある施設へ紹介せよ。
▶ Dr.Advice [雷鳴頭痛の定義]: 「突然」とは「何時何分」と言えるレベルの急激さであり、「バットで殴られたような」と表現される。痛みのピークが1分以内に完成する場合は、CTが正常でもルンバール（髄液検査）が必要なSAHの可能性がある。
▶ Dr.Advice [緑内障発作]: 「頭が痛い」と言って来院しても、実は「眼圧上昇（緑内障発作）」であることがある。特に高齢女性の「片頭痛＋吐き気＋眼痛＋充血」は要注意。ペンライトで瞳孔を確認し、散瞳・対光反射消失があれば即眼科へ。
