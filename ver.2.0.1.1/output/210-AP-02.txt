[B35.1] 爪白癬 (Tinea Unguium)

<A: 評価>:
    ・診断根拠: 爪の白濁、肥厚、混濁。
    ・重症度: ( 軽度:先端のみ / 中等度:1/2以上 / 重度:爪根部まで浸潤 )
    ・性状: ( 楔状混濁 / 線状混濁 / 正常 )

<P: 方針>:
    ・治療戦略: 重症度と肝機能、併用薬を考慮し、外用または内服を選択。

    ▼ 処方・アクション (選択) ▼
    【爪外用液】
    [ID: Rx-Der-Tin-Nail] クレナフィン爪外用液10% 1本 1日1回 爪全体に塗布
    [ID: Rx-Der-Tin-Luc] ルコナック爪外用液5% 1本 1日1回 爪全体に塗布

    【内服療法 (要肝機能チェック)】
    [ID: Rx-Der-Tin-Fos] ネイリンカプセル100mg 1回1カプセル 1日1回 食後 84日間
    [ID: Rx-Der-Tin-Ter] ラミシール錠125mg 1回1錠 1日1回 食後 28日分

    ▼ 指導・計画 (必須) ▼
    ・患者指導: 爪が生え変わるまで（足指で約1年）かかることを説明。内服時は定期的な採血が必要。
    ・再診目安: 4週間後。



▶ Dr.Advice [外用の限界]: 外用薬の完全治癒率は高くない。「厚くなった爪」や「根本まで白い爪」は外用では薬剤が届かないことが多い。「飲み薬の方が確実ですが、肝臓の数値を気にする必要があります。どちらにしますか？」と提案せよ。
▶ Dr.Advice [ネイリン]: 12週間（約3ヶ月）で飲みきり終了だが、薬効は9ヶ月続く。「飲み終わった時点ではまだ爪は汚いままですが、その後半年かけて綺麗な爪が生えてきます」と説明し、焦らせないこと。
