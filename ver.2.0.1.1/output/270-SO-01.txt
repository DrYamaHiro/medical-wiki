[Nephrology] 慢性腎臓病・CKD (初診)

<S: 主訴・ROS>:
    検診異常: ( 蛋白尿 / 潜血 / eGFR低下 / クレアチニン上昇 )
    【自覚症状】
    浮腫: ( なし / 下肢 / 眼瞼 / 夕方に増悪 )
    尿異常: ( 泡立ち / 夜間頻尿 )
    全身症状: ( なし / 全身倦怠感 / 掻痒感 / 食欲不振 ) ；尿毒症症状
    【リスク因子】
    既往歴: ( 糖尿病 / 高血圧 / 脂質異常症 / 痛風 / なし )
    家族歴: ( 透析患者 / 多発性嚢胞腎 / なし )
    内服歴: ( なし / NSAIDs長期連用 )

<O: 所見・検査>:
    バイタル: BP ＿/＿ mmHg
    身体所見: 浮腫 ( なし / 下肢pitting / 全身 ), 眼瞼結膜 ( 正常 / 蒼白 )
    【院内迅速検査】
    尿検査
    ・蛋白(＿) 潜血(＿) 糖(＿)
    【外注検査 (結果は後日)】
    ・尿沈渣、尿蛋白/Cr比(随時)
    ・血液: Cre, eGFR, BUN, K, P, Ca, Alb, Hb
    ・画像: ( 当院では腎エコー・CT不可 )

<Ns: 診察前対応>:
    【院内迅速検査】
    [ID: Lab-Gen-Urine] 尿定性

▶ Dr.Advice [設備制約]: 当院にはエコーがないため、腎後性腎不全（水腎症）や腎萎縮の評価ができない。前立腺肥大症の既往がある男性や、急激なeGFR低下例は、閉塞機転の除外のため一度は画像診断可能な施設へ紹介せよ。
▶ Dr.Advice [検査タイムラグ]: 血液検査（Cre, K）の結果は「後日」となる。「今日現在の腎機能」は分からないため、著しい体調不良（尿毒症疑い）や乏尿がある場合は、結果を待たずに総合病院へ送る判断が必要。
