[I10] 難治性高血圧 / 多剤併用 (HT - Resistant)

<A: 評価>:
    ・定義: 利尿薬を含む3剤以上を使用しても目標血圧未達。
    ・阻害要因チェック:
        1. 塩分摂取: ( 過剰 / 適正 )
        2. 服薬状況: ( 飲み忘れあり / 良好 )
        3. 薬剤性: ( NSAIDs / 甘草 / ステロイド )
        4. 二次性/合併症: ( SAS / 原発性アルドステロン症 / 腎血管性 )

<P: 方針>:
    ・阻害要因（特に減塩不足とNSAIDs）の除去。
    ・Step 3: サイアザイド系利尿薬の追加（必須）。
    ・Step 4: MRA（ミネラルコルチコイド受容体拮抗薬）の追加。
    ・二次性高血圧スクリーニング（ARR測定等）。

    ▼ 処方・処置 (選択) ▼
    【強化療法セット(3剤目・4剤目)】
    [ID: Rx-HT-Diuretics-Flu] フルイトラン1mg
    ・フルイトラン錠1mg 1回1錠 1日1回 朝食後 28日分 ；少量の利尿薬
    [ID: Rx-HT-MRA-Esa] ミネブロ
    ・ミネブロ錠1.25mg-2.5mg 1回1錠 1日1回 朝食後 28日分 ；MRA(夜間高血圧に著効)
    [ID: Rx-HT-Com-MicAP] ミカムロAP
    ・ミカムロ配合錠AP 1回1錠 1日1回 朝食後 28日分 ；ベース薬

    【精査オーダー】
    [ID: Lab-HT-Secondary] 二次性高血圧セット
    ・レニン活性(PRA), アルドステロン(PAC), コルチゾール, ACTH
    [ID: Doc-Gen-Referral] 診療情報提供書
    ・紹介先: 専門医療機関（SAS・副腎腫瘍精査）


▶ Dr.Advice [偽性抵抗性]: 「薬が効かない」原因のトップは「塩分の摂りすぎ」である。
▶ Dr.Advice [MRAの劇的効果]: 難治例の多くは「体液貯留」や「アルドステロンブレイクスルー」が関与している。β遮断薬を漫然と足すより、MRA（ミネブロ、スピロノラクトン）を追加する方が効果的である。
▶ Dr.Advice [画像診断]: 当院にはCT/Echoがないため、副腎腫瘍や腎動脈狭窄の確定診断はできない。ホルモン異常があれば速やかに紹介せよ。
