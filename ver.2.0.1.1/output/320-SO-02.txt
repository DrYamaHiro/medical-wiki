[Hematology] 血液疾患・貧血 (再診)

<S: 主訴・ROS>:
    自覚症状: ( 改善 / 息切れ改善 / 氷食消失 / 変化なし )
    服薬状況: ( 良好 / 胃痛・嘔気あり / 便秘 )
    便の色: ( 黒色 / 普通 ) ；服薬コンプライアンス確認

<O: 所見・検査>:
    Hb推移: ＿ → ＿
    網状赤血球(Ret): ( 上昇 / 不変 ) ；造血反応の確認
    フェリチン: ＿ (貯蔵鉄の回復確認)


▶ Dr.Advice [治療期間]: Hbが正常化しても「貯蔵鉄（フェリチン）」が回復するまで（通常3～6ヶ月）は鉄剤を中止してはならない。早期中止は再発の主因である。
▶ Dr.Advice [黒色便]: 鉄剤服用中は便が真っ黒になる。「腸の中で出血しているわけではなく、薬の色なので安心してください」と必ず事前に説明せよ。
