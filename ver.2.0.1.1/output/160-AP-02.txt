[G47.3] 閉塞性睡眠時無呼吸症候群 (OSAS) / CPAP定期管理

<A: 評価>:
    ・CPAPデータ評価 (直近1ヶ月):
        - AHI (残存): ＿ 回/時 (目標<5: 良好 / 5-10: 要注意 / >10: 不良)
        - 使用時間: 平均 ＿ 時間/日 (目標>4時間)
        - 使用率: ＿ % (目標>70%)
        - リーク: ( 許容範囲内 / 多い・要調整 )
        - 圧設定: ＿ cmH2O ( オート / 固定 )
    ・体重変化: 前回 ＿kg → 今回 ＿kg ( ＿kg 減/増 )

<P: 方針>:
    ・治療戦略:
        - データに基づく指導（マスクフィッティング、加湿器、圧調整）。
        - アドヒアランス不良例への介入（マスク変更、ナザール等の併用）。
        - 合併症（高血圧・脂質異常症）の管理。

    ▼ 管理・処方 (選択) ▼
    【在宅管理料】
    [ID: Tx-Resp-CPAP] 在宅持続陽圧呼吸療法指導管理料

    【鼻閉対策】
    [ID: Rx-Gen-Ent-Nas-Ava] アラミスト点鼻
    ・アラミスト点鼻液27.5μg 1瓶 1日1回 各鼻腔 噴霧

    ▼ 指導・計画 (必須) ▼
    [ID: Doc-Gen-Lifestyle] 生活習慣病指導用パンフレット


▶ Dr.Advice [4時間ルール]: 心血管イベント（脳卒中・心筋梗塞）の抑制効果を得るには「毎日4時間以上」の使用が必要とされる。「寝ている間ずっと」が無理なら、「寝入りの4時間だけは頑張ろう」とハードルを下げて指導せよ。
▶ Dr.Advice [残存AHIが高い時]: CPAPをしていてもAHIが下がらない場合、①マスクからの空気漏れ（リーク）、②口呼吸、③圧不足、④中枢性無呼吸(CSA)の出現、を疑う。安易に圧を上げるとリークが増えて逆効果になることがある。まずはマスク調整とチンストラップ（口呼吸防止）を検討せよ。
