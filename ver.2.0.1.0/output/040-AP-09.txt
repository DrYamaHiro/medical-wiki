[J30.1] スギ花粉症 / 舌下免疫療法・維持 (SLIT - Cedar Maintenance)

<A: 評価>:
    ・治療経過: シダキュア5,000JAUによる維持療法中。
    ・アドヒアランス: ( 毎日服用 / 週に数回忘れ / 中断気味 )
    ・副作用: ( なし / 口腔内違和感 / 腹痛 / 喘息発作 )
    ・効果実感: ( 症状軽減 / 薬が減った / まだ不明 )

<P: 方針>:
    ・維持量（5,000JAU）の継続処方。
    ・生活指導（服用後2時間は激しい運動・入浴・飲酒を避ける）。
    ・治療期間（3～5年推奨）の再確認とモチベーション維持。

    ▼ 処方・アクション (選択) ▼
    【維持期処方セット】
    [ID: Rx-Ent-SLIT-Cida5000] シダキュア5000(維持)
    ・シダキュアースギ花粉舌下錠5,000JAU 1回1錠 1日1回 舌下 28日分
    [ID: Rx-Gen-Ent-Bil] ビラノア
    ・ビラノア錠20mg 1回1錠 1日1回 空腹時 28日分 ；飛散期・症状ある時


▶ Dr.Advice [前後2時間のルール]: 服用直後の血行促進（運動・入浴・飲酒）は、アレルゲンの吸収を早め、アナフィラキシーのリスクを高めるため避けるよう毎回指導せよ。
▶ Dr.Advice [抜歯・口内炎]: 抜歯後や口内炎がひどい時は、傷口からアレルゲンが直接血中に入る恐れがあるため、治るまで休薬させること。
▶ Dr.Advice [飲み忘れ対応]: 「昨日の分を飲み忘れたからといって、今日2回分まとめて飲む」のは副作用リスクが高まるため厳禁である。「忘れた分は飛ばして、今日1回分だけ飲む」よう指導せよ。
▶ Dr.Advice [長期中断のリスク]: 数日の飲み忘れならそのまま再開可能だが、1ヶ月以上中断した後にいきなり維持量(5,000JAU)を再開するとアナフィラキシーを起こす危険がある。長期中断例は必ず受診させ、場合によっては導入(2,000JAU)からやり直す判断が必要。
▶ Dr.Advice [モチベーション維持]: 症状がない時期（夏～冬）に毎日薬を飲むことは苦痛である。「今の頑張りが来年の春に報われる」「3年頑張れば、一生楽になるかもしれない」と、長期的なメリットを繰り返し伝えて励ませ。
▶ Dr.Advice [喘息発作時]: 風邪などを契機に喘息発作（咳・喘鳴・呼吸困難）が出現した場合は、シダキュアを「直ちに中止」し、喘息治療を優先すること。喘息が完全に落ち着くまで再開してはならない。
