[Orthopedics] 脊椎・運動器疼痛 (初診)

<S: 主訴・ROS>:
    部位: ( 腰 / 頸 / 肩 / 膝 / 手首 / 足底 ) ( 右 / 左 / 両側 )
    疼痛スケール(NRS): ＿/10
    受傷機転: ( なし / 重いものを持った / 転倒 / 寝違え / スポーツ )
    増悪因子: ( 前屈 / 後屈 / 起立時 / 歩行時 / 夜間 / 起床時 )
    しびれ・放散痛: ( なし / あり: 上肢・下肢 )
    【Red Flags確認】
    排尿・排便障害: ( なし / あり ) ；馬尾症状
    全身症状: ( なし / 発熱 / 体重減少 / 癌の既往 )
    安静時・夜間痛: ( なし / あり ) ；感染・腫瘍疑い

<O: 所見・検査>:
    歩行様式: ( 正常 / 跛行 / 前傾姿勢 )
    圧痛点: ( 棘突起 / 脊柱起立筋 / 関節裂隙 / 腱付着部 )
    可動域(ROM): ( 正常 / 屈曲・伸展制限 / 疼痛制限あり )
    【神経学的所見】
    徒手筋力(MMT): ( 正常 / 低下: ＿ )
    感覚障害: ( なし / あり: デルマトーム ＿ )
    腱反射(PTR/ATR): ( 正常 / 低下 / 亢進 )
    【誘発テスト】
    腰椎: SLR ( - / + ＿° ), FNS ( - / + )
    頸椎: Spurling ( - / + ), Jackson ( - / + )
    その他: Finkelstein ( - / + ), Drop arm ( - / + )


▶ Dr.Advice [Red Flags]: 「安静にしていても痛い（内臓痛・腫瘍・化膿性脊椎炎）」「熱がある」「尿が出にくい（馬尾症候群）」は緊急事態である。当院にはMRIがないため、これらの徴候があれば鎮痛剤で様子を見ず、直ちに専門病院へ紹介せよ。
▶ Dr.Advice [画像診断の適応]: Red Flagsがない単純な急性腰痛（ぎっくり腰）や寝違えであれば、初診時のX線撮影は医学的に必須ではない（治療方針が変わらないため）。患者が「骨折がないか心配」している場合や、高齢者（圧迫骨折リスク）の場合に撮影を行え。
