[Health-Check] 健診・書類 (初診)

<S: 主訴・ROS>:
    目的: ( 特定健診 / 雇用時健診 / 診断書作成 / 抗体検査 / 予防接種相談 )
    提出先: ( 会社 / 学校 / 保健所 / 保険会社 / 警察署[銃砲] )
    期限: ＿月＿日までに必要
    気になる症状: ( 特になし / ＿ )
    既往歴: ＿
    【生活習慣】
    喫煙: ( なし / あり ＿本/日 × ＿年 )
    飲酒: ( なし / ＿合/日 )
    運動: ( なし / 週2回以上 )

<O: 所見・検査>:
    身長: ＿cm, 体重: ＿kg, BMI: ＿, 腹囲: ＿cm
    血圧: ＿/＿ mmHg, 脈拍: ＿ bpm
    視力: ( 右 ＿ / 左 ＿ ) ( 矯正 / 裸眼 )
    聴力(1000Hz): ( 右 ＿ / 左 ＿ ), (4000Hz): ( 右 ＿ / 左 ＿ )
    【診察】
    心音・呼吸音: ( 異常なし )、貧血: ( 無 / 有 )
    【検査実施】
    胸部X線: ( 異常なし / ＿ )
    心電図: ( 正常洞調律 / ＿ )
    尿検査: 蛋白(＿) 糖(＿) 潜血(＿)
    採血: ( 実施済 / 項目確認 ) ；結果は後日


▶ Dr.Advice [結果渡し]: 当院の血液検査は外注（SRL）であるため、診断書や健診結果の即日発行は不可能である。「結果が出るまで数日かかります」と受付段階で必ず説明し、トラブルを回避せよ。
▶ Dr.Advice [健診の意義]: 健診は「異常を見つける」だけでなく、「自分の体を知り、生活習慣を見直すきっかけ」であることを伝え、結果説明の受診を促せ。
