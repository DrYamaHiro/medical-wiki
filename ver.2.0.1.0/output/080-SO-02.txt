[Hypertension] 高血圧・血管疾患 (再診)

<S: 主訴・ROS>:
    家庭血圧平均: 朝 ＿/＿ mmHg, 夜 ＿/＿ mmHg
    自覚症状: ( なし / ふらつき / 動悸 / 頭痛 )
    服薬状況: ( 良好 / 飲み忘れあり / 残薬 ＿日分 )
    生活改善: ( 減塩意識 / 運動継続 / 体重変化 ＿kg )

<O: 所見・検査>:
    診察室血圧: ＿/＿ mmHg, 脈拍: ＿ bpm
    浮腫: ( なし / あり ；Ca拮抗薬副作用等 )


▶ Dr.Advice [季節変動]: 冬場は血圧が上がりやすく、夏場は下がりやすい。夏場に立ちくらみ（低血圧症状）が出る場合は、降圧薬の減量や休薬を検討せよ。
▶ Dr.Advice [残薬調整]: 「飲み忘れで薬が余っている」と言われた場合、責めずに「正直に教えてくれてありがとう」と肯定し、余っている分だけ処方日数を調整せよ（残薬調整）。
