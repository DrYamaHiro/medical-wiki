[Osteoporosis] 骨粗鬆症・骨代謝 (初診)

<S: 主訴・ROS>:
    検診異常: ( 骨密度低下 / YAM値指摘 )
    身長低下: ( なし / あり: 若い頃より ＿cm 低下 ) ；2cm以上はリスク
    背部痛: ( なし / あり: いつの間にか骨折疑い )
    【リスク因子(FRAX)】
    年齢: ＿歳, 性別: ( 女 / 男 ), 閉経年齢: ＿歳 ( 早発閉経 / 卵巣摘出 )
    骨折歴: ( なし / あり: 部位・時期 )
    家族歴: ( なし / 大腿骨近位部骨折 / 脊椎圧迫骨折 )
    生活歴: ( ステロイド使用 / 喫煙 / アルコール多飲 )

<O: 所見・検査>:
    バイタル: 身長 ＿cm, 体重 ＿kg, BMI ＿
    姿勢: ( 正常 / 円背 ), 壁際直立時の後頭部・壁距離 ( 0cm / >0cm )
    【骨密度 (他施設DEXA法等の結果)】
    腰椎(L2-4): YAM ＿ %, T-score ＿
    大腿骨頚部: YAM ＿ %, T-score ＿
    【採血 (外注)】
    骨代謝マーカー
    ・TRACP-5b(骨吸収), P1NP(骨形成), 25(OH)VitD
    ・Ca, P, ALP, Cre, eGFR

<Ns: 診察前対応>:
    【採血 (外注)】
    [ID: Lab-Ost-Marker] 骨代謝マーカー

▶ Dr.Advice [身長低下]: 若い頃より「2cm以上」縮んでいれば椎体骨折の可能性が高い。「4cm以上」ならほぼ確実である。痛みがなくても胸腰椎X線を撮像せよ。
▶ Dr.Advice [YAM値]: 診断基準はYAM 70%未満（または80%未満＋危険因子）である。同年代比較(Age Match)ではなく、常に「若年成人平均(YAM)」を基準に判断せよ。
▶ Dr.Advice [設備制約]: 当院にはDEXA（骨密度測定装置）がないため、確定診断および詳細な経過評価には近隣のDEXA保有施設へ検査を依頼せよ。
