[Psychiatry] 不眠・不安・心身症 (再診)

<S: 主訴・ROS>:
    睡眠: ( 入眠改善 / 中途覚醒残存 / 熟眠感あり / 悪夢あり )
    日中の眠気・持ち越し: ( なし / あり )
    気分の変化: ( 改善 / 波がある / 意欲低下 )
    服薬状況: ( 指示通り / 頓服利用 / 自己増量 / お酒と併用 )

<O: 所見・検査>:
    表情の変化、身だしなみ。
    副作用確認: ( ふらつき / 転倒 / 健忘 / 吐き気 )


▶ Dr.Advice [減薬のタイミング]: 「よく眠れる」状態が1ヶ月以上続いたら、少しずつ減薬（漸減）や、頓服への切り替えを提案せよ。「急にゼロ」にすると反跳性不眠（リバウンド）が起きるため注意。
▶ Dr.Advice [専門医紹介]: 希死念慮がある場合、双極性障害（躁うつ）が疑われる場合、幻覚妄想がある場合は、プライマリケアで抱えず精神科専門医へ紹介せよ。
▶ Dr.Advice [賦活症候群]: 抗うつ薬（SSRI等）の開始初期（1～2週間）に、不安・焦燥・衝動性が一時的に高まる「賦活症候群(Activation Syndrome)」が起こりうる。「最初は少し薬が合わない感じがするかもしれないが、勝手に止めずに連絡してほしい」と伝えておくことが脱落防止になる。
