[M51.2] 腰椎椎間板ヘルニア (Lumbar Disc Herniation)

<A: 評価>:
    ・診断根拠: 腰痛に加え、片側下肢への放散痛（坐骨神経痛）。SLRテスト陽性（L4/5/S1）またはFNSテスト陽性（L2/3/4）。
    ・神経所見: ( 拇指背屈力低下 / アキレス腱反射消失 / 感覚鈍麻 / なし )
    ・進行性: ( なし / 麻痺の進行あり )

<P: 方針>:
    ・治療戦略: 神経障害性疼痛薬の導入。保存療法で3ヶ月以内に約80%は自然縮小する。排尿障害や進行する麻痺（下垂足）があれば手術適応として即紹介。

    ▼ 処方・処置 ▼
    【神経痛治療 (選択)】
    [ID: Rx-Der-HZ-Mir-10] タリージェ
    ・タリージェ錠5mg 1回1錠 1日2回 朝夕食後 14日分 ；7日以上かけて効果をみながら増量
    [ID: Rx-Neuro-Vit-Met] メチコバール
    ・メチコバール錠500μg 1回1錠 1日3回 毎食後 28日分 ；神経修復

    【鎮痛薬 (炎症期)】
    [ID: Rx-Gen-Pain-Lox-Set-14] ロキソプロフェン・レバミピドセット(定期)
    ・ロキソプロフェンNa錠60mg 1回1錠 1日3回 毎食後 14日分
    ・レバミピド錠100mg 1回1錠 1日3回 毎食後 14日分

    【紹介】
    [ID: Doc-Gen-Referral] 診療情報提供書 (整形外科へ紹介：MRI精査・ブロック注射依頼)

    ▼ 指導・計画 ▼
    ・再診目安: 2週間後。


▶ Dr.Advice [自然吸収]: ヘルニアはマクロファージに貪食され自然縮小・消失することが多い。手術は「待てないほどの痛み」や「麻痺（脱力）」がある場合に検討される。
▶ Dr.Advice [SLRテスト]: 仰臥位で膝を伸ばしたまま足を挙上し、70度以下で下肢に放散痛が走れば陽性（L4/5, L5/S1ヘルニア示唆）。
