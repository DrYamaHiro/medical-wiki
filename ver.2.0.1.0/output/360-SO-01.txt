[Ped-Checkup] 乳幼児健診・育児相談 (定期)

<S: 主訴・ROS>:
    月齢: ＿ヶ月/歳 ( 1M / 3-4M / 6-7M / 9-10M / 1歳半 / 3歳 )
    出生歴: ＿週 ＿g分娩 ( 経腟 / 帝切 )、合併症 ( 無 / 有 )
    栄養: ( 母乳 / 混合 / 人工乳 / 離乳食 ＿回 )
    食欲: ( 良好 / ムラあり / 遊び食べ )
    排泄: 便 ＿回/日、尿 ＿回/日
    睡眠: ( まとまって寝る / 夜泣きあり / 昼夜逆転 )
    心配事: ( 発達 / 体重 / 皮膚 / 陰部 / 臍 / 予防接種 )

<O: 所見・検査>:
    【身体計測】
    身長: ＿cm ( SD), 体重: ＿kg ( SD)
    頭囲: ＿cm ( SD), 胸囲: ＿cm ( SD)
    カウプ指数: ＿ ( 15-18正常 / >20肥満傾向 / <13やせ )
    成長曲線: ( 曲線に沿っている / 逸脱 / 横ばい )
    【診察】
    大泉門: ( 開 / 閉 / 膨隆 / 陥凹 )、赤色反射: ( 有 / 無 )
    口腔: ( 歯数 ＿本 / 鵞口瘡 / 上皮真珠 )
    心音・呼吸音: ( 異常なし )、腹部腫瘤: ( 無 )
    股関節開排: ( 制限なし / あり・クリック音 )
    外陰部: ( 精巣下降 / 陰唇癒合 / 鼠径ヘルニア )
    【発達確認】
    追視・あやし笑い・定頸 (3-4M)
    寝返り・お座り・持ち替え (6-7M)
    つかまり立ち・ハイハイ・パラシュート反射 (9-10M)
    独歩・有為語・指差し・積み木 (1歳半)


▶ Dr.Advice [成長曲線]: 「平均より小さい」こと自体は問題ではない。「成長曲線（パーセンタイル枠）に沿って増えているか」が重要。曲線から下方に逸脱していく（Cross down）場合は、栄養不足や器質的疾患を疑え。
▶ Dr.Advice [事故防止 (Anticipatory Guidance)]:
▶ Dr.Advice [母親のメンタル]: 子供だけでなく、母親の表情や言動（産後うつ、育児ノイローゼの兆候）にも注意を払い、必要なら地域保健師と連携せよ。
▶ Dr.Advice [安全確認]: 肥満児や痩せ型児では、年齢/体重別セットをそのまま適用すると過量・過小投与になるリスクがある。必ず mg/kg での用量確認を行え。
