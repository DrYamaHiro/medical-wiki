[Diabetes] 糖尿病・糖代謝 (初診)

<S: 主訴・ROS>:
    主訴: ( 検診異常 / 口渇 / 多飲 / 多尿 / 体重減少 / だるさ )
    【代謝・内分泌】
    口渇・多飲: ( なし / あり ), 多尿・夜間尿: ( なし / あり ＿回 )
    体重変化: ( なし / 増加 / 減少 ＿kg )
    【合併症関連 (ROS)】
    眼: ( 視力低下 / 霧視 / 飛蚊症 )
    神経: ( 手足のしびれ / こむら返り / 立ちくらみ / 便通異常 )
    心・腎: ( むくみ / 息切れ / 胸痛 )
    【生活習慣・背景】
    食事: ( 規則的 / 間食多い / 糖質過多 / アルコール )
    運動: ( なし / 定期的 ), 喫煙: ( なし / あり )
    家族歴: 糖尿病 ( 父 / 母 / 兄弟 )
    妊娠歴: ( 巨大児出産 / 妊娠糖尿病 )
    他院治療歴: ( 無 / 有: 薬剤名 ＿ )

<O: 所見・検査>:
    バイタル: BP: ＿/＿, HR: ＿, BMI: ＿ (身長 ＿cm / 体重 ＿kg), 腹囲: ＿cm
    【神経所見】
    アキレス腱反射: ( 消失 / 低下 / 正常 )
    振動覚(内果): ( ＿秒 / 低下 / 正常 )
    Touch Test: ( 感覚鈍麻 / 正常 )
    【フットチェック】
    足背動脈: ( 触知 / 減弱 ), 爪白癬・鶏眼・胼胝 ( 有 / 無 )
    【院内迅速検査】
    尿: 糖(＿) 蛋白(＿) 潜血(＿) ケトン(＿)
    血糖: ＿ mg/dL, HbA1c: ＿ %
    【外注検査】
    CPR(インスリン分泌能), 尿中微量アルブミン


▶ Dr.Advice [1型除外]: 「急激な発症」「著明な体重減少」「ケトン体陽性」「若年」「痩せ型」は1型糖尿病（インスリン依存状態）を疑う。見逃すとDKAで死に至るため、必ずCPRと抗体検査を行え。
▶ Dr.Advice [眼科受診]: 糖尿病網膜症は自覚症状なく進行し、失明原因の上位である。当院では眼底検査ができないため、初診時に必ず眼科への対診依頼 [ID: Doc-Gen-Referral] を行え。
▶ Dr.Advice [処方日数]: オンライン診療において、基礎疾患情報（お薬手帳、健診結果、紹介状等）がない初診患者の場合は「7日分を上限」とする。情報ありなら通常処方可。
