    外用状況: ( 指示通り / ベタつくので薄く / 痒い時だけ / 中止 )
    副作用: ( 刺激感 / 赤み / 皮膚萎縮 / なし )

<O: 所見・検査>:
    皮疹の改善度: ( 紅斑消失 / 色素沈着のみ / 苔癬化残存 )
    掻破痕: ( 有 / 無 )


▶ Dr.Advice [プロアクティブ療法]: アトピー等では「ツルツルになっても直ちに止めない」ことが重要。見た目が治っても顕微鏡レベルの炎症は残っている。「毎日」から「週2回（週末など）」へ頻度を減らしながら塗り続けることで、再燃を劇的に抑制できる。
