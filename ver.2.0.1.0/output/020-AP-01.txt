[J15.9] 細菌性肺炎 / 市中肺炎 (Bacterial Pneumonia)

<A: 評価>:
    ・診断根拠: 38℃以上の発熱、膿性喀痰、胸痛。胸部X線で明確な浸潤影（Consolidation）。
    ・重症度: A-DROPスコア ＿点 ( 軽症 / 中等症 / 重症 )
    ・リスク: ( 耐性菌リスクなし / 誤嚥 / COPD / 90日以内の抗菌薬使用 )

<P: 方針>:
    ・抗菌薬投与（若年者ならペニシリン系/マクロライド系、高齢・基礎疾患ありならキノロン系）。
    ・3日以内の再診を指示（解熱しない場合は入院考慮）。

    ▼ 処方・アクション (選択) ▼
    【標準治療セット（第一選択：オグサワ療法）】
    [ID: Rx-Resp-Pneu-AugZ] オグサワ療法(肺炎・咬傷)
    ・オーグメンチン配合錠250RS 1回1錠 1日3回 毎食後 7日分
    ・サワシリンカプセル250mg 1回1カプセル 1日3回 毎食後 7日分
    ※AMPC/CVA比率を最適化し下痢を減らしつつ効果を高める。

    【代替薬（キノロン系：高齢者・ペニシリンアレルギー・外来完結希望）】
    [ID: Rx-Resp-Pneu-Lvfx] レボフロキサシン(呼吸器用)
    ・レボフロキサシン錠500mg 1回1錠 1日1回 朝食後 7日分 ；結核除外必須
    [ID: Rx-Resp-Pneu-Lsv] ラスビック
    ・ラスビック錠75mg 1回1錠 1日1回 朝食後 7日分 ；高齢者・腎機能低下例に使いやすい

    【点滴治療（経口困難・中等症手前）】
    [ID: Div-Uro-Gon-Ctrx] ロセフィン(淋菌・肺炎)
    ・ロセフィン静注用1g 1瓶 + 生食100mL 点滴静注(単回～連日)
    ※非定型肺炎カバーが必要な場合は、内服マクロライド（ジスロマック等）を併用すること。

    【対症療法】
    [ID: Rx-Gen-Pain-Lox-T-Set] ロキソプロフェン・レバミピドセット(頓用)
    ・ロキソプロフェンNa錠60mg 1回1錠 (疼痛時) 10回分
    ・レバミピド錠100mg 1回1錠 (疼痛時) 10回分
    [ID: Rx-Gen-Cold-Exp] ムコダイン単剤
    ・ムコダイン錠500mg 1回1錠 1日3回 毎食後 7日分


▶ Dr.Advice [キノロンの温存と使用]: 若年者の軽症肺炎にいきなりキノロン（レボフロキサシン）を使うと、結核の診断遅延や耐性菌リスクを招くため、まずはペニシリン系（オグサワ）を推奨する。一方、高齢者で「入院せずに外来で治したい」というニーズが強い場合は、広域スペクトルと高い組織移行性を持つキノロン系（ラスビック等）を第一選択とすることは戦略的に正しい。
