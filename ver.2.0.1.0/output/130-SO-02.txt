[Endocrine] 甲状腺・内分泌 (再診)

<S: 主訴・ROS>:
    自覚症状: ( 動悸改善 / 倦怠感残存 / 変化なし )
    体重変化: ＿kg (前回比 ＿kg) ；治療効果の鋭敏な指標
    【副作用確認】
    抗甲状腺薬: ( 咽頭痛 / 発熱 / 皮疹 / 関節痛 ) ；無顆粒球症チェック
    チラージン: ( 動悸 / 不眠 / 暑がり ) ；過量投与徴候
    服薬状況: ( 良好 / 飲み忘れあり / 食後服用中 )

<O: 所見・検査>:
    脈拍: ＿ bpm
    甲状腺腫のサイズ変化。
    採血データ推移: TSH ＿ / FT4 ＿


▶ Dr.Advice [Lagging TSH (TSHの回復遅延)]: バセドウ病治療開始後、FT4が正常化してもTSHは数ヶ月間「抑制されたまま（低いまま）」のことがある。これを「効きすぎ」と誤認して抗甲状腺薬を減量すると再燃する。TSHではなく「FT4」を指標に用量を調整せよ。逆に橋本病の治療でも、FT4が正常化してからTSHが下がるまでにタイムラグがある。
