[Sleep-Apnea] 睡眠時無呼吸・睡眠障害 (初診)

<S: 主訴・ROS>:
    【睡眠関連】
    いびき: ( 指摘あり / 自分の音で起きる / 動画で確認 )
    無呼吸の指摘: ( あり / なし )
    日中の過眠: ( 運転中 / 会議中 / 食後 / テレビ視聴中 )
    ESS(眠気スコア): ＿/24点 (11点以上で病的過眠)
    中途覚醒: ＿ 回/晩, 起床時の症状: ( 頭痛 / 口渇 / 熟眠感欠如 )
    【下肢症状 (RLS疑い)】
    脚の不快感: ( 夕方～夜間に増悪 / じっとしていると悪化 / 動くと楽になる )
    感覚: ( ムズムズする / 虫が這うよう / 痛痒い )
    【背景】
    睡眠時間: ＿時間 (就床 ＿:＿ / 起床 ＿:＿)
    体重変化: ＿kg増/＿年
    既往歴: ( 高血圧 / 糖尿病 / 心房細動 / 脳卒中 )
    職業: ( 運転業務 / 高所作業 / 精密作業 ) ；事故リスク確認

<O: 所見・検査>:
    身体計測: 身長 ＿cm, 体重 ＿kg, BMI ＿
    頸囲: ＿cm (男性>38cm, 女性>35cmでリスク増)
    バイタル: BP: ＿/＿ mmHg
    【咽頭所見】
    口蓋扁桃肥大: ( Mackenzie I / II / III / IV )
    軟口蓋低位: ( 有 / 無 ), 舌根沈下: ( 有 / 無 )
    Mallampati分類: ( Class I / II / III / IV )
    【下肢】
    静脈瘤: ( 有 / 無 ), 貧血眼瞼: ( 有 / 無 )
    【検査 (外注予定)】
    携帯型睡眠時無呼吸検査（簡易検査）
    在宅睡眠時無呼吸検査（OCST/フルPSG）

<Ns: 診察前対応>:
    【検査 (外注予定)】
    [ID: Phy-Resp-HSAT] 携帯型睡眠時無呼吸検査（簡易検査）
    [ID: Phy-Resp-OCST] 在宅睡眠時無呼吸検査（OCST/フルPSG）

▶ Dr.Advice [Mallampati分類]: 開口時に口蓋垂がどれだけ見えるか。Class III（口蓋垂基部のみ）やIV（全く見えない）は気道狭窄を示唆し、SASの予測因子となる。
▶ Dr.Advice [職業運転手]: トラック、バス、タクシー等の運転業務従事者の場合、SASによる居眠り事故は社会的責任を問われる重大事案となる。「仕事を守るために検査が必要です」と説得し、未治療のまま放置させないこと。
