[Hypertension] 高血圧・血管疾患 (初診)

<S: 主訴・ROS>:
    主訴: ( 検診異常 / 自宅血圧高値 / 頭痛 / 肩こり / めまい / 動悸 )
    随伴症状: ( 胸部圧迫感 / 息切れ / なし )
    【生活習慣】
    塩分摂取: ( 多 / 適 / 少 ), 外食頻度: ＿ 回/週
    運動習慣: ( なし / 週＿回 ), 喫煙: ( なし / あり ＿本/日 )
    睡眠: ( 良好 / 不眠 / いびき・無呼吸 )
    ストレス: ( 有 / 無 ), 飲酒: ( なし / あり: ＿ )
    家族歴: ( 高血圧 / 脳卒中 / 心筋梗塞 ), 発症年齢: ＿歳頃

<O: 所見・検査>:
    身体計測: 身長 ＿cm, 体重 ＿kg, BMI ＿
    診察室血圧: ＿/＿ mmHg (左右差: 有 / 無)
    自宅平均血圧: ＿/＿ mmHg
    【診察】
    心雑音: ( 有 / 無 ), 頸動脈雑音: ( 有 / 無 )
    下肢浮腫: ( 有 / 無 ), 足背動脈触知: ( 良好 / 減弱 / 消失 )
    【スクリーニング】
    尿検査: 蛋白(＿) 糖(＿)
    心電図: ( 正常 / 左室肥大 / ST-T変化 )
    採血(外注): Cre, eGFR, K, 血糖, 脂質


▶ Dr.Advice [白衣高血圧]: 診察室血圧のみ高い場合は「白衣高血圧」の可能性がある。薬物治療開始の前に、必ず「家庭血圧」を測定させ、その平均値で判断せよ。
▶ Dr.Advice [眼底検査]: 当院では眼底検査ができないため、高血圧性変化や糖尿病網膜症の評価が必要な場合（特に尿蛋白陽性例や視力低下例）は、眼科へ対診依頼 [ID: Doc-Gen-Referral] を行え。
▶ Dr.Advice [二次性高血圧]: 若年発症（35歳以下）、重症高血圧、低カリウム血症を認める場合は、原発性アルドステロン症などの二次性高血圧を疑いホルモン検査（外注）を行え。
▶ Dr.Advice [オンライン初診処方制限]: オンライン診療において、お薬手帳、健診結果、紹介状等で基礎疾患が把握できない初診患者の場合、処方は「7日分を上限」とせよ。
