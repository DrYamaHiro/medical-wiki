[Dyslipidemia] 脂質・尿酸・肥満 (再診)

<S: 主訴・ROS>:
    食事療法: ( 継続中 / リバウンド / アルコール増 )
    運動療法: ( 継続中 / 中断 / 週＿回 )
    服薬状況: ( 良好 / 飲み忘れあり / 副作用懸念 )
    自覚症状: ( なし / 筋肉痛 / 倦怠感 / 尿の色が濃い ) ；横紋筋融解症チェック

<O: 所見・検査>:
    体重: ＿kg (前回差: ＿kg)
    血圧: ＿/＿ mmHg
    【検査結果説明】
    LDL:＿, TG:＿, HDL:＿, UA:＿
    CK(CPK): ＿ (筋肉痛時のみ測定)


▶ Dr.Advice [目標値未達時のClinical Inertia]: 生活習慣改善を3-6ヶ月行っても目標未達の場合、「様子を見ましょう」と先延ばしにすることは「治療放棄」に等しい。「数値が変わらないのは、今の対策では血管を守れないということです」と伝え、速やかに薬物介入または増量・併用療法へステップアップせよ。
▶ Dr.Advice [SAMS (スタチン関連筋症状)]: CK値が正常でも「筋痛」「脱力感」を訴える場合がある(SAMS)。患者が「薬のせい」と自己判断して黙って中止しているケースが多い。「最近、足がだるかったりしませんか？」と能動的に問いかけ、症状がある場合は休薬・変更（水溶性スタチンへの切り替え等）を提案することで信頼関係を維持せよ。
▶ Dr.Advice [横紋筋融解症のハイリスク]: 以下の因子を持つ患者は筋毒性が出やすい。投与量は慎重に設定せよ。
▶ Dr.Advice [スタチン不耐]: 「薬を飲むとだるい」「筋肉が痛い」という訴えを軽視するな。横紋筋融解症の前兆である可能性がある。CK上昇が軽度でも、自覚症状が強ければ以下の対策をとれ。
▶ Dr.Advice [グレープフルーツ禁止]: アトルバスタチンやシンバスタチンはCYP3A4で代謝されるため、グレープフルーツジュース摂取により血中濃度が上昇し、筋毒性リスクが高まる。好む患者には、影響を受けにくいロスバスタチンやピタバスタチンへの変更を検討せよ。
