[N94.6] 月経困難症・PMS (Dysmenorrhea / PMS)

<A: 評価>:
    ・診断根拠: 月経に伴う強い下腹部痛・腰痛。月経前の精神不安定。
    ・生活支障度: ( 寝込む / 仕事を休む / 鎮痛剤が効かない )
    ・リスク因子: ( 喫煙 / 前兆のある片頭痛 ) ；ピル禁忌確認

<P: 方針>:
    ・治療戦略: LEP(低用量ピル)によるコントロール、または漢方・鎮痛剤による対症療法。

    ▼ 処方・アクション (選択) ▼
    【LEP治療 (ピル)】
    [ID: Rx-Gyn-LEP-Yaz] ヤーズフレックス配合錠 1日1回 定時服用 ；連続服用
    [ID: Rx-Gyn-LEP-Fri] フリウェル配合錠LD 1日1回 定時服用 ；周期投与
    [ID: Rx-Gyn-LEP-Jem] ジェミーナ配合錠 1日1回 定時服用

    【対症療法 (鎮痛・漢方)】
    [ID: Rx-Gen-Pain-Lox-T-Set] ロキソプロフェン・レバミピドセット(頓用)
    ・ロキソプロフェンNa錠60mg 1回1錠 (疼痛時) 10回分
    ・レバミピド錠100mg 1回1錠 (疼痛時) 10回分
    [ID: Rx-Kampo-068] 芍薬甘草湯(68) 2.5g 1回1包 激痛時頓用
    [ID: Rx-Kampo-024] 加味逍遙散(24)
    ・ツムラ加味逍遙散(24)エキス顆粒 2.0g 1回1包 1日3回 食前 28日分 ；PMS主訴

    ▼ 指導・計画 (必須) ▼
    ・患者指導: 血栓症の兆候（脚の痛み、激しい頭痛等）出現時は直ちに中止し受診するよう指導。
    ・再診目安: 1ヶ月後。



▶ Dr.Advice [前兆のある片頭痛]: 「キラキラした光が見える（閃輝暗点）」を伴う片頭痛持ちに、エストロゲン（ピル）を投与すると脳卒中リスクが上昇するため「絶対禁忌」である。必ず問診せよ。
▶ Dr.Advice [血栓症の警告]: ピル服用中に「ふくらはぎの痛み・腫れ」「激しい胸痛・頭痛」が出たら、直ちに服用を中止して受診するよう指導せよ（肺塞栓・脳卒中の前兆）。喫煙者には処方できない（禁煙指導必須）。
