[Smoking-Cessation] 禁煙外来 (再診)

<S: 主訴・ROS>:
    禁煙状況: ( 完全禁煙中 / 減煙中 / スリップしてしまった )
    吸いたい気持ち: ( 消失 / 強い / 飲み会など特定の場面のみ )
    離脱症状: ( イライラ / 眠気 / 集中力低下 / 不眠 / 食欲増進 )
    副作用: ( 皮膚のかぶれ / 悪心 / 異常な夢 / 頭痛 )
    服薬状況: ( 良好 / 貼り忘れ / 飲み忘れ )

<O: 所見・検査>:
    体重変化: ＿kg ( 前回比 ＋＿kg ) ；体重増加のフォロー
    呼気CO濃度: ＿ ppm ( 禁煙成功なら低値化 )


▶ Dr.Advice [スリップ対策]: 「1本吸ってしまったからもう駄目だ」と諦める患者が多い。「1本は失敗ではなく、つまずきです。すぐに戻れば大丈夫」と励まし、ドロップアウトを防げ。
▶ Dr.Advice [体重増加]: ニコチンが抜けると代謝が落ち、味覚が戻るため体重は増えやすい（平均2-3kg）。「今はダイエットより禁煙が優先。健康メリットの方が遥かに大きい」と説明し、運動を促せ。
