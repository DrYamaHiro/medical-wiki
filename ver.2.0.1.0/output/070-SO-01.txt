[Hepatobiliary] 肝胆膵・脂肪肝 (初診)

<S: 主訴・ROS>:
    主訴: ( 検診異常 / 右季肋部痛 / 心窩部痛 / 背部痛 / 黄疸 / 倦怠感 )
    【消化器・肝胆膵】
    腹痛: ( なし / 右季肋部 / 心窩部 / 背部 ), 性状: ( 鈍痛 / 疝痛 / 持続的 )
    自覚症状: ( 皮膚掻痒感 / 尿の色が濃い / 悪心 / 食欲不振 )
    【リスク・背景】
    飲酒歴: ( なし / あり: ＿ ml × ＿ 日/週 [種類: ビール/日本酒/焼酎] )
    体重変化: ( なし / 増加 / 減少 ＿kg )
    既往歴: ( 輸血歴 / タトゥー / 胆石 / 糖尿病 / 脂質異常症 )

<O: 所見・検査>:
    バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃
    身体計測: 身長 ＿cm, 体重 ＿kg, BMI ＿
    【腹部】
    肝・脾: ( 触知せず / 肝腫大 / 脾腫 )
    圧痛: ( なし / 右季肋部 / 心窩部 ), Murphy徴候: ( 陰性 / 陽性 )
    【皮膚・眼】
    黄疸: ( なし / 眼球結膜 / 皮膚 )
    肝硬変徴候: ( 手掌紅斑 / クモ状血管腫 / 腹壁静脈怒張 / なし )
    【検査 (外注)】
    肝臓詳細セット (AST/ALT/γGTP/Bil/ALP/Alb/PT/Plt/ウイルス)
    腫瘍マーカーセット (AFP/PIVKA-II/CEA/CA19-9)

<Ns: 診察前対応>:
    【検査 (外注)】
    [ID: Lab-Hep-Full] 肝臓詳細セット
    [ID: Lab-Hep-Tumor] 腫瘍マーカーセット

▶ Dr.Advice [設備制約]: 当院には腹部エコーおよびCTがない。肝臓の形状評価、胆石の有無、胆管拡張の確認ができないため、身体所見で異常（黄疸、Murphy徴候、腫瘤）があれば、迷わず画像診断可能な施設へ紹介せよ。
▶ Dr.Advice [FIB-4 Index]: 脂肪肝を見たら必ずFIB-4 indexを算出せよ。スコア2.67以上の場合は肝硬変への進展リスクが高いため、肝臓専門医へ紹介すること。
