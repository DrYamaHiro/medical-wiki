[T20] 熱傷 (Burn)

<A: 評価>:
    ・受傷機転: ( 熱湯 / 接触 / 火炎 / 化学薬品 / 低温やけど )
    ・深達度: ( I度 / II度浅[SDB] / II度深[DDB] / III度 )
    ・範囲: ＿ ％ (手掌法：手のひら=1%)

<P: 方針>:
    ・治療戦略: 冷却。水疱温存。湿潤療法（軟膏＋ドレッシング）。顔面・手・陰部・広範囲・DDB以上は専門医へ紹介。

    ▼ 処方・処置 ▼
    【外用薬 (深達度で選択)】
    [ID: Rx-Gen-Ext-RinVG] リンデロンVG軟膏
    ・リンデロンVG軟膏0.12% 1本 1日1-2回 塗布 ；I度～SDB初期(炎症抑制)
    [ID: Rx-Gen-Ext-Azul] アズノール軟膏
    ・アズノール軟膏0.033% 1本 1日1-2回 たっぷり塗布 ；SDB～DDB(保護)
    [ID: Rx-Der-Imp-Gen] ゲンタシン軟膏
    ・ゲンタシン軟膏0.1% 1本 1日1回 塗布 ；感染合併時

    【鎮痛セット】
    [ID: Rx-Gen-Pain-Lox-T-Set] ロキソプロフェン・レバミピドセット(頓用)
    ・ロキソプロフェンNa錠60mg 1回1錠 頓用
    ・レバミピド錠100mg 1回1錠 頓用

    【紹介】
    [ID: Doc-Gen-Referral] 診療情報提供書 (皮膚科・形成外科へ紹介：植皮検討・重症熱傷)

    ▼ 指導・計画 ▼
    ・患者指導: ガーゼ直貼りは固着するため避ける。
    ・再診目安: 翌日または2日後。


▶ Dr.Advice [低温やけど]: 湯たんぽ等による低温熱傷は、見た目が軽度でも深部はIII度（壊死）であることが多い。「治るのに数ヶ月かかる」「手術が必要になるかもしれない」と初期から厳しめの見通しを伝え、早めに専門医へ送れ。
▶ Dr.Advice [ラップ療法]: 家庭用ラップでの被覆は、浸出液が貯留し細菌感染の温床になりやすいため、安易に推奨しない。清潔なドレッシング材の使用か、こまめな交換を指導せよ。
