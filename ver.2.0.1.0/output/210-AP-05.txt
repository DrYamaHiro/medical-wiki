[L03.9] 蜂窩織炎 (Cellulitis)

<A: 評価>:
    ・診断根拠: 境界不明瞭な紅斑、熱感、腫脹、圧痛。発熱。
    ・重症度: ( 軽症 / 中等症:全身症状あり / 重症:壊死性筋膜炎疑い )
    ・侵入門: ( 足白癬 / 外傷 / 潰瘍 / 虫刺され / なし )

<P: 方針>:
    ・治療戦略: 適切な抗菌薬投与と局所の挙上・冷却。

    ▼ 処方・アクション (選択) ▼
    【抗菌薬・鎮痛】
    [ID: Rx-Der-Cel-Cef] ケフレックスカプセル250mg 1回2カプセル 1日3回 7日分
    [ID: Rx-Der-Cel-Min] ミノマイシン錠100mg 1回1錠 1日2回 朝夕食後 7日分 ；MRSA疑い
    [ID: Rx-Gen-Pain-Lox-Set-5] ロキソプロフェン・レバミピドセット(定期)
    ・ロキソプロフェンNa錠60mg 1回1錠 1日3回 毎食後 5日分
    ・レバミピド錠100mg 1回1錠 1日3回 毎食後 5日分

    ▼ 指導・計画 (必須) ▼
    ・患者指導: 患部を心臓より高く保ち、冷却すること。悪化時は直ちに再診。
    ・再診目安: 2-3日後（効果判定）。



▶ Dr.Advice [壊死性筋膜炎の除外]: 「紫斑・水疱形成」「激痛（または知覚麻痺）」「握雪感（ガス産生）」「ショックバイタル」は、緊急手術が必要な壊死性筋膜炎のサインである。1時間単位で進行するため、疑わしければ直ちに救急搬送せよ。
▶ Dr.Advice [水虫の治療]: 足白癬の亀裂から菌が入ることが多い。しかし、蜂窩織炎の急性期に水虫薬を塗るとかぶれやすく、紛らわしい。「赤みが引いてから水虫の治療を始めましょう」と指導せよ。
