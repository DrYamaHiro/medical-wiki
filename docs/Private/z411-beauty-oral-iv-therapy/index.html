[Z41.1] 美容内服・点滴療法 / Beauty Oral / IV Therapy

主訴・ROS

悩み: 薄毛 / 肥満 / 疲労 / 肌荒れ / シミ / しわ 希望治療: AGA / ダイエット / プラセンタ / 美容内服 / 点滴

確認事項

既往歴: 糖尿病、膵炎、甲状腺疾患、肝障害、心疾患 内服薬: 抗凝固薬 / 糖尿病薬 / ホルモン剤 妊娠・授乳の可能性: 無 / 有 ；多くの自費薬で禁忌

所見・検査

身長: ＿cm, 体重: ＿kg, BMI: ＿ 血圧: ＿/＿ mmHg ；処方前必須

評価

脱毛パターン: Hamilton-Norwood / Ludwig 皮膚所見: 肝斑 / 老人性色素斑 / ニキビ痕

診察前対応

（データなし）

評価

  • 悩み：肝斑、シミ、くすみ、疲労、肌荒れ。

方針・処方

  • 内服セットによるメラニン抑制・抗酸化。
  • 症状に応じた点滴・注射療法の実施。

美容内服セット (自費・30日分)

  • ビタミンB群 (肌荒れ防止・代謝促進)
  • シナール配合錠 (ビタミンC・パントテン酸：メラニン抑制・コラーゲン生成)
  • L-シスチン (ターンオーバー促進・抗酸化)
  • 上記3種 ＋
  • トラネキサム酸錠250mg (抗プラスミン作用：肝斑改善・抗炎症)
  • ユベラ錠50mg (ビタミンE：血行促進・抗酸化)

美容点滴・注射 (自費)

  • アスコルビン酸2000mg (高濃度ビタミンC：美白、抗酸化、コラーゲン生成)
  • パントール100mg (パントテン酸：皮膚・粘膜の健康維持、代謝促進)
  • 強力ネオミノファーゲンシーP 20mL (グリチルリチン：肝機能改善、抗アレルギー、抗炎症)
  • トランサミン10% 2.5mL (トラネキサム酸250mg：美白、抗炎症)
  • ビオチン1mg (ビタミンH：皮膚・髪・爪の健康維持)
  • フルスルチアミン50mg 20mL (ビタミンB1誘導体：糖代謝促進、疲労回復)
  • グルタチオン注200mg 3管 (計600mg：強力な抗酸化作用、メラニン生成抑制、解毒)
  • アスコルビン酸注 2管 (ビタミンC：美白相乗効果)
  • アリナミンF注10mg 1管 (フルスルチアミン：即効性の疲労回復、乳酸分解)
  • 20%ブドウ糖 20mL (エネルギー補給)

💡 Dr.Advice

[混合診療の禁止] 原則として、同日に「保険診療（風邪薬処方など）」と「自費診療（美容薬・点滴）」を併用することは混合診療とみなされ禁止されている。どちらか一方にするか、別日に受診するよう案内せよ。

[同意書] 自由診療は保険適用外であるため、効果・副作用・費用について十分な説明を行い、必ず書面での同意を取得すること。

[休薬期間] トラネキサム酸の長期連用は血栓症リスクを考慮し、「3ヶ月内服したら1ヶ月休薬」等のサイクルを設けることを推奨する（特にピル併用者や喫煙者）。
