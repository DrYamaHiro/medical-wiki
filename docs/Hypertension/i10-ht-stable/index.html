[I10] 本態性高血圧 / 初期・安定期 / HT - Stable

主訴・ROS

主訴: 検診異常 / 自宅血圧高値 / 頭痛 / 肩こり / めまい / 動悸 随伴症状: 胸部圧迫感 / 息切れ / なし

生活習慣

塩分摂取: 多 / 適 / 少, 外食頻度: ＿ 回/週 運動習慣: なし / 週＿回, 喫煙: なし / あり ＿本/日 睡眠: 良好 / 不眠 / いびき・無呼吸 ストレス: 有 / 無, 飲酒: なし / あり: ＿ 家族歴: 高血圧 / 脳卒中 / 心筋梗塞, 発症年齢: ＿歳頃

所見・検査

身体計測: 身長 ＿cm, 体重 ＿kg, BMI ＿ 診察室血圧: ＿/＿ mmHg (左右差: 有 / 無) 自宅平均血圧: ＿/＿ mmHg

診察

心雑音: 有 / 無, 頸動脈雑音: 有 / 無 下肢浮腫: 有 / 無, 足背動脈触知: 良好 / 減弱 / 消失

スクリーニング

尿検査: 蛋白(＿) 糖(＿) 心電図: 正常 / 左室肥大 / ST-T変化 採血(外注): Cre, eGFR, K, 血糖, 脂質

診察前対応

（データなし）

評価

  • 診断根拠: 診察室血圧≧140/90、家庭血圧≧135/85。
  • 家庭血圧平均(直近): 朝 ＿/＿ mmHg, 夜 ＿/＿ mmHg
  • 降圧目標: 75歳未満:<125/75 / 75歳以上:<135/85 / 蛋白尿あり:<125/75
  • リスク因子: 喫煙 / 糖尿病 / 脂質異常症 / CKD / 肥満 / なし

方針・処方

  • Step 1: 生活習慣修正（減塩6g/日、減量）＋ 単剤開始。
  • Step 2: 目標未達なら増量または他系統との2剤併用（合剤推奨）。
  • 家庭血圧手帳の確認と指導。

処方・処置 (選択)​

第一選択セット

  • アムロジピン錠5mg 1回1錠 1日1回 朝食後 28日分 — Ca拮抗薬(確実・安全)
  • テルミサルタン錠40mg 1回1錠 1日1回 朝食後 28日分 — ARB(腎保護・代謝改善)
  • ミカムロ配合錠AP 1回1錠 1日1回 朝食後 28日分 — 合剤(代替)

生活指導

💡 Dr.Advice

[白衣高血圧] 診察室血圧のみ高い場合は「白衣高血圧」の可能性がある。薬物治療開始の前に、必ず「家庭血圧」を測定させ、その平均値で判断せよ。

[眼底検査] 当院では眼底検査ができないため、高血圧性変化や糖尿病網膜症の評価が必要な場合（特に尿蛋白陽性例や視力低下例）は、眼科へ対診依頼 [ID: Doc-Gen-Referral] を行え。

[二次性高血圧] 若年発症（35歳以下）、重症高血圧、低カリウム血症を認める場合は、原発性アルドステロン症などの二次性高血圧を疑いホルモン検査（外注）を行え。

[オンライン初診処方制限] オンライン診療において、お薬手帳、健診結果、紹介状等で基礎疾患が把握できない初診患者の場合、処方は「7日分を上限」とせよ。

[家庭血圧至上主義] 診察室血圧は「白衣効果」で高く出やすい。薬の増減は必ず「家庭血圧の平均値」を基準に行え。診察室が高くても家庭で正常なら増量してはならない（過降圧による転倒リスク）。

[75歳以上の降圧] 高齢者の厳格な降圧（<130）は、ふらつきや腎機能悪化（Jカーブ現象）を招くことがある。忍容性がなければ140台でも許容するなど、フレイル度に応じた「さじ加減」が必要。
