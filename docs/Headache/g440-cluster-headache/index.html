[G44.0] 群発頭痛 / Cluster Headache

主訴・ROS

頭痛 (OPQRST)

発症様式: 突然[雷鳴] / 急性 / 徐々 / いつもの頭痛 増悪・寛解: 体動 / 入浴 / アルコール / 光・音 / 安静 性状: ズキズキ / 締め付けられる / 殴られたような / 電撃的 部位: 片側 / 両側 / 後頭部 / 目の奥 / 全体 強度: NRS ＿/10 生活に支障あり / 寝込む 経過: ＿時間/日 持続、頻度 ＿回/月

随伴症状

悪心・嘔吐、閃輝暗点、流涙・鼻汁、肩こり しびれ、麻痺、言葉が出にくい、発熱

てんかん・発作

意識消失 有 / 無、痙攣 全身 / 一部、前兆 目撃情報: 白目 / 泡を吹く / 硬直 / 脱力

背景

既往歴: 片頭痛 / 高血圧 / 癌 / 副鼻腔炎 / 脳卒中 市販薬使用: なし / あり: ＿錠/月 ；MOHリスク評価

所見・検査

バイタル: BP: ＿/＿, HR: ＿, BT: ＿℃

神経学的所見

意識レベル: 清明 / JCS ＿ / GCS ＿ 髄膜刺激症状: 項部硬直 有 / 無、Kernig徴候 有 / 無 局所徴候: 麻痺 有 / 無、Barre徴候 有 / 無 脳神経: 眼瞼下垂 有 / 無、瞳孔不同 有 / 無 小脳症状: 指鼻試験 正常 / 測定障害

局所

側頭動脈: 正常 / 怒張 / 圧痛 ；GCA疑い 圧痛点: 副鼻腔叩打痛 有 / 無、大後頭神経圧痛 有 / 無

除外項目(Negative ROS)

雷鳴頭痛の否定、神経脱落症状なし、項部硬直なし

診察前対応

（データなし）

評価

  • 診断根拠: 片側の眼窩周囲のえぐられるような激痛（自殺頭痛）。
  • 特徴: 群発期（1-2ヶ月）に毎日、夜間同時刻に発作。
  • 自律神経症状: 流涙、結膜充血、鼻閉、眼瞼下垂（ホルネル徴候）。
  • 誘因: 期間中のアルコール（微量でも必発）。

方針・処方

  • 治療戦略: 発作期間中は「絶対禁酒」。
  • 急性期治療: 純酸素吸入（在宅酸素導入検討）、トリプタン皮下注（点鼻）。
  • 予防療法: ベラパミル、ステロイド短期内服。

処置・処方 (選択)​

急性期

  • 酸素吸入 流量指示: 7-12L/分 (リザーバー付マスク) 15分間 ；高流量必須
  • イミグランキット皮下注3mg 1キット 頭痛発作時 頓用
  • イミグラン点鼻液20 1瓶 疼痛時鼻腔内噴霧

予防療法

  • ワソラン錠40mg 1回1錠 1日3回 毎食後 14日分
  • プレドニン錠5mg 1回6錠 1日1回 朝食後 3日分 — 短期併用

💡 Dr.Advice

[SNOOP4 (Red Flags)] 以下のサインは二次性頭痛（危険）を示唆する。一つでもあれば、当院で抱えずCT/MRIのある施設へ紹介せよ。

[雷鳴頭痛の定義] 「突然」とは「何時何分」と言えるレベルの急激さであり、「バットで殴られたような」と表現される。痛みのピークが1分以内に完成する場合は、CTが正常でもルンバール（髄液検査）が必要なSAHの可能性がある。

[緑内障発作] 「頭が痛い」と言って来院しても、実は「眼圧上昇（緑内障発作）」であることがある。特に高齢女性の「片頭痛＋吐き気＋眼痛＋充血」は要注意。ペンライトで瞳孔を確認し、散瞳・対光反射消失があれば即眼科へ。

[酸素療法] 群発頭痛には「純酸素吸入（7L以上、できれば10-12L）」が極めて有効である（約15分で消失）。頻回発作には在宅酸素療法(HOT)の保険適用も認められている。導入することでQOLが劇的に改善する。
