医療知識ナレッジベース (Medical Knowledge Wiki)

概要​

本Wikiは、外来診療における診療テンプレートと教科書的解説を統合した医療ナレッジベースです。

対象ユーザー​

  • 外来担当医師・研修医
  • 看護師・医療スタッフ

使い方​

左ペイン（サイドバー）​

カテゴリ別に疾患が分類されています。疾患名をクリックすると、その疾患の診療テンプレートページへ移動します。

中央エリア（診療テンプレート）​

各ページは 2カラム構成 になっています：

  • 左カラム（診療テンプレート）: S（主訴・ROS）、O（所見・検査）、Ns（診察前対応）、A（評価）、P（方針）の各セクションと、Dr.Adviceが記載されています
  • 右カラム（教科書的詳細解説）: 最新のガイドラインや病態生理、エビデンスを追記するための共同編集エリアです

編集方法​

各ページ下部の「このページを編集」から直接GitHubで追記・修正が可能です。
